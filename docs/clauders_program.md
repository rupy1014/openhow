# 클로더즈/개인인강 운영안 (v3)

## 1) 기본 방향
- `클로더즈`: 입문 4주 코칭 프로그램(공동 운영)
- `개인인강`: 중급/심화 주제별 녹화 코스(개인 운영)
- `유튜브`: 메인 유입 채널, 클로더즈 중심 CTA
- `공개 웨비나`: 리드 수집 및 클로더즈 전환용 이벤트

## 2) 상품 구조
| 구분 | 상품 | 가격 | 핵심 가치 |
|---|---|---:|---|
| 공동 | 클로더즈 4주 입문 코칭 | 35만원 | 만들고 싶은 결과물을 4주 안에 실행 |
| 개인 | 중급/심화 개인인강 | 주제별 책정 | 특정 주제 심화 학습 |

## 3) 클로더즈 제공 범위
- 사전 녹화 학습 콘텐츠
- 주 1회 라이브 코칭(총 4회)
- 과제 피드백
- 자료실 영구 개방
- 클로더즈 출신 알럼나이 단톡방 참여

## 4) 커뮤니티 운영
- 알럼나이 단톡방 1개 운영
- 운영 목적: 질문 해결, 성과 공유, 재참여 연결
- 최소 운영 리듬: 주 1회 체크인, 월 1회 성과 공유
- 채널 규칙: 공지/질문/성과 공유 중심

## 5) 수익 분배 규칙

### A. 클로더즈 상품 매출
- 순매출 `50:50` 분배

### B. 개인인강 매출 (개인 채널 유입)
- 매출 `100%` 강의 제작자 귀속

### C. 개인인강 매출 (클로더즈 채널 유입)
- 순매출 기준 분배:
  - 강의 제작자: `70%`
  - 클로더즈(플랫폼): `30%`
- 참고: 클로더즈 내부 지분이 5:5이면, 위 30%는 내부 정산 시 각 `15%` 효과

## 6) 순매출 정의
- `순매출 = 총결제액 - VAT - PG수수료 - 환불/차지백 - 사전합의 비용`
- 분배 비율은 순매출에 적용

## 7) 추적 및 정산
- 클로더즈 유입은 전용 링크/쿠폰으로 구분
- 월 1회 정산
- 정산표 항목: 총결제, 공제, 순매출, 분배액

## 8) 유입 동선
- 유튜브 -> 공개 웨비나 -> 클로더즈(메인)
- 클로더즈/알럼나이 -> 개인인강(중급/심화)

## 9) 콘텐츠 공개 정책
- 유튜브에서는 만드는 과정, 의사결정, 시행착오를 공개한다.
- 유튜브 공개 자료는 데모/설명 목적의 축약본 중심으로 운영한다.
- 실제 실무 복붙용 파일(샘플 코드, 템플릿, 스크립트, 운영 문서 원본)은 공개하지 않는다.
- 실사용 가능한 원본 자료는 클로더즈 자료실에서만 제공한다.
- 자료실 자산의 외부 재배포/재판매는 금지한다.

## 10) 한 줄 결론
- 클로더즈는 입문 코칭 공동사업으로 키우고, 개인인강은 제작자 중심으로 확장한다.
- 단, 클로더즈 채널 유입 개인인강 매출은 `강의 제작자 70 / 클로더즈 30`으로 정산한다.
