# 블로그 가이드 품질 검수

사용자 요청: $ARGUMENTS

## 개요

`/review`는 reviewer 에이전트만 독립 실행합니다.
이미 작성된 가이드를 검수만 하고 싶을 때 사용합니다.

```
/review payment-getting-started    # 특정 가이드 검수
/review                            # 가장 최근 가이드 검수
```

---

## 실행

```
┌──────────────────────────────────────────────┐
│ 1. 대상 파일 결정                               │
│   - 파일명 지정 시: docs/guides/{입력}.md       │
│   - 미지정 시: docs/guides/ 에서 가장 최근 파일  │
└──────────────────────────────────────────────┘
                      ↓
┌──────────────────────────────────────────────┐
│ 2. reviewer 에이전트 호출                       │
│                                               │
│ 에이전트: .claude/agents/reviewer.md            │
│ 입력:                                          │
│   - docs/rules/*.md (용어 규칙)                 │
│   - docs/guides/{slug}.md                      │
│   - docs/research/{slug}-research.md (있으면)   │
│   - docs/blog-strategy.md                      │
│ 출력:                                          │
│   - 검수 리포트 (콘솔 출력)                      │
│   - 수정된 docs/guides/{slug}.md                │
└──────────────────────────────────────────────┘
```

---

```
| 삽질 포인트 | ... |

```
