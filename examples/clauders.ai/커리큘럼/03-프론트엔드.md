## ✅ Task 3-1: 프로젝트 생성

터미널을 열고, 작업할 폴더로 이동하세요.

```bash
# Mac: 바탕화면으로 이동
cd ~/Desktop

# Windows: 바탕화면으로 이동
cd %USERPROFILE%\Desktop
```

이제 Claude Code를 실행하세요:
```bash
claude
```

Claude Code가 실행되면 아래 내용을 **복사해서 붙여넣기** 하세요:

  <p style="margin: 0; white-space: pre-wrap; color: #e0e0e0;">Next.js 14 + TypeScript + Tailwind CSS로 AI 캐릭터 채팅 앱 프로젝트를 만들어줘.

프로젝트 이름: my-ai-chat

설정:
- App Router 사용
- TypeScript 사용

완료되면 확인:
```bash
cd my-ai-chat
ls
```

`package.json`, `app/`, `tailwind.config.ts` 같은 파일이 보이면 성공!

## ✅ Task 3-3: 채팅 UI 메인 페이지 만들기

터미널 **새 창**을 열고, 프로젝트 폴더에서 Claude Code 다시 실행:

```bash
cd ~/Desktop/my-ai-chat
claude
```

아래 내용을 붙여넣기하세요. **캐릭터 정보를 내 것으로 교체하세요!**

<div style="background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); border-radius: 16px; padding: 3px; margin: 1.5em 0;">
<div style="background: #0a0a1a; border-radius: 14px; padding: 20px 24px; color: #e0e0e0; line-height: 1.7; font-family: monospace;">
  <p style="margin: 0 0 8px 0; color: #a78bfa;">📋 Claude Code에 붙여넣기 (★ 대괄호 안 내용을 내 것으로 교체 ★)</p>
  <p style="margin: 0; white-space: pre-wrap; color: #e0e0e0;">app/page.tsx를 아래 기준으로 완전히 새로 만들어줘.

캐릭터 정보:
- 이름: [도비] ← 내 캐릭터 이름으로 교체
- 한 줄 설명: ["Harry Potter의 충직한 하우스엘프"] ← 내 것으로 교체
- 대표 이모지: [🧝] ← 내 캐릭터에 맞는 이모지로 교체

디자인:
- 전체 배경: [어두운 남색 (#0f172a)] ← 원하는 색으로 교체
- 채팅 영역: 화면 가득, 스크롤 가능
- AI 메시지: 왼쪽, 회색 말풍선 (rounded-2xl)
- 내 메시지: 오른쪽, [보라색] 말풍선 (rounded-2xl)
- 하단: 흰색 입력 필드 + 전송 버튼

기능 (지금은 UI만, AI 연결 아직):
- useState로 messages 배열 관리
- 전송 버튼/Enter → 내 메시지가 messages에 추가, 입력창 초기화
- AI 답변은 "..." 로딩 표시만
- 화면 처음 열면 AI 캐릭터의 첫 인사말 자동 표시
- 새 메시지 올 때 자동 스크롤 아래로

TypeScript 타입:
type Message = {
  role: 'user' | 'assistant';
  content: string;
  timestamp: Date;
}</p>
</div>
</div>

작업이 완료되면 `http://localhost:3000` 새로고침!

### 마음에 안 드는 부분이 있으면?

Claude Code에 바로 요청하세요. 예시:

```
말풍선 모서리를 더 둥글게 해줘
헤더에 그라데이션 배경 추가해줘
글자 크기를 조금 더 크게 해줘
캐릭터 이모지를 헤더에 더 크게 표시해줘 (64px)
채팅 영역 배경을 약간 더 밝게 해줘
```

이렇게 한 줄씩 구체적으로 요청하면 빠르게 수정돼요.

## ✅ Task 3-5: 모바일 확인

브라우저에서 `F12` (개발자 도구) → 왼쪽 상단 **폰 아이콘** 클릭 → 모바일 화면으로 보기

모바일에서도 레이아웃이 깨지지 않으면 성공!

> 💡 Tailwind CSS는 기본적으로 반응형이에요.
> Claude Code가 만든 코드라면 대부분 자동으로 모바일에서도 잘 보여요.
