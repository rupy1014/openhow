## ✅ Task 1-1: Node.js 설치

Node.js는 자바스크립트를 실행하는 엔진이에요. Claude Code와 Next.js를 쓰려면 필요해요.

**설치 방법:**
1. `nodejs.org` 접속
2. **LTS 버전** (왼쪽, "Recommended For Most Users") 다운로드
3. 설치 프로그램 실행 → 기본값으로 계속 클릭

**설치 확인** (터미널 열기):
```bash
node --version
```

`v20.x.x` 같은 버전 번호가 나오면 성공!


:::info
Mac: Spotlight(⌘+Space) → "터미널" 검색 → Enter

Windows: 시작 버튼 → "PowerShell" 검색 → Enter
:::

## ✅ Task 1-2: Claude Code 설치

Claude Code는 터미널에서 대화하면서 코드를 만들어주는 AI예요.

터미널에서 실행:
```bash
npm install -g @anthropic-ai/claude-code
```

**설치 확인:**
```bash
claude --version
```

`Claude Code x.x.x` 같은 버전 번호가 나오면 성공!

> 💥 **"npm: command not found"** 에러가 나요?
> → Node.js가 설치됐는지 확인하세요. Task 1-1 다시 시도해보세요.

## ✅ Task 1-3: VS Code 설치

VS Code는 코드 파일을 열어서 확인하는 고급 메모장이에요.
Claude Code가 만든 파일을 눈으로 볼 때 씁니다.

1. `code.visualstudio.com` 접속
2. 내 운영체제에 맞는 버전 다운로드
3. 설치 후 실행

왼쪽에 아이콘 5개가 있는 화면이 뜨면 성공!

---

## ✅ Task 1-4: Git 설치 + GitHub 계정

**Git**은 코드의 변경 이력을 관리하는 도구예요. 게임 세이브 파일 같은 것.
**GitHub**는 그 이력을 클라우드에 올리는 서비스예요.

**Git 설치 확인:**
```bash
git --version
```

숫자가 나오면 이미 설치됨! 에러가 나면 `git-scm.com`에서 다운로드

**GitHub 계정 만들기:**
1. `github.com` 접속 → Sign up
2. 이메일 인증까지 완료
3. 무료 플랜(Free)으로 시작

**Git에 내 정보 등록** (처음 한 번만):
```bash
git config --global user.name "내 이름"
git config --global user.email "내 이메일@gmail.com"
```

> 💡 큰따옴표 안에 실제 정보를 넣으세요. (예: "홍길동", "hong@gmail.com")

## ✅ Task 1-5: OpenRouter API 키 발급

OpenRouter는 GPT-4o, Claude, Gemini 등 여러 AI를 한 곳에서 쓸 수 있는 서비스예요.
무료 크레딧이 있어서 오늘 실습에 충분해요.

**API 키 발급 순서:**

1. `openrouter.ai` 접속
2. **Sign Up** → Google 또는 GitHub으로 가입 (빠름)
3. 가입 후 상단 메뉴 → **Keys** 클릭
4. **Create Key** 버튼
5. 이름: `my-chat-app` 입력 → Create
6. 생성된 `sk-or-v1-...` 키를 **메모장에 복사해두기**
