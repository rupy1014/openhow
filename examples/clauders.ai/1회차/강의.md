## 0:00 – 0:10 | AI가 실제로 하는 일

### AI = 지치지 않는 실무 인턴

<div style="background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); border-radius: 16px; padding: 3px; margin: 1.5em 0;">
<div style="background: #1a1a2e; border-radius: 14px; padding: 24px 28px; color: #e0e0e0; line-height: 1.8;">
  <p style="margin: 0 0 10px 0;"><strong>흔한 오해:</strong> AI는 검색 엔진이거나 챗봇이다</p>
  <p style="margin: 0 0 16px 0; color: #94a3b8;">→ 질문하면 답변하는 것, 그 이상은 없는 줄 알았다</p>
  <p style="margin: 0 0 10px 0;"><strong style="color:#4ade80;">실제:</strong> AI는 내가 시키는 걸 해주는 실무 인턴이다</p>
  <p style="margin: 0; color: #94a3b8;">→ 이메일 써줘 → 써준다. 요약해줘 → 요약한다. 아이디어 내줘 → 낸다.</p>
</div>
</div>

**AI가 잘 하는 것:**
- 글 쓰기 (이메일, 보고서, SNS 캡션, 제안서)
- 정리 (회의록 요약, 핵심 추출, 분류)
- 아이디어 (브레인스토밍, 기획 구조화, 피드백)
- 번역 / 다듬기 (어색한 문장 고치기, 톤 조정)

**AI가 못 하는 것:**
- 나만 아는 내부 맥락 (알려줘야 함)
- 최종 판단 (항상 내가 검토)
- 실시간 정보 (최신 뉴스, 오늘 주가)

## 0:20 – 0:50 | 실습 1 — 이메일 / 메시지 쓰기

### 왜 이게 먼저냐

이메일 하나 쓰는 데 평균 15분 걸린다. AI 쓰면 2분.
어색한 표현 고민, 어떻게 시작할지 고민 — 다 AI한테 넘기면 된다.
