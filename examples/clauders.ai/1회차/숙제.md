# 1회차 숙제: 내 캐릭터로 만들기

> "도비는 예시였어요. 이제 진짜 내가 원하는 캐릭터로 바꿀 차례입니다."

**⏱ 권장 시간**: 2시간 (분산해서 해도 OK)
**🎯 목표**: 내 캐릭터로 교체 + UI 커스터마이즈 + 스크린샷 준비

## Step 1: 내 AI 캐릭터 설계서 작성

종이나 메모장에 채워보세요. 이게 캐릭터의 핵심 설명서예요.

```
=== 내 AI 캐릭터 설계서 ===

이름: ___________
이모지: ___________
한 줄 설명: ___________  (예: "Harry Potter의 충직한 하우스엘프")

성격 (2~3가지):
  - ___________
  - ___________
  - ___________

말투 특징 (구체적으로):
  - ___________
  - ___________

말투 예시:
  사용자: "안녕!" → 캐릭터: ___________
  사용자: "오늘 심심해" → 캐릭터: ___________
  사용자: "틀린 정보 얘기하면?" → 캐릭터: ___________

절대 하면 안 되는 것:
  - ___________
  - ___________
```

## Step 2: 시스템 프롬프트 교체

개발 서버가 실행 중인지 확인 (`npm run dev`), Claude Code 열기:

```bash
cd ~/Desktop/my-ai-chat
claude
```

Claude Code에 입력:

<div style="background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); border-radius: 16px; padding: 3px; margin: 1.5em 0;">
<div style="background: #0a0a1a; border-radius: 14px; padding: 20px 24px; color: #e0e0e0; font-family: monospace; line-height: 1.7;">
  <p style="margin: 0 0 8px 0; color: #a78bfa;">📋 프롬프트 (★ 대괄호 안을 내 캐릭터로 교체 ★)</p>
  <p style="margin: 0; white-space: pre-wrap;">app/api/chat/route.ts의 시스템 프롬프트를 아래로 교체해줘:

"당신은 [캐릭터 이름]입니다. [캐릭터 설명].
다음 규칙을 반드시 지키세요:
1. [말투 규칙 1]
2. [말투 규칙 2]
3. [말투 규칙 3]
4. 절대 캐릭터에서 벗어나지 마세요
5. [금지 사항]"</p>
</div>
</div>

**예시 (냥냥이 캐릭터):**
```
app/api/chat/route.ts의 시스템 프롬프트를 아래로 교체해줘:

"당신은 냥냥이입니다. 도도하고 자존심 강한 고양이 AI예요.
다음 규칙을 반드시 지키세요:
1. 어미에 '냥' 붙이기 (예: '그렇냥', '모르냥')
2. 짧고 퉁명스럽게 대답
3. '...' 으로 무심함 표현을 자주 활용
4. 칭찬받으면 기쁘지만 절대 티 내지 않기
5. 반말 사용, '입니다'/'요' 같은 격식체 절대 금지"
```

## 막혔을 때

**캐릭터가 말투를 안 지켜요:**
```
app/api/chat/route.ts의 시스템 프롬프트를 더 강하게 바꿔줘.
"절대", "반드시", "항상" 같은 강조어를 더 추가하고,
구체적인 예시도 넣어줘.
```

**디자인이 마음에 안 들어요:**
```
현재 디자인에서 [구체적으로 어떤 부분]을 [어떻게] 바꿔줘.
```

**에러가 났어요:**
```
F12 → Console 탭 → 에러 메시지 전체 복사 → Claude Code에 붙여넣기 + "고쳐줘"
```
