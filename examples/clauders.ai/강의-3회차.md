### Task A: Supabase 프로젝트 생성

1. `supabase.com` 접속 → **Start for free** → GitHub으로 로그인
2. **New project** 클릭
3. 설정:
   - Name: `my-ai-chat`
   - Database Password: 기억할 수 있는 비밀번호 설정
   - Region: **Northeast Asia (Tokyo)** 선택 (한국에서 가장 빠름)
4. **Create new project** 클릭 → 약 1~2분 대기

### Task B: 테이블 만들기

Supabase 대시보드 → **SQL Editor** → 아래 SQL 실행:

<div style="background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); border-radius: 16px; padding: 3px; margin: 1.5em 0;">
<div style="background: #0a0a1a; border-radius: 14px; padding: 20px 24px; color: #e0e0e0; line-height: 1.7; font-family: monospace;">
  <p style="margin: 0 0 8px 0; color: #a78bfa;">📋 SQL Editor에 붙여넣기 → RUN</p>
  <pre style="margin: 0; color: #e0e0e0; font-size: 0.9em;">create table messages (
  id uuid default gen_random_uuid() primary key,
  role text not null,
  content text not null,
  created_at timestamptz default now()
);

-- 누구나 읽고 쓸 수 있도록 (간단한 설정)
alter table messages enable row level security;
create policy "public access" on messages for all using (true);</pre>
</div>
</div>

**Run** 클릭 → "Success" 확인

### Task C: API 키 가져오기

Supabase 대시보드 → **Settings** → **API**

복사해야 할 것:
- **Project URL**: `https://xxxxx.supabase.co`
- **anon public** key: `eyJhbGciOi...` (긴 문자열)

<div style="background: #0a0a1a; border-radius: 14px; padding: 20px 24px; color: #e0e0e0; line-height: 1.7; font-family: monospace;">

### Task E: Vercel 환경변수 추가

Claude Code가 알려주는 키들을 Vercel에도 추가해야 해요:

Vercel 대시보드 → 내 프로젝트 → **Settings** → **Environment Variables**

| KEY | VALUE |
|-----|-------|
| `NEXT_PUBLIC_SUPABASE_URL` | `https://xxxxx.supabase.co` |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | `eyJhbGciOi...` |

추가 후 → **Deployments** 탭 → 최근 배포 → **Redeploy**
