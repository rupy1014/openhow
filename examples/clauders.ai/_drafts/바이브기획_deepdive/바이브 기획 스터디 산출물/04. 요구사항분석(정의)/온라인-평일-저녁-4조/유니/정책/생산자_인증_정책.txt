1. 정책 목적

본 정책은 직접 농산물을 재배하는 생산자만이 플랫폼에서 판매할 수 있도록 
자격을 검증함으로써, 소비자에게 신뢰할 수 있는 로컬 농산물 구매 환경을 
제공하고 중간 유통업자의 무단 진입을 차단하는 것을 목적으로 한다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

2. 적용 대상

2-1. 인증 대상
다음 각 호에 해당하는 자는 생산자 인증을 받아야 판매 활동이 가능하다:

① 신규 판매회원 가입 희망자 (필수)
② 기존 판매회원 중 신규 재배 품목 추가 시 (추가 인증)
③ 재배지 이전 또는 추가 시 (재인증)
④ 연간 정기 재인증 대상자 (매년 1회, 가입 기념일 기준)

2-2. 인증 면제 대상
다음 각 호는 별도 인증 절차 면제:

① 정부 인증 스마트팜 운영자 (자료 제출로 갈음)
② 농협 로컬푸드 직매장 정식 출하자 (출하자 증명서 제출)
③ GAP(우수농산물인증), 유기농인증 보유 농가 (인증서 유효기간 내)
   - 단, 재배지 현장 확인은 1회 이상 필수

3. 정책 기준 (필수 조건)

3-1. 기본 자격 요건

□ 필수 요건 A: 재배 주체 증명
   ├ A-1. 사업자등록증 (개인 또는 법인)
   │      - 업태: 농업, 업종: 작물재배업 필수
   │      - 발급일 기준 6개월 이내 등본 제출
   ├ A-2. 농지원부 또는 토지등기부등본
   │      - 본인 소유 농지 OR 임차 계약서 (3년 이상)
   │      - 재배 예정 작물과 농지 면적 일치 확인
   └ A-3. 본인 확인 서류
          - 주민등록증 또는 운전면허증 사본
          - 신분증 소지 셀카 사진 (위변조 방지)

□ 필수 요건 B: 재배 사실 증명
   ├ B-1. 재배지 현장 사진 (GPS 위치정보 포함)
   │      - 최소 4장: 전경, 작물 근접, 표지판(농장명), 본인 인증
   │      - 촬영일: 신청일 기준 7일 이내
   │      - GPS 좌표와 농지원부 주소 일치 확인
   ├ B-2. 재배 품목 입증 자료
   │      - 비료·농약 구매 영수증 (최근 6개월)
   │      - 농협 거래내역 또는 산지유통인 출하증명서
   │      - 작물별 재배 일지 (선택, 가산점)
   └ B-3. 재배 규모 증빙
          - 주요 품목: 최소 300㎡ (약 90평) 이상
          - 틈새 품목: 최소 100㎡ (약 30평) 이상
          - 농지원부와 현장 사진으로 교차 검증

□ 필수 요건 C: 안전성 확보
   ├ C-1. 원산지 증명 서약서 (자필 서명)
   │      - "본인이 직접 재배한 국내산 농산물임을 확인합니다"
   │      - 허위 작성 시 형사 고발 동의 포함
   ├ C-2. 잔류농약 검사 (고위험 품목)
   │      - 대상: 엽채류, 딸기, 토마토 등
   │      - 주기: 연 1회 이상
   │      - 검사 기관: 국립농산물품질관리원 또는 공인 기관
   └ C-3. 개인정보 제공 동의
          - 재배지 위치 정보 플랫폼 공개 동의
          - 소비자 문의 시 연락처 제공 동의

3-2. 추가 검증 사항

□ 플랫폼 자체 검증
   ├ 1) 재배지 현장 방문 확인 (신규 가입자 필수)
   │     - 방문 시기: 승인 신청 후 14일 이내
   │     - 확인 사항: 재배 여부, 규모, 작물 상태
   │     - 방문 불가 시: 화상통화(영상통화)로 대체 가능
   ├ 2) 기존 유통 이력 조회
   │     - 농산물이력추적시스템(QR코드) 등록 여부
   │     - 타 플랫폼 판매 이력 및 평점 확인
   └ 3) 반경 검색 (중복 방지)
          - 동일 주소지 다중 계정 방지
          - GPS 좌표 기준 반경 500m 내 중복 신청 검토

4. 예외 조건 (허용 범위)

4-1. 서류 제출 예외

상황 A: 고령 농업인 (만 65세 이상)
  ├ 완화 조건: 자녀 또는 대리인이 서류 제출 가능
  │           - 대리인 위임장 및 신분증 사본 첨부
  ├ 현장 확인: 반드시 본인이 재배 중임을 확인
  └ 연락처: 본인 또는 자녀 병기 가능

상황 B: 공동 경작지 (영농조합법인, 협업 농장 등)
  ├ 허용 조건: 법인 대표자 명의로 가입
  │           - 법인등기부등본 제출
  │           - 조합원 명부 제출
  ├ 판매 권한: 법인 대표자만 상품 등록
  └ 정산: 법인 계좌로 일괄 입금

상황 C: 임차 농지 경작자
  ├ 허용 조건: 임대차 계약서 제출 (잔여 기간 3년 이상)
  │           - 임대인 연락처 및 동의서 필수
  ├ 확인 절차: 임대인에게 재배 사실 확인 전화
  └ 제한 사항: 임차 기간 만료 6개월 전 재인증

상황 D: 재배지 사진 촬영 불가 (동절기, 휴경기 등)
  ├ 대체 자료: 전년도 출하증명서 또는 수확 사진
  │           - 농협 출하전표, 산지유통인 계약서 등
  ├ 조건부 승인: 차기 작기 시작 시 현장 사진 재제출
  └ 판매 제한: 조건 이행 전까지 판매 품목 제한

4-2. 재배 규모 예외

상황 E: 소량 다품목 재배 농가
  ├ 허용 조건: 총 재배 면적 300㎡ 이상이면 허용
  │           - 예: 100㎡ 상추 + 100㎡ 토마토 + 100㎡ 고추
  ├ 입증 방법: 품목별 구역 구분 사진 제출
  └ 판매 한도: 품목당 월 판매량 제한 (과다 판매 방지)

상황 F: 희귀 품목 재배자 (산나물, 약용작물 등)
  ├ 완화 기준: 최소 면적 50㎡까지 허용
  ├ 추가 증빙: 종자 구매 영수증 또는 재배 교육 이수증
  └ 모니터링: 월 판매량 기준 재배 가능량 추정

6. 책임 주체

6-1. 신청자(생산자)의 책임
───────────────────────────────────────────
① 모든 제출 서류의 진실성 및 적법성에 대한 책임
② 재배지 현황 변경 시 즉시 플랫폼에 통보할 의무
   - 재배지 이전, 재배 품목 변경, 재배 중단 등
③ 정기 재인증 기간 준수 (기한 경과 시 판매 자동 중지)
④ 원산지 표시 의무 위반 시 형사 책임 (최대 징역 7년)
⑤ 허위 서류 제출 시 계약 해지 및 민·형사상 책임

6-2. 플랫폼의 책임
───────────────────────────────────────────
① 신청자가 제출한 서류의 진위 여부를 상당한 주의로 심사
   - "상당한 주의": 육안 검토, 공공DB 조회, 현장 확인 등
② 현장 확인 시 신청자의 개인정보 및 영업 비밀 보호
③ 부적격 판정 시 구체적 사유 및 개선 방법 안내
④ 정당한 사유 없이 승인을 지연하거나 자의적 판단 금지
⑤ 허위 생산자가 승인된 경우:
   - 플랫폼이 "상당한 주의"를 다했음을 입증하면 면책
   - 태만 또는 고의 간과 시 연대 책임

6-3. 소비자의 권리
───────────────────────────────────────────
① 생산자 정보 열람 권리
   - 재배지 위치, 재배 품목, 인증 이력 등
② 생산자 인증에 의심이 가는 경우 플랫폼에 신고
   - 조사 결과 통지 (개인정보 범위 내)
③ 허위 생산자로 판명 시 전액 환불 및 위자료 청구 가능

7. 분쟁 발생 시 판단 기준

7-1. 주요 분쟁 유형 및 판단 기준
═══════════════════════════════════════════════════

【분쟁 유형 1】 서류 진위 논란
─────────────────────────────────────────
상황: 신청자가 제출한 농지원부가 위조 의심

판단 기준:
  ├ 1단계: 발급 기관(읍·면사무소)에 진위 확인 요청
  ├ 2단계: 진위 확인 불가 시 신청자에게 원본 제출 요청 (기한 3일)
  ├ 3단계: 원본 미제출 또는 위조 확인 시 즉시 반려
  └ 결과: 위조 확인 시 영구 가입 차단 + 수사기관 통보

책임 소재:
  • 위조 서류 제출: 신청자 전적 책임 (형사 처벌 대상)
  • 플랫폼: 발급 기관 확인 의무만 부담


【분쟁 유형 2】 재배 규모 부풀리기
─────────────────────────────────────────
상황: 재배지 300㎡라고 신청했으나 실제는 100㎡

판단 기준:
  ├ 1단계: 현장 방문 시 측정 (보폭법 또는 줄자)
  │        - 오차 범위 ±20%까지 허용
  │        - 예: 신고 300㎡ → 실측 240~360㎡ 허용
  ├ 2단계: 오차 20% 초과 시 신청자에게 소명 기회
  │        - 측정 방법 이의 제기 가능
  │        - 재측정 요청 가능 (1회 한정)
  ├ 3단계: 재측정에도 기준 미달 시 조건부 승인
  │        - 실측 면적 기준 월 판매량 제한
  └ 결과: 고의적 허위 신고 시 승인 거부

책임 소재:
  • 측정 오차: 양측 협의로 해결
  • 고의적 허위: 신청자 책임 (승인 거부 또는 제재)


【분쟁 유형 3】 타인 재배지 도용
─────────────────────────────────────────
상황: A가 B의 재배지를 자신의 것처럼 허위 신청

판단 기준:
  ├ 1단계: GPS 좌표와 농지원부 명의 일치 여부 확인
  ├ 2단계: 현장 방문 시 본인 확인 (신분증 대조)
  ├ 3단계: 본인 부재 시 재배지 소유주/임차인 확인 전화
  └ 결과: 타인 재배지 도용 확인 시 즉시 반려 + 영구 차단

책임 소재:
  • 도용 행위: 신청자 전적 책임 (형사 고발 가능)
  • 피해자(B): 플랫폼에 신고 시 즉시 조치


【분쟁 유형 4】 중간 유통업자 위장
─────────────────────────────────────────
상황: 도매시장에서 농산물을 구매하여 재판매하려는 유통업자

판단 기준:
  ├ 증거 수집:
  │  • 재배지 현장 방문 시 재배 흔적 없음
  │  • 비료·농약 구매 내역 없음
  │  • 사업자등록증 업종이 "도매업" 또는 "소매업"
  │  • 판매량이 신고 재배 면적 대비 과다 (월 150% 초과)
  ├ 판단:
  │  • 위 증거 중 2개 이상 해당 시 추가 소명 요청
  │  • 소명 불충분 시 승인 거부
  └ 모니터링:
     • 승인 후에도 월 판매량 지속 모니터링
     • 이상 징후 발견 시 재조사

책임 소재:
  • 위장 입점: 신청자 전적 책임 (자격 박탈 + 손해배상)


【분쟁 유형 5】 고령 농업인 대리 신청 논란
─────────────────────────────────────────
상황: 자녀가 부모(고령 농업인) 명의로 신청했으나, 실제 재배는 자녀

판단 기준:
  ├ 허용 조건:
  │  • 부모가 실제 재배 중이며, 자녀는 서류 작성만 대리
  │  • 현장 확인 시 부모가 재배 과정 설명 가능
  ├ 불허 조건:
  │  • 부모는 명의만 빌려주고, 자녀가 전적으로 재배
  │  • 부모가 재배 내용을 전혀 모름
  └ 판단 방법:
     • 현장 방문 시 부모와 직접 면담
     • 재배 품목, 재배 방법 등 기본 질문
     • 답변 불가 시 "명의 대여"로 간주 → 승인 거부

책임 소재:
  • 정당한 대리: 문제없음 (부모가 실제 재배자)
  • 명의 대여: 신청자(부모+자녀) 공동 책임


【분쟁 유형 6】 임차 농지 계약 분쟁
─────────────────────────────────────────
상황: 임대인이 "임차인에게 재배 허락한 적 없다"고 주장

판단 기준:
  ├ 1단계: 임대차 계약서 확인
  │        - 계약 목적: "농지 임대" 명시 여부
  │        - 계약 기간: 잔여 기간 3년 이상 여부
  ├ 2단계: 임대인에게 사실 확인 전화
  │        - "○○님이 귀하의 토지에서 농사를 짓고 있습니까?"
  │        - "온라인 판매에 동의하십니까?"
  ├ 3단계: 임대인 부인 시 신청자에게 소명 기회
  │        - 추가 증빙 자료 제출 (기한 7일)
  │        - 임대인 설득 후 동의서 재제출
  └ 결과: 임대인 동의 없으면 승인 불가

책임 소재:
  • 계약서 위조: 신청자 책임 (형사 처벌)
  • 임대인 번복: 민사 분쟁 (플랫폼 개입 불가)


【분쟁 유형 7】 현장 확인 거부
─────────────────────────────────────────
상황: 신청자가 "바쁘다", "개인정보 침해다"며 현장 확인 거부

판단 기준:
  ├ 원칙: 현장 확인은 필수 절차 (약관 명시)
  ├ 대안 제시:
  │  • 일정 조율 (최대 3회까지 연기 가능)
  │  • 영상통화 대체 (실시간 재배지 확인)
  ├ 최종 거부 시:
  │  • 승인 불가 안내
  │  • 사유: "본 플랫폼은 신뢰 기반 거래를 위해
  │           현장 확인을 필수로 합니다"
  └ 예외: 정부 인증 보유자는 서류로 갈음 가능

책임 소재:
  • 정당한 거부 사유 없음 → 승인 불가 (신청자 불이익 감수)
  • 플랫폼의 과도한 요구 → 플랫폼 재검토

8. 운영 질문 (FAQ) - 내부 운영자용

Q1. 사업자등록증 업종이 "작물재배업"이 아니라 "기타 농업"인 경우?
A1. 허용. 세부 업종 코드 확인 후 농업 관련이면 승인.
    단, "농산물 도매업"은 불가.

Q2. 재배지 사진에 GPS 정보가 없는 구형 휴대폰 사용자는?
A2. 대체 방법 제공:
    ├ 방법1: 네이버지도 앱 → 현재 위치 캡처 → 사진과 함께 제출
    ├ 방법2: 재배지 주소가 적힌 표지판과 함께 촬영
    └ 방법3: 현장 방문으로 대체 (비용 무료)

Q3. 동일 주소에 부모·자녀가 각각 다른 작물 재배 시?
A3. 허용. 단, 각자 재배 구역이 명확히 구분되어야 함.
    ├ 요구 사항: 구역별 사진, 각자 판매 품목 상이
    └ 현장 확인 시 구역 구분 실제 확인

Q4. 작년에 승인받았는데, 올해 태풍으로 재배지 전체 유실된 경우?
A4. 즉시 플랫폼에 통보 → 판매 일시 중지.
    ├ 재개 조건: 복구 후 재배지 사진 재제출
    └ 복구 불가 시: 계약 해지 (위약금 없음)

Q5. 재인증 기한을 놓쳤는데, 바쁘게 일하다 보니 깜빡한 경우?
A5. 1차 유예: 알림 후 14일 유예 기간 제공
    └ 14일 내 미제출: 판매 자동 중지 (재신청으로 복구 가능)

Q6. 신청자가 "농지원부가 없다"고 하는 경우?
A6. 농지원부는 농지 소유·임차 시 의무 발급 대상.
    ├ 없는 경우: 읍·면사무소 방문하여 발급 (무료)
    └ 대체 서류: 토지등기부등본 + 재배 확인서 (이장 확인)

Q7. GAP 인증 농가인데, 현장 확인을 꼭 해야 하나?
A7. 1회는 필수. 이후 연간 재인증 시 서류로 갈음 가능.
    └ 사유: 인증 후 재배지 이전 가능성, 위조 방지

Q8. 외국인 농업인(결혼이주여성 등)도 가능한가?
A8. 가능. 단, 사업자등록증 또는 외국인등록증 제출.
    └ 재배 사실만 입증되면 국적 무관
