제1장  총 칙

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

제1조 (정책 목적)

본 정책은 직접 농산물을 재배하는 생산자만이 플랫폼에서 판매할 수 있도록
자격을 검증함으로써, 중간 유통업자의 무단 진입을 차단하고 소비자에게
신뢰할 수 있는 로컬 농산물 구매 환경을 제공하는 것을 목적으로 한다.

제2조 (적용 대상)

① 인증 필수 대상
   다음 각 호에 해당하는 자는 생산자 인증을 받아야 판매 활동이 가능하다:

   1. 신규 판매회원 가입 희망자 (필수)
   2. 기존 판매회원 중 신규 재배 품목 추가 시 (추가 인증)
   3. 재배지 이전 또는 추가 시 (재인증)
   4. 연간 정기 재인증 대상자 (매년 1회, 가입 기념일 기준)

② 인증 간소화 대상 (제7조 참조)
   다음 각 호는 서류 제출이 간소화되나 최소 본인 확인은 필수:

   1. 정부 인증 스마트팜 운영자
   2. 농협 로컬푸드 직매장 정식 출하자
   3. GAP·유기농·무농약 인증 보유 농가

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

제3조 (용어 정의)

본 정책에서 사용하는 용어의 정의는 다음과 같다:

1. "생산자": 본인 소유 또는 임차한 농지에서 직접 농산물을 재배하는 자
2. "재배지": 농산물을 실제로 재배하는 토지 (농지원부 또는 임대차계약서 상 주소)
3. "재배 가능량": 재배 면적 × 품목별 ㎡당 표준 수확량 × 안전계수(0.8)
4. "현장 확인": 재배지 방문 또는 실시간 영상통화를 통한 재배 사실 확인
5. "조건부 승인": 일부 조건 이행을 전제로 한 임시 승인 (최대 6개월)
6. "상당한 주의": 제13조 제2항에 정한 5개 필수 항목 모두 이행
7. "이상 판매": 월 판매량이 재배 가능량의 120% 이상 초과
8. "품목군": 재배 난이도와 수익성에 따른 3단계 분류 (1군/2군/3군)

제2장  인증 기준

제4조 (필수 요건)

생산자 인증을 받기 위해서는 다음 3대 필수 요건을 모두 충족하여야 한다:

【필수 요건 A: 재배 주체 증명】

① 사업자등록증 (개인 또는 법인)
   - 업태: 농업, 업종: 작물재배업 또는 기타 농업
   - 발급일 기준 6개월 이내 등본 제출
   - 국세청 홈택스를 통한 사업자 상태 조회 필수

② 농지 소유 또는 임차 증빙
   - 본인 소유: 농지원부 또는 토지등기부등본
   - 임차: 임대차 계약서 (잔여 기간 3년 이상)
     • 임대인 연락처 기재
     • 임대인 동의서 첨부 (온라인 판매 동의 포함)
     • 임대인 본인 확인 전화 실시 (통화 내용 녹취)

③ 본인 확인 서류
   - 주민등록증 또는 운전면허증 사본
   - 신분증 소지 셀카 사진 (위변조 방지)


【필수 요건 B: 재배 사실 증명】

① 재배지 현장 사진 (GPS 위치정보 포함)
   - 최소 4장: 전경, 작물 근접, 표지판(농장명 또는 주소), 본인 인증
   - 촬영일: 신청일 기준 7일 이내
   - GPS 좌표와 농지원부 주소 ±1km 이내 일치 확인
   - 메타데이터 자동 추출 및 검증

② 재배 품목 입증 자료
   - 필수: 비료·농약 구매 영수증 (최근 6개월 이내)
   - 선택(가산점): 농협 거래내역, 산지유통인 출하증명서, 재배 일지

③ 재배 규모 증빙 (제5조 참조)


【필수 요건 C: 안전성 확보】

① 원산지 증명 서약서 (자필 서명 필수)
   - 서약 내용: "본인이 직접 재배한 국내산 농산물임을 확인합니다"
   - 허위 작성 시 형사 고발 동의 포함
   - 「농수산물의 원산지 표시에 관한 법률」 위반 시 최대 징역 7년 고지

② 잔류농약 검사 (제6조 참조)

③ 개인정보 제공 동의
   - 재배지 위치 정보 플랫폼 공개 동의 (지도 표시)
   - 소비자 문의 시 연락처 제공 동의

① 품목군별 최소 재배 면적

   │           허브류, 특수 버섯, 새싹채소       │
   │ ※ 전체 19개 품목은 [별첨3] 참조             │
   └─────────────────────────────────────┘

② 리스트에 없는 품목

   1. 신청자는 사전에 플랫폼에 문의 (이메일 또는 전화)
   2. 플랫폼은 문의 후 3영업일 이내 회신
   3. 판단 기준: 단위 면적당 예상 수확량 및 시장 거래량
   4. 유사 품목 기준 적용

③ 혼작 재배 (여러 품목 동시 재배)

   1. 총 재배 면적이 가장 많은 품목군의 기준 적용
      예) 1군 150㎡ + 2군 150㎡ = 총 300㎡ → 1군 기준(300㎡) 적용 ○
      예) 2군 150㎡ + 3군 100㎡ = 총 250㎡ → 2군 기준(200㎡) 적용 ○

   2. 품목별 구역이 명확히 구분되어야 함 (구역별 사진 제출)

④ 재배 면적 측정 방법

   1. 신청자 자가 측정 시:
      - 권장 앱: 네이버지도, 카카오맵 면적 측정 기능
      - 측정 방법 가이드 영상 제공 (QR코드 스캔)

   2. 플랫폼 현장 확인 시:
      - 1차 측정: GPS 측량 앱 사용
      - 측정 과정 사진 촬영 및 GPS 로그 저장

   3. 신청자 이의 제기 시:
      - 공인측량사 측량 실시 (비용 플랫폼 부담)
      - 공인측량 결과가 최종 기준

⑤ 측정 오차 허용 범위

   - 오차 ±10% 이내: 신고 면적 그대로 인정
   - 오차 10~20%: 실측 면적으로 조정 (페널티 없음)
   - 오차 20% 초과: 제18조(분쟁 해결) 절차 진행

⑥ 대체 기준: 판매 목표 금액 (선택)

   재배 면적 기준 충족이 어려운 경우, 다음 기준으로 대체 가능:

   - 월 최소 판매 목표: 30만 원 이상 달성 가능
   - 제출 서류: 재배 계획서 (품목, 예상 수확량, 판매 단가)
   - 승인 후 3개월 연속 목표 미달 시 재심사

② 검사 시기 및 주기

   - 최초 검사: 가입 신청 시 (6개월 이내 발급된 검사 결과서)
   - 정기 검사: 이후 연 1회 (재인증 시 제출)
   - 특례: 동일 품목 재배 시 2년마다 1회로 완화 가능

③ 검사 비용 부담

   - 원칙: 생산자 부담
   - 지원: 플랫폼이 연 1회 검사 비용 50% 지원 (최대 10만 원)
   - 특전: 우수 등급 판매자는 100% 지원

④ 검사 면제 조건

   다음 중 하나에 해당하면 잔류농약 검사 면제:

   ☑ 유기농 인증 보유 (유효기간 내)
   ☑ 무농약 인증 보유 (유효기간 내)
   ☑ GAP 인증 1등급 보유
   ☑ 농협 로컬푸드 직매장 출하 농가 (출하 증빙 시)

⑤ 검사 미이수 시 처리

   - 최초 가입: 조건부 승인 (해당 품목 판매 3개월 제한)
   - 정기 재인증: 경고 → 1개월 유예 → 미제출 시 판매 중지

⑥ 검사 기관

   다음 공인 기관의 검사 결과만 인정:

   • 국립농산물품질관리원
   • 식품의약품안전처 지정 검사 기관
   • 농협중앙회 농산물검사소

   ※ 기관 리스트 및 연락처는 [별첨4] 참조

제3장  예외 및 특례

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

제7조 (인증 간소화 대상)

다음 각 호에 해당하는 자는 서류 제출 절차를 간소화하되,
최소한의 본인 확인은 반드시 실시한다:

① 정부 인증 스마트팜 운영자

   ├ 제출 서류: 스마트팜 인증서 + 사업자등록증
   ├ 면제 항목: 재배지 사진, 비료 영수증
   ├ 필수 확인: 영상통화로 본인 확인 + 스마트팜 시설 확인
   ├ 진위 확인: 농림축산식품부 스마트팜 등록 현황 조회
   └ 판매 제한: 인증 품목만 판매 가능

② 농협 로컬푸드 직매장 정식 출하자

   ├ 제출 서류: 출하자 증명서 (농협 발급, 3개월 이내)
   ├ 면제 항목: 재배 사실 증빙 자료
   ├ 필수 확인: 농협 담당자에게 전화로 재확인 (녹취)
   │            "○○님이 현재 귀 직매장에 출하 중입니까?"
   ├ 진위 확인: 출하자 증명서 발급 번호 조회
   └ 판매 제한: 로컬푸드 출하 품목과 동일 품목만 판매

③ GAP·유기농·무농약 인증 보유 농가

   ├ 제출 서류: 인증서 사본 (유효기간 내)
   ├ 면제 항목: 잔류농약 검사
   ├ 필수 확인: 재배지 현장 방문 1회 또는 영상통화
   │            (재배지에서 인증 현판 촬영 필수)
   ├ 진위 확인: 국립농산물품질관리원 인증 조회 시스템
   └ 판매 제한: 인증 품목 및 인증 방식에 한정
                예) 유기농 인증 → 유기농 표시로만 판매

④ 간소화의 한계

   상기 인증이 있어도 다음은 반드시 확인:

   ☑ 본인 확인 (신분증 대조)
   ☑ 재배 주체 확인 (본인이 직접 재배 중인지)
   ☑ 인증서 진위 확인 (발급 기관 조회)
   ☑ 재배지 위치 확인 (GPS 또는 주소)

⑤ 간소화 대상자의 특전

   - 심사 기간 단축: 7일 이내 승인 (일반 15일)
   - 수수료 할인: 첫 3개월 수수료 1%p 할인
   - 우수 등급 부여: 가입 즉시 '일반(Blue)' 등급

제8조 (특수 상황별 예외 조건)

① 고령 농업인 (만 65세 이상)

   【허용 사항】
   ├ 서류 제출: 자녀 또는 대리인이 대행 가능
   │            (대리인 위임장 + 신분증 사본 첨부)
   ├ 연락처: 본인 또는 자녀 병기 가능
   └ 현장 확인: 반드시 본인이 재배 중임을 확인

   【면담 방식】 (연령 차별 방지)
   ├ 질문 수: 최대 3개 이내
   ├ 방식: 압박 금지, 편안한 대화 형태
   ├ 질문 예시: "요즘 농사 어떠세요?" (개방형)
   │           "이 작물 언제 심으셨어요?" (시기 확인)
   └ 인정 기준: 전문 용어 몰라도 OK, 대략적 설명 가능하면 인정

   【배려 조치】
   ├ 청각 장애: 자녀가 대신 전달 가능 (부모 동의하에)
   ├ 치매 의심: 이장 또는 이웃 주민 증언으로 대체
   └ 거동 불편: 영상통화 대체 적극 활용

② 공동 경작지 (영농조합법인, 협업 농장 등)

   【허용 조건】
   ├ 가입 주체: 법인 대표자 명의로 가입
   ├ 제출 서류: 법인등기부등본 + 조합원 명부
   ├ 판매 권한: 법인 대표자만 상품 등록
   ├ 정산: 법인 계좌로 일괄 입금
   └ 확인: 법인 사업장 현장 확인 (조합원 면담)

③ 임차 농지 경작자

   【허용 조건】
   ├ 계약서: 임대차 계약서 제출 (잔여 기간 3년 이상)
   ├ 동의서: 임대인 동의서 필수 (온라인 판매 동의 포함)
   ├ 확인 절차: 임대인에게 사실 확인 전화 (녹취)
   │           "○○님이 귀하의 토지에서 농사를 짓고 있습니까?"
   │           "온라인 판매에 동의하십니까?"
   └ 재인증: 임차 기간 만료 6개월 전 재인증 필수

④ 재배지 사진 촬영 불가 (동절기, 휴경기 등)

   【조건부 승인 요건】

   (1) 제출 자료
       - 대체 자료: 전년도 출하증명서 (최근 1년 이내 발급)
       - 발급 기관: 농협, 산지유통인, 도매시장 등 공신력 있는 기관
       - 출하 품목과 신청 품목 일치 확인
       - 추가 서약: "차기 작기 시작 시 재배지 현장 사진을 제출하며,
                    미제출 시 판매 중지에 동의합니다" (자필 서명)

   (2) 판매 제한
       - 허용 품목: 전년도 출하증명서에 기재된 품목만
       - 신규 품목 추가 불가
       - 판매 한도: 전년도 월평균 출하량의 120%까지
         예) 전년도 월평균 토마토 100kg 출하
             → 올해 월 최대 120kg까지 판매 가능

   (3) 조건 이행 시점
       - 작기 시작 기준: 파종 또는 정식 완료 후 14일
       - 통보 의무: 생산자가 플랫폼에 "작기 시작" 통보
       - 제출 기한: 통보일로부터 7일 이내 현장 사진 제출
       - 예상일 신고: 신청 시 "작기 시작 예상일" 입력 (예: 3월 15일)

   (4) 조건 미이행 시 처리
       - D+14일: 1차 독촉 알림
       - D+30일: 2차 독촉 + 판매 일시 중지
       - D+60일: 계약 해지
       - 예외: 기상 이변 등 불가항력 사유는 소명 후 연장

   (5) 최대 승인 기간
       - 조건부 승인일로부터 최대 6개월
       - 6개월 내 작기 시작되지 않으면 계절적 특성 고려 연장 가능

⑤ 소량 다품목 재배 농가

   【허용 조건】
   ├ 면적 기준: 총 재배 면적 300㎡ 이상이면 허용
   │           예) 100㎡ 상추 + 100㎡ 토마토 + 100㎡ 고추 = 300㎡ ○
   ├ 입증 방법: 품목별 구역 구분 사진 제출
   └ 판매 한도: 품목당 월 판매량 제한 (과다 판매 방지)

⑥ 희귀 품목 재배자 (산나물, 약용작물 등)

   【완화 기준】
   ├ 최소 면적: 50㎡까지 허용
   ├ 추가 증빙: 종자 구매 영수증 또는 재배 교육 이수증
   ├ 모니터링: 월 판매량 기준 재배 가능량 추정
   └ 검증: 희귀 품목 전문가 자문 (필요 시)

제9조 (5단계 심사 프로세스)

생산자 인증 심사는 다음 5단계 프로세스로 진행한다:

┌──────────────────────────────────────────┐
│  STEP 1: 신청 (D-Day)                                │
├──────────────────────────────────────────┤
│  • 신청자 수행 사항:                                 │
│    - 판매회원 가입 신청서 작성                       │
│    - 필수 서류 온라인 제출                           │
│      ☑ A그룹(재배 주체) 서류 업로드                  │
│      ☑ B그룹(재배 사실) 자료 업로드                  │
│      ☑ C그룹(안전성) 서약서 제출                     │
│                                                      │
│  • 시스템 자동 검수:                                 │
│    - 서류 누락 여부 즉시 알림                        │
│    - GPS 좌표 자동 추출                              │
│    - 중복 계정 자동 조회                             │
│                                                      │
│  • 접수 완료 안내:                                   │
│    - 이메일 + 앱 푸시 알림                           │
│    - 심사 예상 기간 안내 (15일 이내)                 │
└──────────────────────────────────────────┘

         ↓ 영업일 기준 1~3일

┌──────────────────────────────────────────┐
│  STEP 2: 서류 심사 (D+1 ~ D+3)                       │
├──────────────────────────────────────────┤
│  • 담당자 서류 검토 (체크리스트 사용):               │
│    ☑ 사업자등록증 업태·업종 확인                     │
│    ☑ 국세청 홈택스 사업자 상태 조회 (화면 캡처)      │
│    ☑ 농지원부 진위 확인 (발급 기관 조회)             │
│    ☑ GPS 좌표와 농지원부 주소 일치 여부 (±1km)      │
│    ☑ 재배지 사진 품질 및 촬영일 확인                 │
│    ☑ 중복 계정 여부 조회 (반경 500m 내 검색)        │
│    ☑ 품목군 분류 및 재배 면적 기준 확인              │
│                                                      │
│  • 심사 결과 (3영업일 이내 회신):                    │
│    ├ 적격 → STEP 3 진행                             │
│    ├ 보완 필요 → 보완 요청 (기한 7일)               │
│    └ 부적격 → 반려 (구체적 사유 통지)               │
└──────────────────────────────────────────┘

         ↓ 영업일 기준 4~14일

┌──────────────────────────────────────────┐
│  STEP 3: 현장 확인 (D+4 ~ D+14)                      │
├──────────────────────────────────────────┤
│  • 방문 일정 조율:                                   │
│    - 신청자에게 전화 연락                            │
│    - 편의 시간대 협의 (최대 3회 조정 가능)           │
│                                                      │
│  • 현장 방문 (원칙):                                 │
│    - 인원: 2인 1조 (상호 검증 및 안전 확보)          │
│    - 예외: 섬·산간 벽지는 1인 방문 허용              │
│    - 비용: 플랫폼 부담                               │
│           (단, 편도 200km 초과 시 생산자와 협의)     │
│    - 확인 사항:                                      │
│      ☑ 재배지 실사 (GPS 앱으로 위치 대조)            │
│      ☑ 재배 중인 작물 종류 및 상태 확인              │
│      ☑ 농기구, 비료, 농약 보관 현황 점검             │
│      ☑ 본인 면담 (재배 경력, 판매 계획)              │
│      ☑ 재배 규모 측정 (GPS 앱 사용)                  │
│                                                      │
│  • 영상통화 대체 (제한적 허용):                      │
│    - 허용 사유: 섬 지역, 접근 불가 산간, 인력 부족   │
│    - 소요 시간: 최소 10분 이상 (녹화 필수)           │
│    - 확인 방법:                                      │
│      ☑ 본인 얼굴 + 신분증 대조                       │
│      ☑ 재배지 전경 (카메라 360도 회전)               │
│      ☑ 작물 근접 촬영                                │
│      ☑ 농기구·비료 보관 장소                         │
│      ☑ 재배지 입구 표지판 (주소 확인)                │
│      ☑ 실시간 확인: "5걸음 앞으로" 등 즉시 요청      │
│    - 의심 시: 현장 방문 재요청 (거부 시 승인 불가)   │
│                                                      │
│  • 확인 결과서 작성:                                 │
│    - 사진 첨부 (최소 6장) 또는 영상 링크             │
│    - 재배 규모 측정 결과                             │
│    - 종합 의견: 승인 추천 / 보완 필요 / 승인 불가    │
│    - 팀장 결재 후 STEP 4 진행                        │
└──────────────────────────────────────────┘

         ↓ 영업일 기준 15일

┌──────────────────────────────────────────┐
│  STEP 4: 최종 승인 (D+15)                            │
├──────────────────────────────────────────┤
│  • 승인 여부 결정:                                   │
│    ├ 정식 승인: 판매회원 등급 부여                   │
│    │     - 초보(Green): 가입 후 6개월                │
│    │     - 일반(Blue): 6개월~3년                     │
│    │     - 우수(Gold): 3년 이상 + 무클레임            │
│    │                                                  │
│    ├ 조건부 승인: 제한 사항 명시                     │
│    │     - 판매 가능 품목 제한                       │
│    │     - 월 판매 한도 설정                         │
│    │     - 조건 이행 기한 지정                       │
│    │                                                  │
│    └ 승인 거부: 구체적 사유 및 재신청 안내           │
│          - 6개월 후 재신청 가능                      │
│                                                      │
│  • 승인 통지:                                        │
│    - 이메일 + 앱 푸시 알림 + SMS                     │
│    - 판매회원 약관 최종 동의 요청                    │
│    - 정산 계좌 등록 안내                             │
│                                                      │
│  • 승인 후 자동 조치:                                │
│    - 생산자 프로필 페이지 자동 생성                  │
│    - 재배지 위치 지도 표시                           │
│    - 상품 등록 권한 활성화                           │
│    - 우대 혜택 자동 적용 (간소화 대상자)             │
└──────────────────────────────────────────┘

         ↓ 승인 후 지속

┌──────────────────────────────────────────┐
│  STEP 5: 사후 관리 (승인 후 지속)                    │
├──────────────────────────────────────────┤
│  • 정기 재인증 (제15조 참조):                        │
│    - 연 1회, 가입 기념일 기준                        │

제10조 (현장 확인 방법)

① 현장 확인의 원칙

   생산자 인증 시 재배 사실 확인을 위해 재배지 현장 확인은 필수이며,
   방문 또는 영상통화 중 선택한다.

② 현장 방문 기준

   【인원 구성】
   - 원칙: 2인 1조
   - 목적: 상호 검증 및 안전 확보
   - 예외: 섬 지역, 산간 벽지 등은 1인 방문 허용

   【비용 부담】
   - 플랫폼 부담 (교통비, 인건비)
   - 단, 편도 200km 초과 시 생산자와 협의

   【확인 항목】
   1. 재배지 위치 확인 (GPS 대조)
   2. 재배 중인 작물 종류 및 상태
   3. 농기구, 비료, 농약 보관 현황
   4. 재배 규모 측정 (GPS 앱)
   5. 본인 면담 (재배 경력, 판매 계획)

   【기록 보관】
   - 확인 과정 사진 6장 이상
   - 확인 결과서 작성 (담당자 서명)
   - GPS 로그 저장
   - 5년간 보관

③ 영상통화 대체 조건

   다음 요건을 모두 충족하는 경우에만 영상통화로 대체 가능:

   【허용 사유】
   - 섬 지역 (선박 이용 필수)
   - 접근 불가 산간 지역 (대중교통 없음)
   - 플랫폼 인력 부족 (일시적)

   【사전 준비】
   - 생산자가 영상통화 방식에 동의
   - 통화 시간: 최소 10분 이상
   - 통화 장소: 재배지에서 실시 (자택 불가)

   【확인 절차】
   1. 본인 얼굴 + 신분증 대조
   2. 재배지 전경 (카메라를 천천히 360도 회전)
   3. 작물 근접 촬영 (뿌리, 줄기, 열매 등)
   4. 농기구·비료 보관 장소
   5. 재배지 입구 표지판 (주소 확인)
   6. 실시간 지시 이행 확인: "지금 5걸음 앞으로 이동해주세요"

   【의심 시 조치】
   - 미리 준비한 영상 재생 의심 → 현장 방문 재요청
   - 타인 재배지 의심 → 현장 방문 재요청
   - 재요청 거부 → 승인 불가

   【녹화 보관】
   - 전체 영상통화 녹화 (동의 사전 고지)
   - 5년간 보관

제12조 (생산자의 책임)

생산자는 다음 각 호의 책임을 부담한다:

① 서류 진실성 책임

   - 제출한 모든 서류의 진실성 및 적법성에 대한 전적인 책임
   - 허위 서류 제출 시: 계약 해지 + 민·형사상 책임

② 재배 현황 변경 통보 의무

   다음 사항이 변경된 경우 즉시 플랫폼에 통보:

   - 재배지 이전 (14일 이내 통보)
   - 재배 품목 변경 (신규 품목 판매 전 통보)
   - 재배 중단 (중단일로부터 7일 이내)
   - 재배 면적 변경 (30일 이내)

③ 원산지 표시 의무

   - 「농수산물의 원산지 표시에 관한 법률」 준수
   - 위반 시 형사 책임 (최대 징역 7년 또는 벌금 1억 원)
   - 상품 페이지에 원산지 정확히 표시

④ 정기 재인증 의무

   - 가입 기념일 기준 연 1회 재인증
   - 기한 경과 시 판매 자동 중지

⑤ 판매 한도 준수

   - 재배 가능량 범위 내 판매
   - 플랫폼 요청 시 추가 증빙 제출

제13조 (플랫폼의 책임과 면책 요건)

① 플랫폼의 기본 책임

   - 신청자가 제출한 서류를 성실히 심사할 의무
   - 부적격 판정 시 구체적 사유 및 개선 방법 안내
   - 정당한 사유 없이 승인을 지연하거나 자의적 판단 금지
   - 심사 과정에서 취득한 개인정보 및 영업 비밀 보호

② 면책 요건 (체크리스트)

   플랫폼이 다음 5개 항목을 모두 이행한 경우 "상당한 주의 의무"를
   다한 것으로 간주하며, 허위 생산자가 승인되더라도 면책된다:

   ☑ A. 사업자등록증 진위 확인
       - 국세청 홈택스 사업자 상태 조회
       - 조회 결과 화면 캡처 보관 (5년)

   ☑ B. 농지 소유/임차 확인
       - 농지원부: 정부24 또는 발급 기관 진위 확인
       - 임대차계약서: 임대인 본인 확인 전화 (녹취)

   ☑ C. GPS 좌표 검증
       - 사진 메타데이터 추출
       - 농지원부 주소와 좌표 ±1km 이내 일치 확인

   ☑ D. 재배 사실 확인 (다음 중 하나)
       - 방법1: 현장 방문 (사진 6장 이상 촬영)
       - 방법2: 영상통화 (최소 10분, 녹화 보관)
       - 방법3: 제3자 확인서 (이장, 농협 조합장 등)

   ☑ E. 심사 기록 보관
       - 심사 일지 작성 (검토 일시, 담당자, 판단 근거)
       - 모든 증빙 자료 5년간 보관

③ 면책 조건 충족 시 효과

   위 5개 항목을 모두 이행하고, 다음 요건을 충족하면 플랫폼은 면책:

   1. 심사 당시 제출된 자료에서 위조·허위 정황을 발견할 수 없었음
   2. 허위 생산자로 판명 후 즉시 판매중지 조치 (24시간 이내)
   3. 피해 소비자에게 전액 환불 조치 (7일 이내)

④ 면책 불가 사유 (연대 책임)

   다음 중 하나에 해당하면 플랫폼은 연대 책임을 부담:

   1. 필수 이행 항목(A~E) 중 하나라도 누락
   2. 명백한 위조 정황을 인지하고도 승인 (예: 사진 합성 흔적)
   3. 소비자 또는 제3자의 신고를 받고도 재조사하지 않음
   4. 허위 생산자 판명 후에도 판매 지속 허용

⑤ 플랫폼 책임 범위

   플랫폼이 면책되지 못하는 경우 다음 범위에서 책임:

   - 소비자 직접 손해: 구매 금액 + 배송비 전액 배상
   - 간접 손해: 입증된 범위 내 배상 (상한: 구매 금액의 3배)
   - 위자료: 인정되지 않음 (단, 플랫폼의 고의가 입증된 경우 예외)

   - 재배 품목

   - 통지 범위: 개인정보 보호 범위 내에서 조치 결과 안내

③ 환불 청구 권리

제16조 (이상 판매 모니터링)

① 재배 가능량 산정

   - 기준: 품목별 표준 수확량 DB 활용 (농촌진흥청 자료)
   - 계산식: 재배 면적 × 품목별 ㎡당 수확량 × 안전계수(0.8)
   - 예시: 300㎡ 토마토 × 3kg/㎡ × 0.8 = 720kg/작기

② 이상 판매 기준

   ┌──────────────────────────────┐
   │ 황색 경고: 월 판매량이 기준량의 120% 초과 │
   └──────────────────────────────┘

   - 자동 알림: "판매량이 재배 예상량보다 많습니다.
                추가 증빙 제출 부탁드립니다."
   - 요청 자료: 추가 재배지 사진 또는 출하 증빙
   - 제출 기한: 7일 이내
   - 미제출 시: 경고 누적 (3회 시 재조사)

   ┌──────────────────────────────┐
   │ 적색 경고: 월 판매량이 기준량의 150% 초과 │
   └──────────────────────────────┘

   - 즉시 판매 일시 중지
   - 현장 재확인 (영상통화 또는 방문)
   - 소명 요청: "어떻게 이만큼 수확이 가능했는지 설명"
   - 소명 인정: 판매 재개 + 재배 가능량 상향 조정
   - 소명 불인정: 계정 정지 + 상세 조사

③ 정당한 다수확 인정 사례

   다음 증빙 제출 시 정당한 다수확으로 인정:

   ☑ 추가 재배지 확보 (농지원부, 임대차계약서)
   ☑ 스마트팜 기술 적용 (시설 사진, 관제 시스템 화면)
   ☑ 품종 개량종 재배 (종자 구매 영수증, 품종 설명서)
   ☑ 저장 농산물 판매 (저온 저장고 보유 증빙, 사진)
   ☑ 공동 출하 (다른 농가와 공동 판매 계약서)

④ 부정 판매 적발 시 조치

   타인 재배 농산물 구매 후 재판매 확인 시:

   - 즉시 계정 영구 정지
   - 기판매 상품 전량 환불
   - 손해배상 청구 (플랫폼 및 소비자)
   - 유사 플랫폼에 정보 공유

⑤ 예외: 저장 농산물

   사과, 배, 감자, 양파 등 장기 저장 가능 품목:

   - 월 판매량이 아닌 연간 누적 판매량 기준
   - 저장 시설 보유 증빙 필수
   - 작기별 총 수확량 범위 내 판매

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

제18조 (분쟁 유형별 판단 기준)

본 조는 실제 운영 중 발생 가능한 주요 분쟁 유형과
각 유형별 판단 기준을 명시한다.

【분쟁 유형 1】 서류 진위 논란
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
상황: 신청자가 제출한 농지원부가 위조 의심

판단 절차:
├ 1단계: 발급 기관(읍·면사무소)에 진위 확인 요청
├ 2단계: 진위 확인 불가 시 신청자에게 원본 제출 요청 (3일)
├ 3단계: 원본 미제출 또는 위조 확인 시 즉시 반려
└ 결과: 위조 확인 시 영구 가입 차단 + 수사기관 통보

책임 소재:
• 위조 서류 제출: 신청자 전적 책임 (형사 처벌 대상)
• 플랫폼: 발급 기관 확인 의무만 부담 (확인 후 면책)


【분쟁 유형 2】 재배 규모 부풀리기
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
상황: 재배지 300㎡라고 신청했으나 실제는 200㎡

판단 기준:
├ 측정 방법:
│  - 1차: GPS 측량 앱 (현장 방문 시)
│  - 2차: 공인측량사 측량 (이의 제기 시, 비용 플랫폼 부담)
│
├ 허용 오차:
│  - ±10% 이내: 신고 면적 인정
│  - 10~20%: 실측 면적으로 조정 (페널티 없음)
│  - 20% 초과: 소명 절차 진행
│
├ 소명 인정 사유:
│  ☑ 일부 구역이 휴경 중이었으나 곧 재배 예정 (증빙 필요)
│  ☑ 타인 소유 인접 토지를 착오로 포함 (경계 불명확)
│  ☑ 측량 방법의 차이 (보폭법 vs GPS, ±20% 오차 허용)
│
├ 소명 불인정:
│  ☒ 단순 착오, 대충 측정
│  ☒ 증빙 자료 없는 구두 해명
│
└ 결과:
   - 소명 인정: 실측 면적 기준 조건부 승인
   - 소명 불인정: 승인 거부 + 6개월 재신청 금지
   - 오차 30% 이상 + 고의: 영구 가입 차단

책임 소재:
• 측정 오차 20% 이내: 양측 협의로 해결
• 고의적 허위 (30% 이상): 신청자 책임


【분쟁 유형 3】 타인 재배지 도용
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
상황: A가 B의 재배지를 자신의 것처럼 허위 신청

판단 절차:
├ 1단계: GPS 좌표와 농지원부 명의 일치 여부 확인
├ 2단계: 현장 방문 시 본인 확인 (신분증 대조)
├ 3단계: 본인 부재 시 재배지 소유주/임차인 확인 전화
├ 4단계: 농지원부 실소유자(B)에게 사실 확인
└ 결과: 타인 재배지 도용 확인 시 즉시 반려 + 영구 차단

책임 소재:
• 도용 행위: 신청자(A) 전적 책임 (형사 고발 가능)
• 피해자(B): 플랫폼에 신고 시 즉시 조치


【분쟁 유형 4】 중간 유통업자 위장
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
상황: 도매시장에서 농산물을 구매하여 재판매하려는 유통업자

판단 증거:
├ 재배지 현장 방문 시 재배 흔적 없음
├ 비료·농약 구매 내역 없음
├ 사업자등록증 업종이 "도매업" 또는 "소매업"
├ 판매량이 신고 재배 면적 대비 과다 (월 150% 초과)
└ 재배 관련 질문에 대한 답변 불가

판단 기준:
- 위 증거 중 2개 이상 해당 시 추가 소명 요청
- 소명 불충분 시 승인 거부

사후 모니터링:
- 승인 후에도 월 판매량 지속 모니터링
- 이상 징후 발견 시 재조사 (제16조 참조)

책임 소재:
• 위장 입점: 신청자 전적 책임 (자격 박탈 + 손해배상)


【분쟁 유형 5】 고령 농업인 대리 신청 논란
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
상황: 자녀가 부모(고령 농업인) 명의로 신청했으나,
      실제 재배는 자녀가 주도

정당한 대리 범위:
☑ 자녀가 온라인 신청서 작성 대행
☑ 자녀가 재배지 사진 촬영 대행
☑ 자녀가 플랫폼 연락 창구 역할
☑ 단, 실제 재배는 부모(명의자)가 주도

부당한 명의 대여:
☒ 부모는 전혀 재배에 관여하지 않음
☒ 자녀가 100% 재배하고 부모 이름만 사용
☒ 부모가 재배 사실조차 모름

구분 방법 (연령 차별 방지):

▶ 방법 1: 관찰 확인 (질문 없이)
   - 재배지 안내 시 부모가 직접 설명하는지
   - 작물 상태를 보고 부모가 자연스럽게 언급하는지
   - 부모가 농기구 위치, 물주는 장소 등을 아는지

▶ 방법 2: 간단한 확인 질문 (압박 금지, 최대 3개)
   - "요즘 농사 어떠세요?" (개방형)
   - "이 작물 언제 심으셨어요?" (시기 확인)
   - "물은 어떻게 주세요?" (방법 확인)
   → 전문 용어 몰라도 OK, 대략적 설명 가능하면 인정

▶ 방법 3: 자녀와의 역할 구분 확인
   - "아드님(따님)은 어떻게 도와주시나요?"
   - 부모: "짐 나르는 것", "온라인 판매" 등 보조 역할
   - 자녀: "부모님이 재배하시고 제가 포장만"
   → 역할이 명확히 구분되면 인정

고령자 배려:
- 청각 장애: 자녀가 대신 전달 가능 (부모 동의하에)
- 치매 의심: 이장 또는 이웃 주민 증언으로 대체
- 거동 불편: 영상통화 적극 활용

판단 결과:
├ 인정: 부모가 재배 주도, 자녀는 보조
│      → 승인 (연락처는 자녀 병기)
├ 불인정: 부모가 재배 전혀 모름, 자녀가 100% 주도
│        → 승인 거부 + 자녀 명의로 재신청 안내
└ 애매: 부모 고령으로 일부 관여 어려움
        → 조건부 승인 (3개월 후 재확인)

책임 소재:
• 정당한 대리: 문제없음 (부모가 실제 재배자)
• 명의 대여: 부모+자녀 공동 책임


【분쟁 유형 6】 임차 농지 계약 분쟁
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
상황: 임대인이 "임차인에게 재배 허락한 적 없다"고 주장

판단 절차:
├ 1단계: 임대차 계약서 확인
│        - 계약 목적: "농지 임대" 명시 여부
│        - 계약 기간: 잔여 기간 3년 이상 여부
│
├ 2단계: 임대인에게 사실 확인 전화 (녹취)
│        - "○○님이 귀하의 토지에서 농사를 짓고 있습니까?"
│        - "온라인 판매에 동의하십니까?"
│
├ 3단계: 임대인 부인 시 신청자에게 소명 기회 (7일)
│        - 추가 증빙 자료 제출
│        - 임대인 설득 후 동의서 재제출
│
└ 결과: 임대인 동의 없으면 승인 불가

책임 소재:
• 계약서 위조: 신청자 책임 (형사 처벌)
• 임대인 번복: 민사 분쟁 (플랫폼 개입 불가)
• 플랫폼: 임대인 확인 의무만 부담


【분쟁 유형 7】 현장 확인 거부
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
상황: 신청자가 "바쁘다", "개인정보 침해다"며 현장 확인 거부

원칙: 현장 확인은 필수 절차 (약관 명시)

대안 제시:
├ 일정 조율: 최대 3회까지 연기 가능
├ 영상통화 대체: 실시간 재배지 확인 (최소 10분)
└ 방문 시간 조정: 신청자 편의 시간대 협의

최종 거부 시:
- 승인 불가 안내
- 사유: "본 플랫폼은 신뢰 기반 거래를 위해
         현장 확인을 필수로 합니다"

예외: 정부 인증 보유자
- 스마트팜, GAP, 유기농 인증 보유자는
  서류 + 영상통화로 갈음 가능 (제7조)

책임 소재:
• 정당한 거부 사유 없음 → 승인 불가 (신청자 불이익 감수)
• 플랫폼의 과도한 요구 → 플랫폼 재검토

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

제19조 (소명 절차)

① 소명 기회 부여

   플랫폼의 부적격 또는 조건부 승인 결정에 대해 신청자는
   소명 기회를 요청할 수 있다.

② 소명 요청 방법

   - 방법: 이메일 또는 고객센터 전화
   - 기한: 결정 통지일로부터 7일 이내

③ 소명 자료 제출

   - 제출 기한: 소명 요청 후 14일 이내
   - 제출 방법: 온라인 업로드 또는 우편
   - 인정 자료: 객관적 증빙 (사진, 계약서, 공문서 등)

④ 소명 심사

   - 심사 기한: 자료 제출일로부터 7일 이내
   - 심사 주체: 운영팀 + 팀장 결재
