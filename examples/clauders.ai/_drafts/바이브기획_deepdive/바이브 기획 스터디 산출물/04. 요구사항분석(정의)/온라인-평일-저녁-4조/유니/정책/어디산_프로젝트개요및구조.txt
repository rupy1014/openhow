【 제안 브랜드명 5가지 】

1. 🌾 FARMLINK (팜링크)
   ─────────────────────────────────────
   컨셉: Farm + Link = 농장과 소비자를 연결하다
   의미: 직거래를 통한 원활한 연결
   타겟: 20~40대 도시형 소비자 (국제적 느낌)
   브랜드톤: 현대적, 신뢰감, 기술 중심

2. 🥬 로컬밥상 (Local Bapsang)
   ─────────────────────────────────────
   컨셉: 한국식 "밥상"의 온정감 + 로컬
   의미: 지역 농산물로 채운 밥상 문화
   타겟: 30~60대 한국 소비자 (정감 있는 느낌)
   브랜드톤: 따뜻함, 신뢰, 한국성

3. 🌱 어디산 (Eodi San)
   ─────────────────────────────────────
   컨셉: "어디서 나왔나?" → 투명성 강조
   의미: 농산물의 출처를 명확히 하다
   타겟: 20~50대 보편적 소비자 (한국식 느낌)
   브랜드톤: 명확함, 신선함, 로컬 정체성

4. 🚜 HARVEST (하비스트)
   ─────────────────────────────────────
   컨셉: Harvest = 수확의 순간을 공유하다
   의미: 생산자의 노고와 신선함을 담다
   타겟: 20~40대 상향층 (프리미엄 느낌)
   브랜드톤: 고급, 신뢰감, 스토리텔링

5. 🌻 들녘마켓 (Deulnyeok Market)
   ─────────────────────────────────────
   컨셉: 들녘(들판) + 마켓 = 자연스러운 시장
   의미: 농장 직거래의 자연스러움
   타겟: 30~70대 전 연령층 (친근한 느낌)
   브랜드톤: 친근함, 정직함, 지역성

1. 서비스 개요
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【 서비스명 】
어디산 (Eodi San) - Local Farmer Direct Market Platform

【 서비스 슬로건 】
"투명한 출처, 신선한 맛, 신뢰하는 거래"
- 어디서 나왔는지 명확히 알고
- 생산자에게서 직접 받고
- 안심하고 구매하다

【 서비스 정의 】
로컬 농산물 생산자와 소비자를 직접 연결하는 플랫폼입니다.
중간 유통업자를 제거함으로써 소비자는 신선하고 저렴한 제품을,
생산자는 정당한 대가를 받을 수 있는 공정거래 시스템을 구축합니다.

【 핵심 가치 】

1️⃣  투명성 (Transparency)
   └─ 생산자 정보 공개, 재배지 공개, 생산 과정 추적

2️⃣  신선성 (Freshness)
   └─ 수확 → 배송 최단 시간, 포장 재배지 직송

3️⃣  공정성 (Fairness)
   └─ 생산자 정당 대가 보장, 중간마진 제거

4️⃣  신뢰성 (Trust)
   └─ 엄격한 생산자 인증, 투명한 원산지 표시

【 타겟 고객 】

주고객층 (Primary)
├─ 연령: 28~50세
├─ 성별: 남 35% / 여 65% (주요 식재료 구매자)
├─ 소득: 월 300만 원 이상 (상향층)
├─ 관심사: 건강식, 친환경, 투명성
└─ 구매빈도: 주 1~2회 (정기 구독 예상)

보조고객층 (Secondary)
├─ 로컬 음식문화 관심층 (20~40대 MZ세대)
├─ 친환경 농산물 선호층 (ESG 관심층)
├─ 지역 경제 지원 고객 (지역사랑 실천층)
└─ 소상공인/식당/급식 (B2B 구매자)

【 차별화 포인트 】

비 어디산 (기존 유통) vs 어디산 (플랫폼)
────────────────────────────────────────────

① 가격
   마트: 1kg 당 8,000원 (농가 1,500원 + 중간마진 60%)
   어디산: 1kg 당 4,500원 (농가 4,000원 + 수수료 12%)

② 신선도
   마트: 3~7일 전 수확 (냉장 유통)
   어디산: 1~2일 전 수확 (직송)

③ 추적성
   마트: 원산지만 표시
   어디산: 생산자 정보, 재배지, 재배 방법, 인증 상태 공개

④ 관계성
   마트: 일회 구매
   어디산: 생산자와 소비자의 지속적 관계

【 수익 모델 】

1. 수수료 (Commission)
   └─ 거래액의 12% (신용카드 수수료 포함)
   └─ 생산자 부담 7% + 소비자 부담 5%

2. 프리미엄 서비스 (Premium)
   └─ 정기 구독 (월 49,900원/박스): 수수료 0% + 배송비 무료
   └─ 판매자 광고: 월 50,000원 ~ (수수료 추가 할인)

3. B2B 벌크 판매
   └─ 식당, 급식업체 계약: 거래액의 10%
   └─ 최소 주문 100kg 이상

4. 데이터 수익화 (향후)
   └─ 농산물 가격 정보 공유 (생산자용 분석 도구)
   └─ 소비 트렌드 분석 리포트 (농협/정부기관용)

【 주요 기능 】

생산자 앱
├─ 생산자 인증 시스템 (POLICY-001 기반)
├─ 상품 등록 및 재고 관리
├─ 주문 관리 및 배송 수수료 자동 계산
├─ 정산 시스템 (주 1회 자동 송금)
├─ 고객 평가 및 리뷰 관리
└─ 판매 분석 대시보드

소비자 앱
├─ 검색 / 필터링 (지역, 품목, 인증상태)
├─ 생산자 프로필 / 재배지 지도 표시
├─ 정기 구독 서비스
├─ 결제 / 배송 추적
├─ 직접 연락 기능 (1:1 문의)
└─ 커뮤니티 (요리 팁, 레시피 공유)

어드민 대시보드
├─ 생산자 인증 관리 (자동화 + 수동 심사)
├─ 거래 모니터링 (이상 거래 탐지)
├─ 정산 관리
├─ 고객 지원 / 분쟁 조정
└─ 데이터 분석 및 리포팅
