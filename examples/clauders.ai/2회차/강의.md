## 📦 0:20 — GitHub 레포 생성 + 코드 올리기

### 배포 흐름 이해

```
내 코드 → GitHub (코드 보관함) → Vercel (자동 운영)
```

**왜 GitHub을 거치나요?**
Vercel이 GitHub에서 직접 코드를 가져다 실행해요.
GitHub에 올리는 순간부터, 코드를 바꿀 때마다 Vercel이 자동으로 업데이트해줘요!

### Task A: GitHub 레포 만들기

1. `github.com` 접속 → 로그인 (계정 없으면 지금 바로 회원가입)
2. 오른쪽 상단 **`+`** 클릭 → **New repository**
3. Repository name: `my-ai-chat` (또는 원하는 이름)
4. **Private** 선택 ← API 키 관련 코드이므로 비공개
5. **Create repository** 클릭
6. 생성된 페이지에서 **HTTPS 주소** 복사

> 예: `https://github.com/내아이디/my-ai-chat.git`

## ✨ 1:20 — UX 고도화

### 현재 서비스의 아쉬운 점들

지금 만든 서비스를 직접 써보면 이런 불편함이 있어요:

- 메시지 보내고 나서 답 오는 동안 아무 표시가 없음 → "고장난 건가?" 싶음
- 에러 나면 그냥 멈춰버림 → 사용자가 뭔지 모름
- 채팅이 길어지면 스크롤이 자동으로 내려가지 않음
- 핸드폰에서 키보드가 올라오면 입력창이 가려짐

### Task D: 한 번에 다 고치기

Claude Code에 붙여넣으세요:

<div style="background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); border-radius: 16px; padding: 3px; margin: 1.5em 0;">
<div style="background: #0a0a1a; border-radius: 14px; padding: 20px 24px; color: #e0e0e0; line-height: 1.7; font-family: monospace;">
  <p style="margin: 0 0 8px 0; color: #a78bfa;">📋 Claude Code에 붙여넣기</p>
  <p style="margin: 0; white-space: pre-wrap; color: #e0e0e0;">app/page.tsx 를 아래 4가지를 개선해줘:

1. 로딩 상태
   - AI가 답변 생성 중일 때 "..." 타이핑 애니메이션 표시
   - 로딩 중에는 전송 버튼 비활성화

2. 에러 처리
   - API 호출 실패 시 "잠시 후 다시 시도해주세요" 메시지 표시
   - 빨간 배경으로 구분되게

3. 자동 스크롤
   - 새 메시지가 올 때마다 자동으로 아래로 스크롤
   - useRef와 useEffect 활용

4. 모바일 최적화
   - 입력창이 화면 하단에 고정되도록
   - 키보드가 올라와도 입력창이 가려지지 않게
   - 버튼 크기를 모바일에서 탭하기 쉽게</p>

</div>
