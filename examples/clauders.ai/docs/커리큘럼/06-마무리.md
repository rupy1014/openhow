## 개념 용어집

| 용어 | 설명 | 비유 |
|------|------|------|
| 프론트엔드 | 사용자가 보는 화면 | 식당 홀 |
| 백엔드 | 눈에 안 보이는 서버 로직 | 식당 주방 |
| API | 서비스 간 대화하는 방법 | 주문서 양식 |
| 환경변수 | 비밀 설정값 (.env.local) | 금고 속 열쇠 |
| 배포 | 인터넷에 서비스 공개 | 가게 오픈 |
| Git | 코드 버전 관리 | 게임 세이브 |
| GitHub | 코드 저장 클라우드 | 코드용 구글 드라이브 |
| Vercel | 무료 배포 + 호스팅 서비스 | 임대 건물 |
| OpenRouter | 여러 AI API를 하나로 모은 서비스 | AI 모델 백화점 |
| 시스템 프롬프트 | AI에게 캐릭터를 알려주는 설명서 | 역할극 지문 |
| localhost | 내 컴퓨터 주소 | 집 내부 주소 |
| npm run dev | 개발 서버 실행 | 가게 내부에서만 보는 미리보기 |
| git push | GitHub에 코드 올리기 | 클라우드에 백업 |
