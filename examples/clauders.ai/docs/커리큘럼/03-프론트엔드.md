# PART 3: 프론트엔드 — 채팅 화면 만들기

> "Claude Code에게 도면을 주고 집을 짓게 하는 단계예요."

**⏱ 예상 시간**: 2시간
**🎯 목표**: 브라우저에서 보이는 채팅 화면 완성 (아직 AI 연결 전)

---

## 개념 먼저: 프론트엔드가 뭐예요?


> 지금은 화면만 만들어요. AI가 실제로 답하는 건 PART 4에서요.

  일단 예쁜 채팅창이 먼저 있어야, 거기에 AI를 연결할 수 있어요.


---

## ✅ Task 3-1: 프로젝트 생성

터미널을 열고, 작업할 폴더로 이동하세요.

```bash
# Mac: 바탕화면으로 이동
cd ~/Desktop

# Windows: 바탕화면으로 이동
cd %USERPROFILE%\Desktop
```

이제 Claude Code를 실행하세요:
```bash
claude
```

Claude Code가 실행되면 아래 내용을 **복사해서 붙여넣기** 하세요:


:::tip 📋 Claude Code에 붙여넣기 (그대로 사용 가능)
Next.js 14 + TypeScript + Tailwind CSS로 AI 캐릭터 채팅 앱 프로젝트를 만들어줘.

프로젝트 이름: my-ai-chat

설정:
- App Router 사용
- TypeScript 사용
- Tailwind CSS 포함
- ESLint 포함
- 프로젝트 생성 후 npm install까지 완료해줘
:::


Claude Code가 명령어를 실행하도록 승인해주세요 (y 또는 Enter).

완료되면 확인:
```bash
cd my-ai-chat
ls
```

`package.json`, `app/`, `tailwind.config.ts` 같은 파일이 보이면 성공!

---

## ✅ Task 3-2: 개발 서버 실행

```bash
npm run dev
```

터미널에 `localhost:3000` 이 표시되면 성공!
브라우저에서 `http://localhost:3000` 접속 → Next.js 기본 화면이 보여요.


:::info
localhost:3000 = 내 컴퓨터 주소 3000번 포트

다른 사람은 아직 못 봐요. 배포는 PART 5에서 해요.
:::


> 💡 개발 서버를 켜놓은 채로 다음 Task를 진행하세요.
> 코드가 바뀔 때마다 브라우저가 자동으로 업데이트돼요.

---

## ✅ Task 3-3: 채팅 UI 메인 페이지 만들기

터미널 **새 창**을 열고, 프로젝트 폴더에서 Claude Code 다시 실행:

```bash
cd ~/Desktop/my-ai-chat
claude
```

아래 내용을 붙여넣기하세요. **캐릭터 정보를 내 것으로 교체하세요!**


:::tip 📋 Claude Code에 붙여넣기 (★ 대괄호 안 내용을 내 것으로 교체 ★)
app/page.tsx를 아래 기준으로 완전히 새로 만들어줘.

캐릭터 정보:
- 이름: [도비] ← 내 캐릭터 이름으로 교체
- 한 줄 설명: ["Harry Potter의 충직한 하우스엘프"] ← 내 것으로 교체
- 대표 이모지: [🧝] ← 내 캐릭터에 맞는 이모지로 교체

디자인:
- 전체 배경: [어두운 남색 (#0f172a)] ← 원하는 색으로 교체
- 채팅 영역: 화면 가득, 스크롤 가능
- AI 메시지: 왼쪽, 회색 말풍선 (rounded-2xl)
- 내 메시지: 오른쪽, [보라색] 말풍선 (rounded-2xl)
- 하단: 흰색 입력 필드 + 전송 버튼

기능 (지금은 UI만, AI 연결 아직):
- useState로 messages 배열 관리
- 전송 버튼/Enter → 내 메시지가 messages에 추가, 입력창 초기화
- AI 답변은 "..." 로딩 표시만
- 화면 처음 열면 AI 캐릭터의 첫 인사말 자동 표시
- 새 메시지 올 때 자동 스크롤 아래로

TypeScript 타입:
type Message = {
  role: 'user' | 'assistant';
  content: string;
  timestamp: Date;
}
:::


작업이 완료되면 `http://localhost:3000` 새로고침!

### 마음에 안 드는 부분이 있으면?

Claude Code에 바로 요청하세요. 예시:

```
말풍선 모서리를 더 둥글게 해줘
헤더에 그라데이션 배경 추가해줘
글자 크기를 조금 더 크게 해줘
캐릭터 이모지를 헤더에 더 크게 표시해줘 (64px)
채팅 영역 배경을 약간 더 밝게 해줘
```

이렇게 한 줄씩 구체적으로 요청하면 빠르게 수정돼요.
