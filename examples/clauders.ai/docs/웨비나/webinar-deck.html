Architecture

지금 어디까지 왔나

고객 (카카오톡)   ↕ 메시지 사장님 카카오톡 (서브폰)   ↕ 알림 캡처 Android SoulEngine   ├─ 의도 분류 + 프롬프트 조립   ├─ OpenRouter API (Gemini Flash)   ├─ 도구 실행 (예약, 입금확인, 전화)   └─ RemoteInput 자동 답장   ↕ HTTPS Cloudflare Worker + D1   ├─ 상태 저장 + 결제/파트너 API   └─ 관리자 대시보드 API   ↕ Web Admin (tokdaeri.com)
