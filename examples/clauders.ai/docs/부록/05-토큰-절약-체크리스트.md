**토큰이 새는지 볼 때는 이것만 체크해.**

길게 읽기 싫을 때 보는 압축판이야.
자세한 설명은 [Claude 토큰 낭비 막기](../레시피/05-claude-토큰-낭비-막기.md)에 있어.

---

## 지금 바로 뭘 보면 돼?

- [ ] 주제가 바뀌었는데 `/clear` 안 했어?
- [ ] 대화가 긴데 `/compact` 안 했어?
- [ ] 쉬운 작업인데 `effort`가 너무 높아?
- [ ] 답 길이를 안 묶었어?
- [ ] 안 바뀌는 시스템 규칙을 매번 다시 보내?
- [ ] 안 쓰는 도구 설명까지 같이 넣어?
- [ ] 대량 작업인데 배치로 안 뺐어?
- [ ] 비용을 안 보고 감으로만 쓰고 있어?

여기서 2개 이상 체크되면 거의 새고 있는 거야.

---

## Claude Code에선 뭘 치면 돼?

```text
/clear
/compact
/cost
/effort low
```

- `/clear` → 주제 갈아탈 때
- `/compact` → 세션 길어졌을 때
- `/cost` → 지금 얼마나 쓰는지 볼 때
- `/effort low` → 쉬운 일로 내릴 때

---

## API에선 뭘 먼저 붙이면 돼?

```json
{
  "output_config": {
    "effort": "low"
  }
}
```

쉬운 일인데도 무거운 추론이 돌면 토큰이 빨리 녹아.

그리고 고정 규칙은 캐시해.

```json
{
  "cache_control": {
    "type": "ephemeral"
  }
}
```

복붙용 전체 예시는 [여기](../레시피/claude-cache-request.example.json) 있어.

## 어떤 일을 가볍게 보내면 돼?

| 작업 | 시작 추천 |
|------|----------|
| 분류 | Haiku + low |
| 짧은 요약 | Sonnet + low |
| 일반 코딩 | Sonnet + medium |
| 깊은 리뷰 | Opus + high |

모든 걸 같은 모델로 보내지 마.
그게 제일 쉬운 절약이야.
