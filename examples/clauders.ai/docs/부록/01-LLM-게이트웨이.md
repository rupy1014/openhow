**API 키 3개 관리하기 귀찮지 않아?**

Claude, ChatGPT, Gemini — 각각 API 키 발급하고, 각각 결제하고, 각각 코드 바꾸고.
이걸 한 곳에서 해결하는 게 LLM 게이트웨이야.

---

## 게이트웨이가 뭐야?

여러 AI 서비스를 **하나의 창구**로 모아주는 도구야.

비유하면 환전소 같은 거야.
달러, 엔화, 유로 다 따로 바꿀 필요 없이 한 곳에서 다 해주잖아.
LLM 게이트웨이도 마찬가지야. API 키 하나(또는 0개)로 여러 AI를 쓸 수 있어.

---

## 왜 필요해?

**키 관리가 사라져.**

API 키를 서비스마다 따로 발급받고, 환경변수에 넣고, 요금 따로 확인하고 — 이게 귀찮잖아.
게이트웨이가 있으면 한 곳에서 끝나.

그리고 코드도 하나로 통일돼.
OpenAI 형식으로 요청하면 게이트웨이가 알아서 각 서비스에 맞게 바꿔서 보내줘.

---

## 뭐가 있어?

| | llm-mux | LiteLLM | OpenRouter |
|---|---|---|---|
| **한 줄 설명** | 구독만 있으면 API 키 없이 | 100개 모델을 한 줄로 | 설치 없이 바로 |
| **설치** | Go 바이너리 하나 | pip install | 없음 (웹 서비스) |
| **API 키 필요?** | 없어. OAuth 로그인이면 돼 | 필요해. 각 서비스별로 | 필요해. OpenRouter 키 하나 |
| **지원 모델** | Claude, ChatGPT | 100개 이상 | 100개 이상 |
| **로컬 실행** | localhost:8317 | localhost:4000 | 클라우드 |
| **특징** | 구독 계정으로 바로 사용 | 팀 공유용 프록시 | 통합 결제 |
