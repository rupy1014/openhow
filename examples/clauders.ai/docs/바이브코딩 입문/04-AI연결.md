**OpenRouter API 키 하나면, 캐릭터가 진짜 대답해.**

---

## OpenRouter가 뭐야?

AI 모델을 골라서 쓸 수 있는 중계소야.

Claude, GPT, Llama, Gemini — 여러 모델이 한 곳에 있어.
API 키 하나로 전부 쓸 수 있어.
모델마다 가격이 다르니까, 싸고 괜찮은 걸 골라도 돼.

---

## 어떻게 연결해?

**Step 1.** OpenRouter에서 API 키를 발급받아

[openrouter.ai](https://openrouter.ai) 가입 → 대시보드 → API Keys → 새 키 생성.
키가 나오면 복사해둬.

**Step 2.** Claude한테 연결해달라고 해

```
OpenRouter API로 캐릭터 대화를 연결해줘.
API 키는 환경변수로 관리하고, 모델은 사용자가 선택할 수 있게 해줘.

OpenRouter API 키: sk-or-v1-xxxxxxxx
```

Claude가 API 연결 코드를 짜고, 캐릭터 시스템 프롬프트를 대화에 적용해줘.

**Step 3.** 캐릭터를 골라서 말을 걸어봐

브라우저에서 캐릭터 하나 선택하고, 아무 말이나 보내봐.
캐릭터가 자기 성격대로 대답해.

이 순간이 하이라이트야.

---

## 모델 바꿔보면 재밌어

같은 캐릭터인데 AI 모델에 따라 느낌이 달라.

- **Claude** — 맥락을 잘 기억하고, 캐릭터 유지를 잘해
- **GPT** — 말이 좀 더 많고, 친절한 느낌
- **Llama** — 가볍고 빠른데, 캐릭터가 가끔 풀려

```
모델 선택 드롭다운 추가해줘. Claude, GPT-4o-mini, Llama 3개.
```

이렇게 말하면 Claude가 모델 선택 기능을 달아줘.
같은 캐릭터한테 같은 질문 던져보면 차이가 보여.

---

## 한 줄 정리

API 키 하나 넣으면 캐릭터가 살아나. 모델 바꾸면 같은 캐릭터도 다르게 말해.
