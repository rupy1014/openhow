**캐릭터 이미지 5장 던지면, AI가 알아서 페르소나를 만들어.**

이름, 성격, 말투, 시스템 프롬프트까지 자동이야.

---

## 어떻게 해?

**Step 1.** `characters/` 폴더에 이미지 5장을 넣어

캐릭터 이미지를 미리 준비해. 일러스트, 사진, AI 생성 이미지 — 뭐든 돼.
`my-chat/characters/` 폴더에 넣어.

```
my-chat/
└── characters/
    ├── 01.png
    ├── 02.png
    ├── 03.png
    ├── 04.png
    └── 05.png
```

**Step 2.** Claude한테 캐릭터를 만들어달라고 해

```
characters/ 폴더에 이미지 5장 넣어놨어.
각 이미지를 보고 캐릭터를 만들어줘.

캐릭터마다:
- 이름
- 성격 한 줄 설명
- 말투 특징
- 시스템 프롬프트 (이 캐릭터가 대화할 때 AI에게 보내는 설정)

만들고 나서 캐릭터 선택 화면에도 반영해줘.
```

Claude가 이미지를 분석해. 외형에서 성격을 유추하고, 거기에 맞는 말투와 프롬프트를 짜.

**Step 3.** 마음에 안 드는 캐릭터만 수정해

```
2번 캐릭터 말투 좀 더 까칠하게 바꿔줘
```

```
4번 캐릭터 이름 "루나"로 바꿔줘
```

전부 다시 만들 필요 없어. 하나만 골라서 수정하면 돼.

---

## 시스템 프롬프트가 뭐야?

**캐릭터의 영혼이야.**

AI한테 "너는 이런 캐릭터야"라고 알려주는 설정 문서야. 사용자한테는 안 보여. AI만 읽어.

예를 들면:

```
너는 츤데레 고양이 캐릭터 "냥이"야.
겉으로는 퉁명하지만 은근 챙겨줘.
반말 써. 이모지는 가끔 고양이 관련 것만.
대답은 짧게, 3문장 이내로.
```

이 한 줄 한 줄이 캐릭터의 성격을 결정해.
프롬프트를 바꾸면 같은 AI인데 완전히 다른 캐릭터가 돼.
