**만들었으면 보여줘야지. URL 하나면 누구한테든 공유할 수 있어.**

로컬에서만 돌리면 나밖에 못 봐. 배포하면 링크 하나로 세상에 공개돼.

---

## 배포가 뭐야?

내 컴퓨터에서 돌아가는 걸 **인터넷에 올리는 거야.**

`localhost:3000`은 나만 봐. 배포하면 `https://my-app.vercel.app` 같은 URL이 생겨. 이 링크를 보내면 누구든 써볼 수 있어.

---

## Vercel이 제일 쉬워

바이브코딩으로 만든 앱을 배포하는 방법은 여러 개야. 그중 **Vercel**이 가장 간단해.

왜?

- GitHub 연결하면 끝
- Next.js를 만든 회사라서 Next.js 배포가 완벽해
- 무료 플랜으로 충분해
- 환경변수 설정이 쉬워

---

## 어떻게 해?

캐릭터 채팅 앱을 Vercel에 배포하는 과정이야.

**Step 1.** GitHub에 코드를 올려

아직 GitHub에 안 올렸으면 Claude한테 시켜.

```
이 프로젝트를 GitHub에 올려줘.
리포지토리 이름은 character-chat으로.
```

이미 올렸으면 이 단계는 건너뛰어.

**Step 2.** Vercel에 가입해

[vercel.com](https://vercel.com)에서 **GitHub 계정으로 가입**해.

별도 회원가입 필요 없어. GitHub 로그인만 하면 돼.

**Step 3.** 프로젝트를 가져와

1. Vercel 대시보드에서 **"Add New → Project"** 클릭
2. GitHub 리포지토리 목록에서 `character-chat` 선택
3. **"Deploy"** 클릭

이게 끝이야. Vercel이 알아서 빌드하고 배포해.

**Step 4.** 환경변수를 넣어

캐릭터 채팅은 OpenRouter API 키가 필요해. 이걸 환경변수로 넣어야 해.

1. Vercel 프로젝트 → **Settings** → **Environment Variables**
2. 이름: `OPENROUTER_API_KEY`
3. 값: OpenRouter에서 발급받은 API 키
4. **Save** 클릭

그다음 **Deployments** 탭에서 **"Redeploy"** 를 눌러. 환경변수가 적용돼.

**Step 5.** 확인해

Vercel이 준 URL을 열어봐. `https://character-chat-xxxx.vercel.app` 같은 주소야.

캐릭터 선택 → 대화가 되면 성공이야.

---

## OpenRouter API 키는 어떻게 받아?

**Step 1.** [openrouter.ai](https://openrouter.ai)에서 가입해

Google 계정으로 바로 가입돼.

**Step 2.** Keys 메뉴에서 API 키 생성

대시보드 → **"Keys"** → **"Create Key"** → 이름 적고 생성.

**Step 3.** 무료 모델을 써

OpenRouter는 여러 AI 모델을 하나의 API로 쓸 수 있게 해줘. 무료 모델도 있어.

PRD에 이렇게 적어두면 돼:

```
AI 모델은 OpenRouter의 무료 모델을 사용해.
모델 ID: meta-llama/llama-4-maverick:free
API 엔드포인트: https://openrouter.ai/api/v1/chat/completions
```

---

## API 키를 코드에 적으면 안 돼

```
❌ const API_KEY = "sk-or-v1-abc123..."   // 코드에 직접
✅ const API_KEY = process.env.OPENROUTER_API_KEY  // 환경변수에서
```

코드에 직접 적으면 GitHub에 올라가. 누구나 볼 수 있어. API 키가 노출되면 남이 내 돈으로 API를 쓸 수 있어.

Claude한테 처음부터 이렇게 말해:

```
API 키는 환경변수로 관리해. 코드에 하드코딩하지 마.
.env.local 파일에 넣고, .gitignore에 추가해줘.
```

---

## 배포 후에 고치고 싶으면?

코드를 고치고 GitHub에 push하면 돼. Vercel이 자동으로 다시 배포해.

```
[수정 내용] 고쳐줘.
고치면 git으로 저장하고 push까지 해줘.
```

Push하면 1~2분 안에 새 버전이 올라가. 별도로 Vercel에서 뭘 할 필요 없어.

---

## 커스텀 도메인은?

기본 URL이 `character-chat-xxxx.vercel.app`처럼 길어. 내 도메인을 연결하고 싶으면:

1. Vercel 프로젝트 → **Settings** → **Domains**
2. 내 도메인 입력 (예: `chat.mydomain.com`)
3. DNS 설정을 안내대로 바꾸면 끝

지금은 몰라도 돼. 나중에 필요할 때 하면 돼.

---

## 한 줄 정리

GitHub에 올리고, Vercel에 연결하고, 환경변수 넣으면 끝. 5분이면 URL이 나와.
