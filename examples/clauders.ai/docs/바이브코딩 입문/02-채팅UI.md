**PRD.md 기반으로 실행하면, 채팅 화면이 한 번에 나와.**

---

## 어떻게 만들어?

**Step 1.** PRD.md 기반으로 프로젝트를 만들어달라고 해

```
PRD.md 기반으로 프로젝트 만들어줘. 캐릭터 선택 화면이랑 채팅 화면부터.
```

Claude가 파일을 쭉 만들어. 프레임워크가 뭔지, 컴포넌트가 뭔지 몰라도 돼.
끝나면 로컬에서 확인할 수 있게 알려줘.

**Step 2.** 브라우저에서 확인해

Claude가 알려주는 주소(보통 `http://localhost:숫자`)를 브라우저에서 열어봐.
캐릭터 선택 화면이 보여.

**Step 3.** 마음에 안 드는 건 말로 고쳐

```
캐릭터 카드 크기 좀 더 키워줘
```

```
배경 어둡게 바꿔줘
```

```
말풍선 모서리 좀 더 둥글게
```

한 번에 완벽하게 나오는 법이 없어. 보고, 말하고, 고치는 게 바이브코딩이야.

---

## 말로 UI를 깎는 팁

**스크린샷을 활용해.** 비슷한 디자인을 찾아서 "이거랑 비슷하게 해줘"가 가장 빨라.

**한 번에 하나만 바꿔.** "색도 바꾸고 레이아웃도 바꾸고 폰트도 바꿔줘"하면 어디가 바뀐 건지 확인이 안 돼. 하나씩 바꾸고 확인해.

**구체적으로 말해.** "좀 예쁘게"보다 "카드 사이 간격 20px로"가 결과가 정확해. 수치를 몰라도 "간격 좁혀줘", "글자 크게"면 충분해.

---

## 한 줄 정리

PRD.md 기반으로 실행하면 화면이 나와. 안 맞으면 말로 고쳐.
