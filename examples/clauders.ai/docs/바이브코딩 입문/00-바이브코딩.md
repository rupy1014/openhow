**코드를 몰라도 만들어. 원하는 걸 말하면 AI가 짜.**

이걸 바이브코딩이라고 해.

---

## 바이브코딩이 뭐야?

2025년 2월, AI 연구자 Andrej Karpathy(전 OpenAI, 전 Tesla AI 총괄)가 X에 올린 글에서 나온 말이야.

> "코드가 존재한다는 것도 잊고, 바이브에 완전히 맡긴다."

코드를 이해하려 하지 않아. 원하는 결과를 말로 설명하고, 나온 걸 보고, 또 설명해.
이 반복이 전부야.

그해 Collins 영어 사전이 바이브코딩을 **올해의 단어**로 뽑았어.

---

## 기존 코딩이랑 뭐가 달라?

기존 코딩은 프로그래밍 언어를 배워야 해. HTML(뼈대), CSS(꾸미기), JavaScript(동작) — 각각 문법이 다르고 조합도 배워야 해.

바이브코딩은 달라.

```
링크 모음 페이지 만들어줘. 카드 형태로, 배경은 어둡게.
```

이게 코딩이야. 언어는 한국어고, 도구는 Claude Code야.

코드를 모르는 게 문제가 아니야.
원하는 걸 말로 설명 못하는 게 문제야. 그건 연습하면 돼.

---

## 뭘 만들 수 있어?

페이지, 도구, 앱 — 어지간한 건 다 돼.

- 내 링크 모음 페이지
- 이력서 / 포트폴리오 사이트
- 계산기, 타이머, 체크리스트 같은 작은 도구
- 블로그, 소개 페이지

디자인도 "좀 더 어둡게", "폰트 크게", "여백 줄여줘"로 조정해.
코드 한 줄 안 봐도 돼.

---

## 한 줄 정리

바이브코딩은 만들고 싶은 걸 말로 설명하는 거야. 코드는 몰라도 돼.
