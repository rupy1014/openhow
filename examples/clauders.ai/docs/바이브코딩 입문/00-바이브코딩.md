**코드를 몰라도 만든다. 원하는 걸 말하면 AI가 짠다.**

이걸 바이브코딩이라고 한다.

---

## 바이브코딩이 뭐야?

2025년 2월, OpenAI 공동창업자 Andrej Karpathy가 X에 올린 글에서 나온 말이야.

> "코드가 존재한다는 것도 잊고, 바이브에 완전히 맡긴다."

코드를 이해하려 하지 않는다. 원하는 결과를 말로 설명하고, 나온 걸 보고, 또 설명한다.
이 반복이 전부야.

그해 Collins 영어 사전이 바이브코딩을 **올해의 단어**로 뽑았다.

---

## 기존 코딩이랑 뭐가 달라?

기존 코딩은 프로그래밍 언어를 배워야 한다. HTML(뼈대), CSS(꾸미기), JavaScript(동작) — 각각 문법이 다르고 조합도 배워야 해.

바이브코딩은 다르다.

```
링크 모음 페이지 만들어줘. 카드 형태로, 배경은 어둡게.
```

이게 코딩이야. 언어는 한국어고, 도구는 Claude Code야.

코드를 모르는 게 문제가 아니야.
원하는 걸 말로 설명 못하는 게 문제다. 그건 연습하면 된다.

---

## 뭘 만들 수 있어?

페이지, 도구, 앱 — 어지간한 건 다 된다.

- 내 링크 모음 페이지
- 이력서 / 포트폴리오 사이트
- 계산기, 타이머, 체크리스트 같은 작은 도구
- 블로그, 소개 페이지

디자인도 "좀 더 어둡게", "폰트 크게", "여백 줄여줘"로 조정한다.
코드 한 줄 안 봐도 된다.

---

## 이 섹션에서 뭘 배워?

첫 페이지를 처음부터 만든다.

한마디로 파일이 생기는 경험을 직접 해본다.
마음에 안 들면 어떻게 고치는지, 참고 자료를 어떻게 던지는지도 같이 익힌다.

코드는 Claude Code가 짠다. 나는 원하는 걸 설명하는 데만 집중하면 된다.

---

## 한 줄 정리

바이브코딩은 만들고 싶은 걸 말로 설명하는 것이다. 코드는 몰라도 된다.
