```
# VS Code 터미널
~/my-site $ dsclaude

> 링크 모음 페이지 만들어줘. 카드 형태로, 배경은 어둡게.
index.html 생성 중...
style.css 생성 중...
완료.
```

HTML, CSS 한 줄도 안 썼다. 프롬프트 하나로 페이지가 나왔다.

이게 바이브 코딩이다.

---

## 뭐가 필요해?

두 가지만 있으면 된다.

1. **VS Code** — 에디터. 코드 파일을 열고 확인하는 프로그램이다.
2. **Claude Code** — 터미널에서 실행하는 AI 도구. 프롬프트를 주면 파일을 직접 만들어준다.

설치는 [설치 가이드](/입문/01-설치)에서. 이미 했으면 바로 아래로.

---

## 어떻게 만들어?

**Step 1.** VS Code에서 터미널을 열고 새 디렉토리를 만든다

```bash
mkdir my-site
cd my-site
```

디렉토리 = 폴더야. 여기에 파일이 생긴다.

**Step 2.** 터미널에서 `dsclaude` 실행

```bash
dsclaude
```

**Step 3.** 만들고 싶은 걸 프롬프트로 말한다

```
링크 모음 페이지 만들어줘. 카드 형태로, 배경은 어둡게.
```

Claude Code가 HTML과 CSS 파일을 직접 만든다. VS Code 왼쪽 파일 탐색기에 `index.html`, `style.css`가 생겨 있어. 브라우저로 열어봐.

**Step 4.** 마음에 안 들면 그냥 말한다

```
카드 색상 좀 더 밝게 바꿔줘
```

```
카드마다 설명 텍스트 추가해줘
```

코드 볼 필요 없어.
Claude Code가 파일을 직접 열고, 고치고, 저장한다.
원하는 대로 나올 때까지 계속 프롬프트를 던지면 돼.

프론트엔드를 몰라도 괜찮다. JavaScript가 뭔지 몰라도 페이지는 나온다.
