```
# VSCode 터미널
~/my-site $ dsclaude

> 링크 모음 페이지 만들어줘. 카드 형태로, 배경은 어둡게.
index.html 생성 중...
style.css 생성 중...
완료.
```

HTML, CSS 한 줄도 안 썼다.

---

## 어떻게 만드냐

**Step 1.** 새 폴더를 만들고 VSCode로 연다

```bash
mkdir my-site
cd my-site
code .
```

**Step 2.** 터미널에서 `dsclaude` 실행

```bash
dsclaude
```

**Step 3.** 원하는 걸 말로 설명한다

```
링크 모음 페이지 만들어줘. 카드 형태로, 배경은 어둡게.
```

VSCode 파일 탐색기에 `index.html`, `style.css`가 생긴다.
브라우저로 열면 된다.

**Step 4.** 마음에 안 들면 그냥 말한다

```
카드 색상 좀 더 밝게 바꿔줘
```

```
카드마다 설명 텍스트 추가해줘
```

코드 볼 필요 없다. 말하면 Claude가 직접 수정한다.

---

## 한 줄 정리

만들고 싶은 걸 말하면 파일이 생긴다. 코드는 Claude가 짠다.
