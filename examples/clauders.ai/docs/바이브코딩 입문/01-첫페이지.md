```
# VS Code 터미널
~/my-site $ dsclaude

> 링크 모음 페이지 만들어줘. 카드 형태로, 배경은 어둡게.
index.html 생성 중...
style.css 생성 중...
완료.
```

HTML, CSS 한 줄도 안 썼어. 요청 하나로 페이지가 나왔어.

이게 바이브 코딩이야.

---

## 뭐가 필요해?

두 가지만 있으면 돼.

1. **VS Code** — 파일을 열고 확인하는 프로그램이야. 메모장의 업그레이드 버전이라고 생각하면 돼.
2. **Claude Code** — 터미널에서 실행하는 AI 도구. 원하는 걸 말하면 파일을 직접 만들어줘.

설치는 [설치 가이드](/입문/01-설치)에서. 이미 했으면 바로 아래로.

---

## 어떻게 만들어?

**Step 1.** VS Code에서 터미널을 열고 새 폴더를 만들어

```bash
mkdir my-site
cd my-site
```

폴더를 만드는 거야. 여기에 파일이 생겨.

**Step 2.** 터미널에서 `dsclaude` 실행

```bash
dsclaude
```

**Step 3.** 만들고 싶은 걸 말해

```
링크 모음 페이지 만들어줘. 카드 형태로, 배경은 어둡게.
```

Claude Code가 파일을 직접 만들어. VS Code 왼쪽 파일 목록에 `index.html`, `style.css`가 생겨 있어. `index.html`을 브라우저(크롬 같은 거)로 열어봐.

**Step 4.** 마음에 안 들면 그냥 말해

```
카드 색상 좀 더 밝게 바꿔줘
```

```
카드마다 설명 텍스트 추가해줘
```

코드 볼 필요 없어.
Claude Code가 파일을 직접 열고, 고치고, 저장해.
원하는 대로 나올 때까지 계속 말하면 돼.

코딩을 몰라도 괜찮아. HTML이 뭔지 몰라도 페이지는 나와.
