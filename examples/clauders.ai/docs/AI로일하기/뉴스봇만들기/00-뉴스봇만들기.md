**매일 아침 카톡에 AI 뉴스가 온다. 내가 안 해도.**

이게 완성된 모습이야:

```
안녕하세요. 3월 9일(월) 바이브 코더를 위한 AI 브리핑입니다.
오늘 당장 써먹을 수 있는 '실전 AI 기술' 15가지를 요약했습니다.

[Today's AI News]

1. Claude Code, 멀티에이전트 워크플로우 지원 추가
2. Perplexity, 제로에서 90억 달러까지
3. MCP 서버 디버깅 도구 Inspector 출시
...
```

근데 이걸 만들려고 스크립트를 하나하나 코딩할 필요 없어.

---

## 핵심: 너는 명령어 하나만 치면 돼

```bash
/workflow
```

이거 하나면 RSS 수집 → AI 리뷰 생성 → 퇴고까지 자동으로 돌아가.

진짜로. Claude Code 열고 `/workflow` 치면 끝이야.

---

## 전체 흐름

```
너: /workflow 실행
  ↓
1. RSS 피드 10개에서 뉴스 수집      ← 자동
2. AI가 한국어 리뷰 생성             ← 자동
3. 퇴고 (번역투 제거, 톤 맞춤)       ← 자동
4. 카테고리/태그 검토                ← 자동
5. DB 배포 or 카톡 발송             ← 네가 선택
```

1~3단계는 `/workflow` 한 번이면 끝나. 4~5단계는 결과 보고 "배포해줘"라고 말하면 돼.

---

## 이 구조를 바꾸면 뭐든 돼

RSS 소스를 바꾸고, AI 톤을 바꾸고, 발송 채널을 바꾸면 — 전부 같은 코드야.

| 응용 | 바꾸는 것 |
|------|----------|
| 경쟁사 모니터링 봇 | 소스 → 경쟁사 블로그 |
| 업계 트렌드 봇 | 필터 → 특정 키워드만 |
| 쓰레드 자동화 | 발송 → SNS 포맷으로 |
| 일일 리포트 | 소스 → 사내 데이터 |

전부 같은 구조야. **수집 → AI 가공 → 발송.**

바꾸는 방법도 AI한테 시키면 돼. 다음 글에서 하나씩 볼 거야.

---

## 한 줄 정리

`/workflow` 한 번이면 뉴스봇이 돌아가. 나머지는 AI가 해.
