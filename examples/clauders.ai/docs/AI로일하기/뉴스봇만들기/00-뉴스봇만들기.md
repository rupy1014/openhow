## 딸깍 한 번이면 돼

Claude Code 열고:

```
/workflow
```

이거 하나면 RSS 수집 → AI 리뷰 생성 → 퇴고까지 전부 돌아가.

코드를 짤 필요도, 이해할 필요도 없어. 딸깍.

---

## 뭐가 자동으로 되냐

```
/workflow 딸깍
  ↓
1. 해외 뉴스 10개 소스에서 수집    ← 자동
2. AI가 한국어 리뷰 생성           ← 자동
3. 퇴고 (번역투 제거, 톤 맞춤)     ← 자동
  ↓
content-workflow/ 폴더에 완성된 리뷰가 쌓여
```

브라우저에서 바로 확인도 돼. DB 설정 같은 거 없어도.

---

## 이 구조를 바꾸면 뭐든 돼

소스를 바꾸고, AI 톤을 바꾸고, 발송 채널을 바꾸면 — 전부 같은 코드야.

| 응용 | Claude Code에게 이렇게 말해 |
|------|---------------------------|
| 경쟁사 모니터링 봇 | "소스를 경쟁사 블로그로 바꿔줘" |
| 업계 트렌드 봇 | "AI 키워드만 필터링해줘" |
| 쓰레드 자동화 | "리뷰를 트위터 쓰레드 형식으로 바꿔줘" |
| 일일 리포트 | "슬랙으로 매일 보내줘" |

바꾸는 것도 딸깍이야. AI한테 말하면 돼.
