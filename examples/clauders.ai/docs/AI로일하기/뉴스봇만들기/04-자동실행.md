## 완전 자동으로 바꾸기

내가 아무것도 안 해도 매일 아침 8시에 돌아가게 하고 싶으면?

이것도 딸깍:

```
> 매일 아침 8시에 콘텐츠를 자동 수집하고 리뷰 생성까지 하는 cron 만들어줘.
> GitHub Actions로 하고 싶어.
```

:::info cron이 뭐야?
알람 시계야. 근데 알람 대신 프로그램을 실행해. "매일 아침 8시에 이 스크립트 돌려줘"를 설정하는 거야.
:::

:::info GitHub Actions가 뭐야?
GitHub에서 코드를 올리면 자동으로 뭔가를 실행해주는 서비스야. "매일 아침 자동 실행" 같은 게 가능해. 무료야.
:::

---

## 방법은 여러 가지야

| 방법 | 특징 |
|------|------|
| 내 컴퓨터 cron | 컴퓨터 켜져 있어야 함. 제일 간단 |
| GitHub Actions | 무료. 24시간 동작. 제일 많이 씀 |
| 서버 (VPS) | 월 $5. 자유도 높음 |

뭘 선택하든 Claude Code한테 물어보면 설정 파일을 만들어줘.

**내 컴퓨터에서 돌리고 싶으면:**

```
> 매일 아침 8시에 수집 + 리뷰 생성을 자동으로 돌리는 cron 설정 만들어줘
```

**GitHub Actions로 하고 싶으면:**

```
> GitHub Actions로 매일 아침 8시(한국시간)에 콘텐츠 수집하고
> 리뷰 생성하는 워크플로우 만들어줘
```

딸깍.

## 전체 정리

```
딸깍 1: zip 다운로드 → npm install       ← 프로젝트 받기 (1번만)
딸깍 2: claude → /workflow              ← 수집 + 리뷰 + 퇴고
딸깍 3: npm run dev                     ← 브라우저에서 확인
딸깍 4: "슬랙/카톡으로 보내줘"            ← 발송 채널 (선택)
딸깍 5: "매일 자동으로 돌려줘"            ← 자동화 (선택)
```

1~3이면 뉴스봇이 돌아가. 4~5는 원할 때 추가하면 돼.
