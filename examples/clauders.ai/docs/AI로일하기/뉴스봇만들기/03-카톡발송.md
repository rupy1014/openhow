**리뷰가 만들어졌으면, 이제 카톡으로 보내보자.**

`/workflow`로 생성된 리뷰를 뉴스레터로 만들고, 카카오톡 "나에게 보내기"로 전송하는 거야.

---

## 이것도 AI한테 시켜

Claude Code에서:

```
> content-workflow/ 폴더의 리뷰들을 뉴스레터 텍스트로 만들고
> 카카오톡 나에게 보내기로 전송하는 기능 만들어줘.
>
> 뉴스레터 형식:
> - 인사말 + 날짜
> - 중요한 순서대로 15개
> - 각 항목은 제목 + 한줄 설명
>
> 카카오 API 설정도 알려줘.
```

Claude Code가 스크립트를 만들면서, 카카오 API 셋업 방법도 같이 알려줘:

1. [developers.kakao.com](https://developers.kakao.com)에서 앱 만들기
2. "나에게 보내기" 권한 추가
3. 토큰 발급
4. `.env`에 토큰 저장

:::info 토큰이 뭐야?
API를 쓰기 위한 임시 비밀번호야. "나 인증된 사람이야"를 증명하는 거야.
:::

## 카톡 말고 다른 채널은?

| 채널 | Claude Code에게 이렇게 말해 |
|------|---------------------------|
| 슬랙 | "슬랙 웹훅으로 보내는 기능 추가해줘" |
| 디스코드 | "디스코드 웹훅으로 보내줘" |
| 이메일 | "이메일로 뉴스레터 발송해줘" |
| 텔레그램 | "텔레그램 봇으로 보내줘" |

채널을 바꾸는 건 발송 부분만 바꾸면 돼. 수집-리뷰 생성은 그대로야.
