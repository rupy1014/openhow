**리뷰가 만들어졌으면, 이제 카톡으로 보내보자.**

이것도 Claude Code한테 시키면 돼.

---

## 딸깍: 카톡 발송 기능 만들기

```
> content-workflow/ 폴더의 리뷰들을 뉴스레터 텍스트로 만들고
> 카카오톡 나에게 보내기로 전송하는 기능 만들어줘.
> 카카오 API 설정 방법도 알려줘.
```

Claude Code가 해주는 일:
1. 뉴스레터 텍스트 생성 스크립트 작성
2. 카카오 API 셋업 안내 (앱 만들기 → 권한 설정 → 토큰 발급)
3. 발송 스크립트 작성
4. `.env`에 넣을 값 안내

네가 직접 해야 하는 건 **카카오 디벨로퍼스 사이트에서 앱 만들기** 하나야. 그것도 Claude Code가 "여기 가서 이거 눌러"라고 알려줘.

:::info 토큰이 뭐야?
API를 쓰기 위한 임시 비밀번호야. "나 인증된 사람이야"를 증명하는 거야.
:::
