**네 일 중에 하나 골라서, 도구로 만들어와.**

---

## 뭘 해와야 해?

**Step 1.** 네 일상 업무 중에 **매주 반복되는 것** 하나를 골라

- 주간 보고서
- 회의록 정리
- 경쟁사 모니터링
- SNS 콘텐츠 제작
- 데이터 취합
- 뉴스레터

뭐든 돼. "이거 매번 귀찮다"고 느끼는 게 정답이야.

**Step 2.** AI한테 시켜서 한 번 수동으로 해봐

프롬프트를 직접 쳐서 결과를 뽑아봐. 쓸만한지 확인해.

**Step 3.** 그걸 커스텀 커맨드나 스크립트로 만들어

- 간단하면 → `.claude/commands/`에 커맨드 파일
- 파일 처리가 필요하면 → 바이브코딩으로 스크립트

---

## MAX5가 만든 도구들 참고

| 도구 | 하는 일 | 방법 |
|------|---------|------|
| `fetch-all.js` | 10개 소스에서 뉴스 수집 | 스크립트 |
| `generate-newsletter.js` | 뉴스레터 텍스트 생성 | 스크립트 |
| `/deploy` 커맨드 | DB 배포 실행 | 커스텀 커맨드 |
| `newneek-reviewer` | 뉴닉 스타일 리뷰 생성 | 에이전트 |
| `run-workflow.sh` | 전체 파이프라인 한 번에 | 스크립트 |

전부 처음에는 수동이었어. 반복되니까 도구로 만든 거야.

---

## 제출

- 만든 커맨드 파일 또는 스크립트
- 실행 결과 스크린샷
- "이걸 왜 만들었는지" 한 줄
