**코딩만 AI야? 아니야. 네 일도 AI한테 시킬 수 있어.**

글 쓰고, 자료 찾고, 데이터 정리하고, 콘텐츠 만들고.

이것도 전부 "해줘" 한마디면 돼.

---

## 예제: MAX5 AI 뉴스레터

이 파트에서는 실제 프로젝트를 예시로 써.

MAX5는 해외 테크 뉴스를 자동으로 모아서, AI가 한국어 리뷰를 만들고, 뉴스레터로 보내는 사이트야.

```
뉴스 수집 → AI가 리뷰 작성 → 사이트에 올리기 → 뉴스레터 발송
```

이 전체를 AI한테 시켜서 만들었어.

## 핵심은 "반복되면 도구로 만들어"

01~04에서 수동으로 해봐.

"AI한테 시키면 되는구나" — 이 감이 잡히면 05에서 그걸 도구로 만들어.

매번 프롬프트 칠 필요 없이, 실행 한 번이면 끝나게.

자동화에서 배운 커스텀 커맨드 + 바이브코딩에서 배운 빌드.

이 두 개가 합쳐지는 지점이야.

01~06으로 감을 잡았으면, 바로 아래 실전 파트인 [뉴스봇 만들기](./뉴스봇만들기/00-뉴스봇만들기)로 이어가.
