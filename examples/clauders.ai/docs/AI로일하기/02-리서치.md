**"이거 좀 조사해줘" — 이 한마디가 진짜 먹혀.**

MAX5에서는 매일 10개 이상의 소스에서 뉴스를 수집해.

Hacker News, Product Hunt, TechCrunch, 인디해커 블로그...

사람이 하면 매일 2시간. AI한테 시키면 한마디면 돼.

---

## 뉴스 소스 조사부터 시켜

**Step 1.** 어떤 소스를 긁을지 AI한테 물어봐

```
> AI/테크 뉴스를 수집하고 싶어.
> RSS 피드가 있는 곳 위주로 추천해줘.
> 실제 URL도 같이 알려줘.
```

:::info RSS가 뭐야?
웹사이트가 "새 글 올렸어요"를 알려주는 방법이야. 뉴스 구독 같은 거야. RSS 피드 주소만 알면 사이트에 직접 안 가도 새 글을 자동으로 가져올 수 있어.
:::

MAX5에서 실제로 쓰는 소스들:

| 소스 | 뭘 얻어 |
|------|---------|
| Hacker News | 테크 커뮤니티 인기 글 |
| Product Hunt | 신제품 런칭 정보 |
| MarkTechPost | AI 연구 + 적용 사례 |
| levels.io | 1인 창업자 실전기 |
| Lenny's Newsletter | 제품 전략, 그로스 인사이트 |

**Step 2.** 각 소스 특성을 분석시켜

```
> 이 5개 소스를 비교해줘.
> 어떤 종류의 콘텐츠가 나오는지,
> 얼마나 자주 올라오는지,
> 비개발자한테도 유용한지 비교표로 정리해줘.
```

---

## 경쟁사 분석도 같은 방식이야

뉴스 소스를 경쟁사 블로그로 바꾸면 끝.

```
> 경쟁사 3곳 블로그를 모니터링하고 싶어.
> 각 블로그의 RSS 피드 주소 찾아줘.
> 없으면 다른 방법도 알려줘.
```

그리고:

```
> 이번 달 경쟁사 블로그 글 전부 읽고,
> 어떤 주제에 집중하는지,
> 우리랑 뭐가 다른지 정리해줘.
```

## 쌓이면 자산이 돼

MAX5에서는 수집한 원문을 날짜별로 저장해:

```
content/
├── 2025-03-01/    ← 3/1에 수집한 뉴스들
├── 2025-03-02/
└── 2025-03-08/
```

이렇게 쌓이면 나중에 이런 질문이 가능해져:

```
> 지난 한 달간 AI 업계에서 가장 많이 나온 주제가 뭐야?
```

데이터가 쌓이면 질문의 질이 달라져.
