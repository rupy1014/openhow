**마크다운으로 내용 쓰고, AI한테 발표 자료로 뽑아.**

PPT 안 열어도 돼. 내용만 정리하면 AI가 디자인까지 해줘.

[예시 파일 다운로드 (바이브기획 발표자료)](./vibe-ppt-example.html)

---

## 왜 마크다운으로 쓰냐

PPT는 디자인에 시간 다 쓰잖아. 내용은 10분, 디자인은 2시간.

마크다운으로 쓰면 **내용에만 집중**할 수 있어. 디자인은 AI가 해.

```
기존 방식:
  내용 정리 → PPT 열기 → 레이아웃 → 디자인 → 수정 → 수정 → 수정

바이브 방식:
  마크다운에 내용 정리 → AI한테 변환 요청 → 끝
```

---

## 어떻게 만드냐

**Step 1.** 마크다운으로 발표 내용을 정리해

실제로 아래 예시 파일을 만들 때 이렇게 썼어:

```markdown
# 바이브 기획

## 슬라이드 1: 타이틀
바이브기획 방법론 - AI 시대, 기획 산출물의 형태가 바뀌어야 한다

## 슬라이드 2: 문제 제기
PM/PO의 3가지 현실적 고민
1. 산출물 관리의 혼란 - PRD_v1, v2, final, final_final...
2. 프로젝트 히스토리 부재 - 담당자 바뀌면 맥락 파악 2주
3. 기획 속도 느림 - 개발은 빨라졌는데 기획이 병목

## 슬라이드 3: 핵심
- 구현 능력보다 도메인 지식이 희소 자원
- AI가 읽을 수 있는 형식으로 써야 한다

## 슬라이드 4: 도구
- Claude Code, 지침 파일(CLAUDE.md)
- 마크다운 → PPT, 노션, 기능정의서로 변환
```

이렇게 슬라이드별로 내용만 적으면 돼. 디자인 지시는 다음 단계에서 해.

**Step 2.** AI한테 프레젠테이션으로 변환 요청해

```
> 이 마크다운 내용으로 발표 자료를 만들어줘.
> HTML 단일 파일로, 스크롤 방식 슬라이드로.
> 폰트는 Noto Sans KR, 색상은 로즈 계열.
> 각 섹션은 100vh 전체 화면으로.
```

**Step 3.** 결과물 확인하고 피드백 줘

```
> 3번 슬라이드에 비교 테이블 추가해줘.
> 카드 디자인 좀 더 둥글게.
> 폰트 크기 좀 키워줘.
```

이게 루프백이야. 한 번에 완벽할 필요 없어. 피드백 주면 AI가 고쳐줘.

## 다른 방법도 있냐

마크다운 → HTML 말고도 방법이 많아. 상황에 맞게 골라.

### AI에게 직접 만들게 하기

| 도구 | 특징 |
|------|------|
| **Claude Artifacts** | claude.ai에서 바로 HTML 슬라이드 생성. 브라우저에서 즉시 확인 |
| **Claude Code + MCP** | SlideSpeak MCP, Office-PowerPoint MCP 등으로 PPTX 직접 생성 |
| **Manus Slides** | 프롬프트 주면 리서치까지 해서 슬라이드 자동 생성. PPTX/PDF 다운로드 |
| **Google Gemini** | Gemini 앱에서 프레젠테이션 생성 → Google Slides로 내보내기 |

### SaaS 도구 쓰기

| 도구 | 특징 |
|------|------|
| **Gamma** | 가장 많이 쓰는 AI 슬라이드 도구. 웹 기반, 무료 플랜 있음 |
| **Beautiful.ai** | 자동 레이아웃, 디자인 품질 좋음 |
| **Canva AI** | 템플릿 풍부. 비디자이너한테 제일 편함 |
| **MS Copilot** | PowerPoint 안에서 바로 AI 생성. 기존 PPT 사용자한테 자연스러움 |

### 마크다운 전용 도구 (개발자용)

| 도구 | 특징 |
|------|------|
| **Marp** | 가장 심플. VS Code 확장 지원. md → HTML/PDF/PPTX |
| **Slidev** | Vue 기반, 라이브 코딩 가능, 인터랙티브 애니메이션. 개발자 경험 최상 |
| **Reveal.js** | 가장 오래된 HTML 프레젠테이션 프레임워크. 플러그인 생태계 풍부 |

## 뭘 쓸지 모르겠으면

```
빠르게 한 번 만들고 싶다 → Claude Artifacts 또는 Gamma
PPTX로 납품해야 한다   → MCP (SlideSpeak) 또는 Manus
매번 마크다운으로 관리하고 싶다 → Marp 또는 Slidev
기존 PPT에 AI 붙이고 싶다 → MS Copilot
```

어떤 도구든 핵심은 같아. **내용을 먼저 마크다운으로 정리**하고, 디자인은 AI한테 맡겨.
