📈 삼성증권 국내주식 매매창

기획 문서 v1.0

🎯 핵심 기능 정의

주요 사용자 니즈

  • 빠른 호가 확인 및 주문 실행
  • 실시간 체결 내역 모니터링
  • 보유 계좌 및 잔고 확인
  • 주문 정정/취소 편의성

🖥️ 매매창 구조

1. 상단 영역

  • 종목 검색/선택 기능
  • 현재가, 등락률, 거래량 실시간 표시
  • 차트 간략 보기

좌측: 호가창

  • 매도/매수 호가 10단계
  • 호가잔량, 호가건수
  • 호가 클릭으로 주문가 자동입력

우측: 주문 입력

  • 계좌 선택
  • 매도/매수 탭
  • 주문유형 선택
  • 수량/가격 입력
  • 예상 체결금액 표시
  • 주문 실행 버튼

3. 하단 영역 (탭 구조)

  • 미체결: 주문 정정/취소 기능
  • 체결내역: 오늘의 체결 리스트
  • 보유종목: 평가손익, 수익률 표시
  • 계좌잔고: 예수금, 주문가능금액

📋 주요 사용자 시나리오

시나리오 1: 빠른 매수 주문

종목 검색 → 호가창에서 원하는 가격 클릭 → 수량 입력 → 주문 실행

시나리오 2: 미체결 주문 정정

하단 미체결 탭 선택 → 주문 선택 → 정정 버튼 → 가격/수량 수정 → 재주문

시나리오 3: 보유종목 매도

하단 보유종목 탭 → 종목 선택 → 매도 버튼 → 수량 선택 (전량/일부) → 주문 실행

⭐ 차별화 포인트

원클릭 주문

호가 클릭만으로 즉시 주문가 입력

스마트 알림

체결, 정정, 취소 시 푸시 알림

자동주문

손절/익절 목표가 설정 기능

실시간 손익

보유종목의 실시간 평가손익 표시

🔄 사용자 플로우 차트

매매창 진입부터 주문 실행, 체결 확인까지의 전체 프로세스를 시각화한 플로우입니다. 로그인 확인, 계좌 선택, 종목 검색, 호가 확인, 주문 입력, 주문 실행, 체결 모니터링의 단계별 흐름과 분기 처리를 포함합니다.

💡 플로우 차트는 별도의 Mermaid 다이어그램으로 제공됩니다

매매창 진입 → 로그인 확인 → 계좌 선택 → 종목 검색 → 실시간 데이터 로딩 → 호가/주문 화면 표시 → 주문 입력 → 잔고 확인 → 주문 전송 → 체결 모니터링
