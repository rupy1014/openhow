## 1. 시스템 개요

### 1.1 프로젝트 목표

어디산은 생산자와 소비자를 직접 연결하는 **마이크로서비스 기반 농산물 직거래 플랫폼**입니다.

**핵심 목표:**
- 생산자 인증 시스템 (POLICY-001 기반)
- 실시간 거래 및 정산 시스템
- 블록체인 기반 원산지 추적
- AI 기반 부정거래 탐지
- 당일 배송 지원

### 1.2 시스템 범위

**포함 모듈:**
```
1. 사용자 관리 (User Management)
2. 생산자 인증 시스템 (Producer Authentication)
3. 상품 카탈로그 (Product Catalog)
4. 실시간 거래 (Real-time Trading)
5. 결제 및 정산 (Payment & Settlement)
6. 배송 추적 (Logistics Tracking)
7. 원산지 추적 (Blockchain Traceability)
8. 모니터링 및 분석 (Analytics & Monitoring)
```

**미포함 (향후 개발):**
- 음성 주문 (AI Voice Assistant)
- AR 기반 제품 검색
- 농가 경영 관리 시스템

**AI 기반 부정거래 탐지 (Anomaly Detection):**
```python
# 파이썬 코드 예시
import numpy as np
from sklearn.ensemble import IsolationForest

class AnomalyDetector:
    def __init__(self):
        self.model = IsolationForest(
            contamination=0.05,  # 상위 5% 이상치
            random_state=42
        )
    
    def train(self, transaction_data):
        """
        거래 데이터로 모델 학습
        특성: [order_quantity, price, delivery_time, producer_history]
        """
        features = self.extract_features(transaction_data)
        self.model.fit(features)
    
    def detect(self, order):
        """
        단일 주문에 대한 이상 탐지
        반환: {'is_anomaly': bool, 'confidence': float}
        """
        features = self.extract_features([order])
        prediction = self.model.predict(features)
        return {
            'is_anomaly': prediction[0] == -1,
            'confidence': abs(self.model.score_samples(features)[0])
        }
    
    def extract_features(self, transactions):
        """
        거래 데이터에서 특성 추출
        """
        features = []
        for tx in transactions:
            feature_vector = [
                tx['quantity'],  # 주문량
                tx['price'],     # 가격
                tx['delivery_time'],  # 배송 시간
                tx['producer_history_score'],  # 생산자 신뢰도
                tx['time_of_day'],  # 주문 시간대
            ]
            features.append(feature_vector)
        return np.array(features)

# 사용 예시
detector = AnomalyDetector()

# 1. 정상 거래로 학습
normal_transactions = get_historical_transactions()
detector.train(normal_transactions)

# 2. 새 주문 감지
new_order = {
    'quantity': 10000,  # 비정상적으로 많음
    'price': 100,
    'delivery_time': 2,
    'producer_history_score': 0.3,  # 낮은 신뢰도
    'time_of_day': 3  # 새벽 주문
}

result = detector.detect(new_order)
if result['is_anomaly']:
    # 경고 발송
    alert_admin(f"이상 거래 감지: 신뢰도 {result['confidence']:.2%}")
    # 거래 보류
    hold_order(new_order)
```

## 7. 보안 아키텍처

### 7.1 데이터 보안

**암호화:**
```
1. 전송 계층 (Transport Layer)
   - TLS 1.3 (HTTPS)
   - 모든 API 통신 암호화
   
2. 저장소 계층 (Storage Layer)
   - 비밀번호: bcrypt (salt rounds: 12)
   - 결제 정보: AES-256
   - 민감한 데이터: Field-level 암호화
```

**비밀번호 해시:**
```javascript
// bcrypt 설정
const bcrypt = require('bcrypt');
const saltRounds = 12;

async function hashPassword(password) {
  const salt = await bcrypt.genSalt(saltRounds);
  return bcrypt.hash(password, salt);
}
```

### 7.2 인증 및 인가 (Authentication & Authorization)

**JWT 토큰 구조:**
```
Header: {
  "alg": "HS256",
  "typ": "JWT"
}

Payload: {
  "sub": "user_123",
  "roles": ["consumer"],
  "email": "user@example.com",
  "iat": 1704067200,
  "exp": 1704153600
}
```

**토큰 관리:**
- Access Token TTL: 1시간
- Refresh Token TTL: 30일
- Token Rotation: Refresh마다 새 토큰 발급

### 7.3 API 보안

**Rate Limiting (API Gateway 레벨):**
```
- 로그인 API: 10 requests/minute/IP
- 일반 API: 100 requests/minute/user
- 결제 API: 5 requests/minute/user
```

**SQL Injection 방어:**
```
- Parameterized Queries 사용
- ORM (Sequelize, Typeorm) 활용
```

**CSRF 토큰:**
```
- 상태 변경 API (POST, PUT, DELETE)에 CSRF 토큰 필수
- 토큰 검증 미들웨어
```

### 7.4 PII (개인식별정보) 보호

**마스킹 정책:**
```javascript
// 전화번호 마스킹: 010-1234-5678 → 010-****-5678
function maskPhone(phone) {
  return phone.replace(/(.{3})(.*)(.{4})/, '$1-****-$3');
}

// 계좌번호 마스킹: 123-456-789012 → 123-***-789012
function maskAccount(account) {
  return account.replace(/(.{3})(.*)(.{6})/, '$1-***-$3');
}
```

**개인정보 접근 로깅:**
```
모든 PII 접근 시 감사 로그 기록
- 접근자 ID
- 접근 시간
- 접근 이유
- IP 주소
```

```yaml
      run: |
        kubectl set image deployment/eodisan-api eodisan-api=123456789.dkr.ecr.us-east-1.amazonaws.com/eodisan-api:${{ github.sha }} -n production
```
