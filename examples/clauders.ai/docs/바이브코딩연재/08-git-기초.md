> "코드 짜다가 다 날리면 어떡하지?"
> "어제께 더 나았는데 돌아갈 수 없나?"

개발자들이 가장 두려워하는 건 귀신이 아니라 **'실수로 지운 코드'**야.
그래서 우리는 타임머신을 써. 바로 **Git(깃)**이야.

이름이 왜 깃(Git)이냐고? 만든 사람이 "멍청이(Git)"라고 자학하며 지었다는 설이 있는데,
중요한 건 이게 **전 세계 개발자의 '세이브 파일 관리 시스템'**이라는 거야.

```
📚 이 글을 읽고 나면

✅ 코드를 '세이브(Commit)'할 수 있다
✅ Git이 게임 세이브와 똑같다는 걸 이해한다
✅ GitHub에 '내 코드 저장소'를 만들 수 있다
```

---

## Git = 게임 세이브 시스템

RPG 게임 해봤어?
보스 잡기 전에 꼭 뭐하지? **'저장(Save)'**을 하지.
죽으면 다시 그 시점으로 돌아가려고.

코딩도 똑같아.
"자, 이제 로그인 기능 건드릴 건데, 망치면 귀찮으니까 여기서 일단 저장!"

이걸 전문 용어로 **커밋(Commit)**이라고 해.

- **게임:** Save Point 1
- **코딩:** Commit ID (a1b2c3d...)

차이점은 딱 하나야.
게임은 '로드'하면 현재 상태가 사라지지만,
Git은 '평행 우주'를 만들 수 있어. (이건 나중에 '브랜치' 편에서 다룰게)

지금은 딱 하나만 기억해.
**Commit = 세이브 포인트 찍기**

---

## 1단계: 세이브 포인트 만들기 (로컬 저장)

터미널을 켜고, 아까 작업하던 폴더로 가봐.

### 1. Git 시작하기 (게임 설치)
이 폴더에서 "이제부터 여기서 세이브 기능을 쓸 거야"라고 선언해야 해.

```bash
git init
```

`Initialized empty Git repository`라고 뜨면 성공!

### 2. 세이브할 파일 고르기 (장바구니 담기)
수정한 파일이 100개라도, 그중에 3개만 저장하고 싶을 수 있잖아?
그래서 "저장할 파일"을 골라야 해. 이걸 `add`라고 해.

```bash
git add .
```
(점 `.`은 "모든 파일 다 담아!"라는 뜻이야)

### 3. 세이브하고 이름 붙이기 (Commit)
세이브 파일에 이름 안 붙이면 나중에 헷갈리잖아?
"보스 잡기 전", "아이템 획득 후"처럼.

```bash
git commit -m "첫 번째 세이브: 안녕 파일 생성"
```

이제 네 코드는 안전하게 박제됐어.
파일을 다 지워버려도, 이 시점으로 언제든 돌아올 수 있어.

---

## 2단계: 구름 위에 저장하기 (GitHub)

내 컴퓨터가 고장 나면 세이브 파일도 날아가겠지?
그래서 **'클라우드(서버)'**에 백업을 해. 그게 바로 **GitHub**이야.

1. [GitHub](https://github.com/) 회원가입 해.
2. 우측 상단 `+` 버튼 -> `New repository` 클릭
3. 이름 대충 짓고(예: `my-first-vibe`) `Create repository` 클릭

그럼 요상한 코드들이 나오는데, 다 무시하고 **"...or push an existing repository..."** 부분만 봐.
거기 있는 3줄을 복사해서, **내 터미널에 붙여넣기** 해.

```bash
git remote add origin https://github.com/... (주소)
git branch -M main
git push -u origin main
```

엔터 치고 로그인하라고 하면 해.
다 되면?
github 페이지를 새로고침 해봐. 네 코드가 거기 올라가 있을 거야!

## 정리: 3단계 주문만 외워

앞으로 코드를 수정할 때마다 이 3단계만 반복하면 돼.

1. **담기:** `git add .` (수정한 거 다 담아)
2. **저장:** `git commit -m "뭐뭐 수정했음"` (이름표 붙여서 세이브)
3. **올리기:** `git push` (구름 위로 발사)

![Git 흐름: 담기(Add) -> 이름표(Commit) -> 트럭(Push)](assets/08-git-flow.png)

```
add ➡️ commit ➡️ push
(담고 ➡️ 찍고 ➡️ 밀고)
```

이것만 알면 네 코드는 **절대** 날아가지 않아.
