> "터미널은 대화창이라면서. 근데 코드는 어디에 적어?"
> "메모장으로 하면 안 돼?"

메모장으로 해도 돼.
근데 요리사가 식칼 대신 커터칼로 요리하는 것과 같아. 할 수는 있지만, 무지하게 힘들지.

우리는 **VS Code (Visual Studio Code)**라는 전문가용 조리대를 쓸 거야.
전 세계 개발자의 70% 이상이 이걸 써.

왜 쓸까?
- **자동 완성:** `cl`만 쳐도 `console.log`를 추천해줌
- **에러 표시:** 틀린 문법은 빨간 줄로 알려줌
- **터미널 내장:** 따로 까만 창 안 열어도 됨

오늘은 이 최고의 조리대를 우리 부엌에 들여놓을 거야.

```
📚 이 글을 읽고 나면

✅ VS Code를 설치하고 설정할 수 있다
✅ 코드를 열어보고, 수정하고, 저장할 수 있다
✅ 필수 확장 프로그램(Extension)을 깔아서 기능을 강화할 수 있다
```

## 설치하기 (너무 쉬움 주의)

1. [VS Code 공식 홈페이지](https://code.visualstudio.com/) 접속
2. 파란색 'Download' 버튼 클릭
3. 설치 파일 실행해서 '다음, 다음, 완료'

끝이야.

---

## 1단계: VS Code 해부하기

VS Code를 켜면 뭔가 복잡해 보이지? 딱 3군데만 알면 돼.

1. **탐색기 (왼쪽 바):** 파일 목록 보는 곳 (냉장고)
2. **에디터 (가운데):** 코드 짜는 곳 (도마)
3. **터미널 (아래쪽):** 명령어 치는 곳 (가스레인지)

![VS Code 인터페이스 비교: 냉장고(탐색기), 도마(에디터), 가스레인지(터미널)](assets/07-vscode-workbench.png)

이 3개가 한 화면에 다 있어서 편한 거야.

> **💡 터미널이 안 보이면?**
> 상단 메뉴에서 `Terminal` -> `New Terminal`을 누르거나
> 단축키 `Ctrl + \`` (물결 표시 키)를 눌러봐.

## 2단계: 필수 확장 프로그램 설치

VS Code는 '확장 프로그램(Extension)'이라는 걸로 기능을 추가할 수 있어.
스마트폰에 앱 까는 거랑 똑같아.

왼쪽 바에서 '네모 4개 아이콘(Extensions)'을 눌러.
그리고 검색창에 딱 하나만 검색해서 설치하자.

### 1. Korean Language Pack (한국어 패키지)
- 검색: `Korean`
- 맨 위에 나오는 'Korean Language Pack for Visual Studio Code' 클릭
- 'Install' 버튼 클릭
- 오른쪽 아래에 'Restart' 버튼 뜨면 클릭

이제 메뉴가 한글로 바뀔 거야. 마음이 편안해지지?

## 3단계: 터미널에서 VS Code 열기

자, 여기서 "바이브 코더"만의 꿀팁 나간다.
보통은 VS Code 아이콘 클릭해서 열고 -> 파일 -> 폴더 열기... 이렇게 하잖아?

우리는 터미널 전사들이잖아. 멋있게 열어보자.

1. 터미널을 켜.
2. 작업하고 싶은 폴더로 이동 (예: `cd my-project`)
3. 이렇게 쳐:

```bash
code .
```

`code`는 "VS Code 실행해!", `.`(점)은 "현재 폴더를 열어!"라는 뜻이야.
엔터를 치면?
**방금 그 폴더가 VS Code로 딱! 열려.**

이게 진짜 개발자 바이브지.

---

## 실습: 파일 만들고 저장해보기

1. **code .** 명령어로 VS Code를 열어.
2. **탐색기(왼쪽)** 빈 공간에 우클릭 -> `새 파일`
3. 이름 입력: `hello.txt`
4. 내용 입력: `안녕! 바이브 코딩이야.`
5. **저장:** `Ctrl + S` (맥은 `Command + S`)

> **🚨 주의:**
> 파일 이름 옆에 **동그라미(●)**가 떠 있으면 "저장 안 됨"이라는 뜻이야.
> 저장하면 **X** 표시로 바뀌어.
> AI한테 "코드 고쳤어"라고 하기 전에 꼭 저장했는지 확인해!
