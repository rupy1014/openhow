**지금 배운 걸 써먹지 않으면 3일 안에 잊어.**

숙제는 하나야. 나만의 연재 작업 환경을 만들고 1화를 발행해봐.

---

## 뭘 만들까?

블로그가 됐든, 책이 됐든, 라이트노벨이 됐든 상관없어.

조건은 하나 — **연재**. 1회로 끝내지 않아. 최소 5화 이상 계획을 세워.

주제 예시:
- 카페 창업 준비 일기 (블로그)
- 직장인 부업 도전기 (블로그)
- 판타지 단편 시리즈 (라이트노벨)
- 내가 배운 AI 정리 (책)

뭐든 돼. 지금 연재하고 싶었던 것, 막막해서 못 시작했던 것.

---

## 어떻게 시작해?

**Step 1.** 이전 글(01-작업맡기기)을 다시 열어봐

세팅 내용을 그대로 따라가면 돼.

**Step 2.** 새 폴더 만들고 환경 세팅

```bash
mkdir my-series && cd my-series
dsclaude
```

세팅 내용을 통째로 복사해서 붙여넣으면 파일 구조가 자동으로 만들어져.

**Step 3.** `CLAUDE.md`와 `docs/tone-and-manner.md`를 내 것으로 고쳐

블로그 이름, 독자, 말투를 실제 내 것으로 채워. 이 두 파일이 모든 담당자의 기준이 돼. 안 바꾸면 `/write`가 엉뚱한 글을 써.

**Step 4.** 기획 실행

```
/plan "[내 시리즈 제목]" 10편
```

**Step 5.** 1화 발행

```
/write 1화
```

---

## 왜 직접 해봐야 해?

강의 보는 것과 손으로 만드는 건 달라.

에러가 나고, 예상대로 안 되고, 다시 고치는 과정에서 실력이 생겨. 강의만 보면 이해한 것 같아도 막상 혼자선 못 만들어. AI 공부도 마찬가지야.

매일 30분이든 1시간이든 상관없어. **매일이 중요해.**

AI 도구는 매주 바뀌어. 주 1회 몰아서 하면 따라가기 어려워. 짧아도 매일 손을 대는 사람이 6개월 뒤에 완전히 달라져.
