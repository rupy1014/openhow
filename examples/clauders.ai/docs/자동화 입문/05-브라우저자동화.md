**Claude Code한테 "네이버 로그인해봐" 하면 진짜로 해.**

브라우저를 열고, 아이디 치고, 비밀번호 넣고, 로그인 버튼까지 누르는 걸 직접 해.

근데 기본 상태에선 못 해. **Playwright MCP**를 깔아야 돼.

---

## Playwright MCP가 뭐야?

Claude Code는 기본적으로 눈이 없어. 코드는 짜는데, 브라우저 화면은 못 봐.

Playwright MCP를 깔면 눈이 생겨. 브라우저를 열고, 클릭하고, 입력하고, 스크린샷까지 찍을 수 있게 돼.

쉽게 말하면 **Claude Code 전용 브라우저 리모컨**이야.

---

## 어떻게 설치해?

**Step 1.** 터미널에서 아래 한 줄 복사해서 붙여넣어

```bash
claude mcp add playwright -- npx @playwright/mcp@latest
```

끝. 이게 전부야.

**Step 2.** Claude Code를 껐다 다시 켜

```bash
claude
```

**Step 3.** 설치됐는지 확인해봐

```
/mcp
```

목록에 `playwright`가 보이면 성공이야.

---

## 네이버 로그인 시켜봐

설치 확인 겸 바로 써보자. Claude한테 이렇게 말해봐:

```
네이버(naver.com)에 로그인해봐.
아이디: myid123
비밀번호: mypass456
```

Claude가 하는 일:

1. 브라우저를 열어
2. naver.com으로 이동해
3. 로그인 페이지를 찾아
4. 아이디, 비밀번호를 입력해
5. 로그인 버튼을 눌러

중간중간 스크린샷을 찍어서 "지금 이 화면이야"라고 보여줘.

> 캡챠(보안 문자)가 뜰 수 있어. 네이버가 자동 로그인을 막거든. 그건 정상이야. 중요한 건 **Claude가 브라우저를 직접 조작한다**는 거야.

---

## 로그인 말고 뭘 더 할 수 있어?

로그인은 맛보기야. 진짜 쓸모는 이런 거야:

- "내 블로그 들어가서 글 목록 스크린샷 찍어와"
- "이 URL 열어서 버튼이 제대로 동작하는지 확인해"
- "구글 스프레드시트 열어서 데이터 읽어와"
- "관리자 페이지에서 설정값 바꿔줘"

브라우저로 할 수 있는 건 다 시킬 수 있어.

손으로 마우스 클릭하던 반복 작업 — 이제 말 한마디로 끝나.

---

## 한 줄 정리

Playwright MCP 한 줄이면 Claude Code에 눈이 생겨. 브라우저를 보고 직접 조작할 수 있게 돼.
