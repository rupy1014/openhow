## 뭘 할 수 있어?

크게 두 가지다.

**자동화** — 반복 작업을 넘긴다

```
이 폴더 글 전부 읽고 각각 3줄 요약 만들어줘
```

```
post-1.md부터 post-10.md 제목 뽑아서 index.md에 정리해줘
```

파일이 많아도 Claude가 다 읽고 처리한다. 내가 할 필요 없다.

---

**바이브코딩** — 코드 몰라도 만든다

```
링크 모음 페이지 만들어줘. 카드 형태로, 배경은 어둡게.
```

```
방문자가 이메일 남길 수 있는 폼 추가해줘
```

원하는 걸 말로 하면 Claude가 HTML, CSS, JS를 직접 짠다.

---

## 한 줄 정리

자동화로 반복을 없애고, 바이브코딩으로 만드는 걸 시작한다. 둘 다 같은 방식이다. 말로 시키면 된다.
