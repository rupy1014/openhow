**스크립트 한 줄 붙여넣으면 Claude Code가 설치된다. Node.js, Git 포함.**

---

## 어떻게 설치하냐

**Step 1.** 터미널 열기
- **Windows**: 시작 → "PowerShell" 검색 → **관리자 권한으로 실행** (중요)
- **Mac**: ⌘+Space → "터미널" 검색 → Enter

**Step 2.** 아래 스크립트를 복사해서 붙여넣고 Enter

**Step 3.** 설치 완료 후 확인
```bash
claude
```
`>` 프롬프트가 뜨면 성공.

**Step 4.** [claude.ai](https://claude.ai)에서 **Pro 플랜** 결제 ($20/월)

Claude Code는 claude.ai Pro 계정이 있어야 제대로 쓴다.

- Git 설치: https://youtu.be/FrYgchuu3xE
- Claude Code 설치: https://youtu.be/7Spk6medZY8
:::

## 직접 해보기

**Windows** — PowerShell을 관리자 권한으로 실행:
```powershell
irm https://raw.githubusercontent.com/rupy1014/base-install/main/window-install.ps1 | iex
```

**Mac** — 터미널에서 실행:
```bash
curl -fsSL https://raw.githubusercontent.com/rupy1014/base-install/main/mac-install.sh | bash
```
