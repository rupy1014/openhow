**AI한테 말로 시키면, AI가 직접 파일을 만들고 고쳐.**

이게 Claude Code야.

---

## Claude Code가 뭐야?

Claude를 만든 회사(Anthropic)가 만든 AI 도구야. 터미널(검은 창)에서 실행해.

ChatGPT나 Claude.ai처럼 대화하는 방식이 맞아. 근데 결정적으로 다른 게 있어.

웹 AI는 **말로** 알려줘. "이렇게 하면 돼요."
Claude Code는 **직접** 해. 파일을 열고, 만들고, 수정하고, 저장하게 돼.

```
> 이 폴더 글 전부 읽고 각각 3줄 요약 만들어줘

post-1.md 읽는 중...
post-2.md 읽는 중...
summaries.md 생성 중...
완료.
```

내가 코드를 짜는 게 아니야. Claude Code가 짜.
나는 원하는 걸 말하기만 하면 돼.

---

## 웹 AI랑 뭐가 다른 거야?

웹 AI는 내 컴퓨터 파일에 손을 못 대.
코드를 알려줘도 내가 직접 복사해서 파일에 붙여넣어야 해.

Claude Code는 내 컴퓨터 안에서 실행돼.
파일을 직접 읽고, 만들고, 고쳐. 내가 손댈 필요 없어.

| | 웹 AI | Claude Code |
|---|---|---|
| 어디서 실행돼? | 브라우저 (크롬 같은 거) | 내 컴퓨터 터미널 |
| 파일 만져? | 못 만져 | 직접 읽고, 만들고, 고쳐 |
| 설정 저장은? | 웹 계정에 저장 | 내 폴더 안에 파일로 저장 |
| 다시 쓸 때? | 매번 다시 설명해야 해 | 설정 파일 한 번 만들면 끝 |

---

## 왜 배워야 해?

AI한테 "이렇게 해줘"라고 말만 해도 실제 결과물이 나와.

페이지를 만들고 싶으면 만들고, 글을 쓰고 싶으면 쓰고, 반복 작업을 자동으로 돌리고 싶으면 돌려.
코딩을 몰라도 돼. 필요한 건 원하는 걸 설명하는 능력뿐이야.

그리고 한 번 설정을 잡아두면 다음부터는 실행만 하면 돼.
"말투는 구어체로"라고 한 번 써두면 매번 설명 안 해도 돼.

---

## 이 섹션에서 뭘 배워?

세 가지를 순서대로 해.

**설치** — 스크립트 한 줄로 Claude Code를 깔아.

**프로젝트 설정** — 폴더마다 다른 규칙을 잡는 방법. 블로그 폴더엔 블로그 규칙, 업무 폴더엔 업무 규칙이 자동으로 붙게 돼.

**Hello World** — 직접 말을 걸어봐. 어떻게 말해야 잘 나오는지, 계획 먼저 받는 방법도 익혀.
