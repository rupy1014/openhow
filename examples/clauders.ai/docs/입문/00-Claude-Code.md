**AI한테 말로 시키면, AI가 직접 파일을 만들고 고친다.**

이게 Claude Code다.

---

## Claude Code가 뭐야?

Anthropic이 만든 AI 코딩 도구야. 터미널에서 실행한다.

ChatGPT나 Claude.ai처럼 대화하는 방식이 맞아. 근데 결정적으로 다른 게 있어.

웹 AI는 **말로** 알려준다. "이렇게 하면 돼요."
Claude Code는 **직접** 한다. 파일을 열고, 만들고, 수정하고, 저장한다.

```
> 이 폴더 글 전부 읽고 각각 3줄 요약 만들어줘

post-1.md 읽는 중...
post-2.md 읽는 중...
summaries.md 생성 중...
완료.
```

내가 코드를 짜는 게 아니야. Claude Code가 짠다.
나는 원하는 걸 말하기만 하면 된다.

---

## 웹 AI랑 뭐가 다른 거야?

웹 AI는 내 컴퓨터 파일에 접근을 못 해.
코드를 알려줘도 내가 직접 복사해서 파일에 붙여넣어야 한다.

Claude Code는 내 컴퓨터 안에서 실행돼.
파일을 직접 읽고, 만들고, 고친다. 내가 손댈 필요 없어.

| | 웹 AI | Claude Code |
|---|---|---|
| 실행 위치 | 브라우저 | 내 컴퓨터 터미널 |
| 파일 접근 | 불가 | 직접 읽고 씀 |
| 설정 | 계정에 저장 | 프로젝트 폴더에 저장 |
| 재현성 | 매번 다시 설명 | 설정 한 번으로 반복 |

---

## 왜 배워야 해?

AI한테 "이렇게 해줘"라고 말만 해도 실제 결과물이 나온다.

페이지를 만들고 싶으면 만들고, 글을 쓰고 싶으면 쓰고, 반복 작업을 자동화하고 싶으면 자동화한다.
코딩 지식은 없어도 돼. 필요한 건 원하는 걸 설명하는 능력뿐이야.

그리고 한 번 환경을 잡아두면 다음부터는 실행만 하면 돼.
"말투는 구어체로"라고 한 번 써두면 매번 설명 안 해도 된다.

---

## 이 섹션에서 뭘 배워?

세 가지를 순서대로 한다.

**설치** — 스크립트 한 줄로 Claude Code를 설치한다. Node.js, Git까지 한 번에 끝난다.

**프로젝트 설정** — `CLAUDE.md`로 폴더마다 다른 설정을 잡는 방법. 이걸 알면 블로그 폴더엔 블로그 규칙, 업무 폴더엔 업무 규칙이 자동으로 적용된다.

**Hello World** — 직접 요청해본다. 어떻게 말해야 잘 나오는지, 플랜 먼저 받는 방법도 익힌다.
