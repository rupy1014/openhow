## 결과가 좋아지는 팁

**플랜 먼저 받기** — 복잡한 작업은 바로 시키지 말고 계획부터 받는다.

```
이력서 페이지에 프로젝트 섹션 추가하고 싶어. 일단 어떻게 할지 계획만 세워줘. 코드는 아직 짜지 마.
```

계획 확인 후 마음에 들면 "진행해줘"라고 하면 된다.
`Shift+Tab` 두 번으로 플랜 모드를 켤 수도 있다.

---

**레퍼런스 주기** — 참고할 URL이나 파일을 같이 넘기면 결과가 훨씬 좋다.

```
참고: https://resume-three-virid.vercel.app/
이 스타일이랑 비슷하게 만들어줘
```

```
post-1.md 파일 말투 참고해서 나머지 글들도 같은 톤으로 바꿔줘
```

Claude가 설계부터 시작하지 않고 레퍼런스를 보고 맞춰서 짠다.
