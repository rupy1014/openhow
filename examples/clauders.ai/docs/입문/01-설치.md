**스크립트 한 줄 붙여넣으면 Claude Code가 설치된다. Node.js, Git 포함.**

---

## 어떻게 설치해?

**Step 1.** 터미널 열기

터미널 처음이야? 검은 창에 텍스트 치는 프로그램이야. 겁먹지 않아도 돼.

- **Windows**: 시작 → "PowerShell" 검색 → **관리자 권한으로 실행** (중요)
- **Mac**: ⌘+Space → "터미널" 검색 → Enter

**Step 2.** 아래 스크립트를 복사해서 붙여넣고 Enter

**Windows** — PowerShell을 관리자 권한으로 실행:
```powershell
irm https://raw.githubusercontent.com/rupy1014/base-install/main/window-install.ps1 | iex
```

**Mac** — 터미널에서 실행:
```bash
curl -fsSL https://raw.githubusercontent.com/rupy1014/base-install/main/mac-install.sh | bash
```

뭔가 죽 설치되는 게 보일 거야. 기다리면 끝난다.

**Step 3.** 설치 완료 후 확인

```bash
claude
```

`>` 프롬프트가 뜨면 성공이야. Claude가 준비된 것.

`dsclaude`도 같이 등록됐어.
`claude --dangerously-skip-permissions`의 단축 명령어야.
매번 "이 파일 수정해도 돼?" 같은 권한 확인 없이 바로 실행된다.

내 컴퓨터, 내 프로젝트에서 쓰는 거라면 편하게 써도 돼.
다만 처음엔 `claude`로 시작해서 Claude가 뭘 하는지 보면서 익숙해지는 걸 추천해.
익숙해지면 `claude` 대신 `dsclaude`를 더 많이 쓰게 될 거야.

**Step 4.** [claude.ai](https://claude.ai)에서 **Pro 플랜** 결제 ($20/월)

무료 플랜으론 금방 한도에 걸려. Pro여야 제대로 쓸 수 있어.

---

## 뭐가 설치됐어?

스크립트 한 번에 세 가지가 깔린다.

- **Node.js** — Claude Code가 돌아가는 런타임. 엔진 같은 거야.
- **Git** — 버전 관리 도구. Claude Code가 내부적으로 쓴다.
- **Claude Code** — AI 코딩 도구 본체

따로 하나씩 설치할 필요 없이 스크립트가 순서대로 알아서 처리해줘.
