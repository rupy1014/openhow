## 어떻게 설치해?

**Step 1.** 터미널 열기

터미널 처음이야? 검은 창에 텍스트 치는 프로그램이야. 겁먹지 않아도 돼.

- **Windows**: 시작 → "PowerShell" 검색 → **관리자 권한으로 실행** (중요)
- **Mac**: ⌘+Space → "터미널" 검색 → Enter

**Step 2.** 아래 스크립트를 복사해서 붙여넣고 Enter

**Windows** — PowerShell을 관리자 권한으로 실행:
```powershell
irm https://raw.githubusercontent.com/rupy1014/base-install/main/window-install.ps1 | iex
```

**Mac** — 터미널에서 실행:
```bash
curl -fsSL https://raw.githubusercontent.com/rupy1014/base-install/main/mac-install.sh | bash
```

뭔가 죽 설치되는 게 보일 거야. 기다리면 끝나.

**Step 3.** 설치 완료 후 확인

```bash
claude
```

`>` 표시가 뜨면 성공이야. Claude가 준비된 거야. 여기에 하고 싶은 말을 치면 돼.

`dsclaude`도 같이 등록됐어. `claude`와 뭐가 다르냐면:

| | `claude` | `dsclaude` |
|---|---|---|
| 파일 수정 전 | "이 파일 수정해도 돼?" 물어봄 | 바로 수정함 |
| 속도 | 느림 (매번 확인) | 빠름 (확인 생략) |
| 추천 시점 | 처음 배울 때 | 익숙해진 뒤 |

`dsclaude`는 `claude --dangerously-skip-permissions`의 단축 명령어야.

## 권한 줘도 괜찮아?

**괜찮아.** Claude Code는 내 컴퓨터, 내 폴더 안에서만 작동해.

할 수 있는 것:
- 현재 폴더 안의 파일을 읽고, 만들고, 수정해
- 터미널 명령어를 실행해 (내가 손으로 치는 것과 똑같아)

할 수 없는 것:
- 인터넷으로 결제하거나 메일 보내거나 안 해
- 내가 연 폴더 바깥의 시스템 파일은 건드리지 않아

`claude`로 시작하면 매번 "이거 해도 돼?"라고 물어보니까 Claude가 뭘 하는지 직접 보면서 익숙해질 수 있어. 익숙해지면 `dsclaude`로 바꾸면 돼.

## dsclaude가 안 돼?

위 스크립트로 설치했는데도 `dsclaude`가 안 되면, 직접 등록하면 돼.

**Mac (zsh)** — 터미널에 아래를 복사해서 붙여넣고 Enter:

```bash
echo 'alias dsclaude="claude --dangerously-skip-permissions"' >> ~/.zshrc && source ~/.zshrc
```

**Mac (bash)** — `.zshrc` 대신 `.bashrc`로:

```bash
echo 'alias dsclaude="claude --dangerously-skip-permissions"' >> ~/.bashrc && source ~/.bashrc
```

등록 후에도 안 되면 **터미널을 완전히 끄고 다시 열어**봐. 새 터미널에서 `dsclaude`를 치면 돼.

**Step 4.** [claude.ai](https://claude.ai)에서 **Pro 플랜** 결제 ($20/월)

무료 플랜으론 금방 한도에 걸려. Pro여야 제대로 쓸 수 있어.

## 뭐가 설치됐어?

스크립트 한 번에 세 가지가 깔려.

- **Node.js** — Claude Code를 돌려주는 엔진. 자동차의 엔진 같은 거야.
- **Git** — 파일 변경 기록을 저장하는 도구. Claude Code가 안에서 몰래 써.
- **Claude Code** — AI 도구 본체. 이걸로 AI한테 일을 시켜.

세 개를 따로 설치할 필요 없어. 스크립트가 알아서 순서대로 깔아줘.
