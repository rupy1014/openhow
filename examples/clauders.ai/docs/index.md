# AI를 내 손발로 만들기

> 코딩 몰라도 된다. 시키는 법만 알면 된다.

:::success 스터디가 끝나면
- 말로 시키면 AI가 파일을 만들고 고친다
- 반복 작업을 자동으로 돌리는 환경을 직접 만든다
- "이건 AI한테 시키면 되겠다"는 눈이 생긴다
:::
