# AI를 내 손발로 만들기

> 코딩 몰라도 돼. 시키는 법만 알면 돼.

:::success 스터디가 끝나면
- 말로 시키면 AI가 파일을 만들고 고쳐
- 반복 작업을 자동으로 돌리는 환경을 직접 만들어
- "이건 AI한테 시키면 되겠다"는 눈이 생겨
:::

**대상**: AI 처음이거나 제대로 못 쓰고 있는 사람
**도구**: Claude Code — 터미널에서 AI한테 일을 시키는 도구
**비용**: Claude Pro ($20/월) 필요
