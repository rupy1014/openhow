# AI가 대신 일하게 만드는 법 - 무료 입문 강좌

> "나도 AI한테 일 시켜보고 싶은데, 뭐부터 해야 하지?" 그 첫 번째 답을 드립니다. 실제로 AI에게 일을 시켜서 생산성을 10배 올린 사람이, 화면을 켜고 직접 보여드립니다.

---

## 먼저 결과부터 보여드립니다

이건 한 사람이 AI에게 시켜서 나오는 결과물입니다.

| 채널 | 발행량 | 방법 |
|------|--------|------|
| **SNS (Threads)** | 매일 10개 | 뉴스 자동 수집 → AI가 글 작성 → 승인만 클릭 |
| **블로그** (max5.ai) | 매일 10개 | 같은 소재 → 블로그용으로 변환 → 자동 발행 |
| **유튜브** | 1일 1편 롱폼 | 대본 → 나레이션 → 편집까지 자동화 |
| **웹소설** | 2,500편 (총 1,000만 자) | 캐릭터 250명 × 10편, 세계관 기반 자동 집필 |

본업(결제 플랫폼 개발)은 따로 하면서, 자동화로 돌려서 나오는 부산물입니다.

**"저 사람이 잘해서 그런 거 아냐?"**

아닙니다. 핵심은 딱 하나입니다.
**AI에게 규칙을 정해주고, 역할을 나눠주고, 시키는 방법**을 아는 것.
이 강좌에서 그 방법을 처음부터 보여드립니다.

---

## 이 강좌를 들으면

- AI에게 "글 써줘" 한마디 시키는 것과, **일하는 방식을 설계해서 시키는 것**의 차이를 알게 됩니다
- 내 관심 분야에서 AI가 대신 콘텐츠를 만들어주는 구조를 이해하게 됩니다
- "나도 해볼 수 있겠다"는 감이 잡힙니다

---

## 뭘로 하는 건가요?

**Claude Code**라는 도구를 씁니다.

ChatGPT처럼 AI와 대화하는 건 같은데, 한 가지가 다릅니다. 내 컴퓨터의 파일을 직접 읽고, 쓰고, 실행할 수 있습니다. "이 글 써서 저장해줘", "이 파일 읽고 요약해줘"를 말로 시키면 AI가 알아서 해줍니다.

코딩을 몰라도 됩니다. 규칙을 정해주고, 역할을 나눠주고, 시키면 됩니다.

---

## 이런 분을 위한 강좌입니다

- ChatGPT는 써봤는데, 그 이상 뭘 할 수 있는지 모르겠는 분
- AI 도구가 많은데 뭘 써야 할지 감이 안 잡히는 분
- 블로그, 유튜브, 뉴스레터 등 콘텐츠를 시작하고 싶은 분
- 1인 창업이나 사이드프로젝트를 준비 중인 분
- 코딩 경험 전혀 없어도 괜찮습니다

---

## 강좌 구성 (약 1시간)

**Part 1. "글 써줘"와 "일 시켜"의 차이**
AI에게 한 번 부탁하는 것과, 규칙과 역할을 정해서 시키는 것. 결과가 완전히 다릅니다. 왜 다른지, 어떻게 해야 하는지 설명합니다.

**Part 2. Claude Code, 이런 도구입니다**
어떤 도구인지, 왜 이걸 쓰는지, 첫 번째 명령은 어떻게 내리는지. 실제 화면으로 보여드립니다. (설치 방법은 별도 가이드로 안내드립니다)

**Part 3. 라이브 시연 - 눈앞에서 AI가 일합니다**
실제로 AI에게 명령을 내려서 콘텐츠가 만들어지는 과정을 라이브로 보여드립니다. SNS 자동 발행, 블로그, 유튜브, 웹소설까지. 수집 → 작성 → 퇴고 → 발행의 전체 흐름.

**Part 4. 오늘부터 시작하는 법**
거창하게 시작할 필요 없습니다. 관심 분야 정보를 모아서 정리하는 것부터. 단계별로 안내합니다.

## 더 깊이 배우고 싶다면 - 클로더즈 3기

이 강좌에서 보여드리는 건 맛보기입니다.
**4주 동안 직접 만들어보는 과정**이 클로더즈 스터디입니다.

| | 이 강좌 (무료) | 클로더즈 3기 (4주) |
|---|---|---|
| **내용** | AI에게 일 시키는 원리 이해 | 직접 나만의 자동화 시스템 구축 |
| **결과** | "이렇게 되는구나" | "내 것이 돌아간다" |
| **지원** | 1시간 강좌 | 매주 라이브 강의 + 1:1 커피챗 + 오피스아워 |
| **동료** | 없음 | 함께 만드는 스터디원 |
| **비용** | 무료 | 35만원 (완주 시 10만원 환급 → 실 25만원) |

1기 20명, 2기 20명이 함께 배우고 만들었습니다.
2기는 출석률 100%, 18명이 데모나잇에서 발표했습니다.

**3기 모집에 대한 자세한 안내는 강좌 마지막에 드립니다.**
