**배포 3단계:**
```
내 코드 → GitHub (코드 저장소) → Vercel (자동 서버 운영)
```

왜 GitHub을 거치나요? → Vercel이 GitHub에서 코드를 가져다 자동으로 배포해줘요.
한 번 연결해두면 `git push` 할 때마다 자동으로 업데이트돼요!

---

## ✅ Task 5-1: GitHub 레포지토리 생성

1. `github.com` 접속 → 로그인
2. 오른쪽 상단 **`+`** → **New repository**
3. Repository name: `my-ai-chat`
4. **Private** 선택 (API 키 관련 코드니까 비공개로)
5. **Create repository** 클릭
6. 생성된 페이지에서 HTTPS 주소 복사 (예: `https://github.com/내이름/my-ai-chat.git`)

## ✅ Task 5-3: Vercel 배포

이제 Vercel에서 GitHub 레포지토리를 연결해요.

1. `vercel.com` 접속
2. **Sign Up** → **Continue with GitHub** (GitHub으로 로그인)
3. **Add New** → **Project**
4. GitHub 레포 목록에서 `my-ai-chat` 찾아서 **Import**

## 배포 에러가 났을 때

**"Build Failed"** 화면이 나왔나요? 당황하지 마세요.

1. Vercel 대시보드 → 내 프로젝트 → **Deployments** 탭
2. 실패한 배포 클릭 → **Logs** 탭
3. 빨간 에러 메시지 전체 복사
4. Claude Code에 붙여넣기 + "Vercel 배포 에러야. 이 로그 보고 고쳐줘"
5. 수정 후 터미널에서:

```bash
git add .
git commit -m "배포 에러 수정"
git push
```

Vercel이 자동으로 다시 배포를 시작해요! (2~3분 기다리기)

## 이제부터는 자동 배포!

한 번 연결해두면, 앞으로는 이렇게만 하면 돼요:

```bash
# 코드 수정 후
git add .
git commit -m "변경사항 설명"
git push
```

→ Vercel이 자동으로 감지 → 자동으로 다시 배포 → 2~3분 후 업데이트 완료!
