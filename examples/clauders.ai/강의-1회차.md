### 오늘의 규칙

1. **코드를 이해하려 하지 마세요** — 나중에 쌓이면 됩니다
2. **구체적으로 시키세요** — "채팅 앱"보다 "도비 캐릭터, 남색 배경, 말풍선 UI"
3. **에러 나면 AI에게 붙여넣으세요** — 빨간 글씨 복사 → Claude Code에 붙여넣기

## 0:15 – 0:45 | 환경 세팅

### ✅ Task: Node.js 설치

`nodejs.org` → **LTS 버전** 다운로드 → 설치

```bash
node --version   # v20 이상이면 OK
```

### ✅ Task: Claude Code 설치

```bash
npm install -g @anthropic-ai/claude-code
claude --version   # 버전 번호가 나오면 OK
```

## 1:00 – 1:45 | 첫 채팅 서비스 만들기

### ✅ Task A: 프로젝트 생성

터미널에서:
```bash
cd ~/Desktop
claude
```

Claude Code가 실행되면 붙여넣기:

  <p style="margin: 0; white-space: pre-wrap;">Next.js 14 + TypeScript + Tailwind CSS로 AI 채팅 앱 만들어줘.

프로젝트 이름: my-ai-chat
설정: App Router, TypeScript, Tailwind CSS, ESLint
생성 후 npm install까지 완료해줘.</p>

완료 후:
```bash
cd my-ai-chat
npm run dev   # localhost:3000 확인
```

### ✅ Task B: 채팅 UI + AI 연결 한 번에

새 터미널 창에서:
```bash
cd ~/Desktop/my-ai-chat
claude
```

붙여넣기:

  <p style="margin: 0; white-space: pre-wrap;">아래 두 파일을 동시에 만들어줘.

[1] app/page.tsx — 채팅 UI
캐릭터: 도비 🧝, "Harry Potter의 충직한 하우스엘프"
디자인: 어두운 남색 배경(#0f172a), 말풍선 채팅 UI
- AI 메시지: 왼쪽, 회색 말풍선
- 내 메시지: 오른쪽, 보라색 말풍선
- 하단 입력창 + 전송 버튼
전송 시: /api/chat POST 호출, 로딩 중 "..." 표시
처음 열면 도비의 인사말 자동 표시
자동 스크롤

[2] app/api/chat/route.ts — AI 연결
OpenRouter (https://openrouter.ai/api/v1/chat/completions)
모델: openai/gpt-4o-mini
API키: 환경변수 OPENROUTER_API_KEY
시스템 프롬프트:
"당신은 도비입니다. Harry Potter의 충직한 하우스엘프.
규칙: 항상 '도비는/도비가'로 자신을 지칭, 마지막에 '주인님' 붙이기,
도움 못 주면 자책, 절대 캐릭터 이탈 금지"

TypeScript 타입 안전하게 작성</p>
</div>
</div>

### ✅ Task C: .env.local 생성

Claude Code에 입력:
```
프로젝트 루트에 .env.local 파일 만들어줘.
내용: OPENROUTER_API_KEY=여기에_실제_키_넣기
```

VS Code에서 `.env.local` 열기 → `여기에_실제_키_넣기` → 실제 OpenRouter 키로 교체

### ✅ Task D: 테스트

브라우저에서 `http://localhost:3000` → 메시지 입력 → 전송

**성공 기준:** "..." 로딩 후 도비 말투로 AI가 대답하면 완성!
