## 번역 품질 — 직접 비교가 제일 빨라

벤치마크 도구 설치할 필요 없어.
같은 기사를 로컬 vs 클라우드로 번역해서 비교하면 돼.

```bash
# 1. 로컬 (Qwen)
curl -s http://localhost:8000/v1/chat/completions \
  -H "Authorization: Bearer vllm-local" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "Qwen/Qwen3.5-122B-A10B",
    "temperature": 0.3,
    "messages": [{
      "role": "user",
      "content": "다음 영문 기사를 한국어 뉴스 기사체로 번역해줘:\n\n[기사 원문]"
    }]
  }' | python3 -c "import sys,json; print(json.load(sys.stdin)['choices'][0]['message']['content'])"
```

```bash
# 2. 클라우드 (GPT-4o) — 같은 기사로
curl -s https://api.openai.com/v1/chat/completions \
  -H "Authorization: Bearer $OPENAI_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "temperature": 0.3,
    "messages": [{
      "role": "user",
      "content": "다음 영문 기사를 한국어 뉴스 기사체로 번역해줘:\n\n[같은 기사 원문]"
    }]
  }' | python3 -c "import sys,json; print(json.load(sys.stdin)['choices'][0]['message']['content'])"
```

10개 정도 비교하면 감이 와.
번역/요약용으로는 Qwen급 로컬 모델이면 대부분 충분해.

---

## 코딩 능력 — HumanEval+

"코딩에도 쓸 수 있을까?" 궁금하면 이걸 돌려봐.

```bash
pip install evalplus

# vLLM에 직접 연결 (164문제, 약 30분)
evalplus.evaluate \
  --model Qwen/Qwen3.5-122B-A10B \
  --dataset humaneval \
  --backend vllm \
  --base-url http://localhost:8000/v1
```

| 모델 | HumanEval+ 점수 |
|------|-----------------|
| GPT-4o | ~87% |
| Claude Sonnet 4.5 | ~89% |
| Qwen3 235B (풀사이즈) | ~85% |
| Qwen3.5 122B (fp4 퀀트) | 직접 측정해봐 |

**70% 이상이면 보조용으로 충분해.**

---

## RAG 능력 — RGB 벤치마크

RAG(문서 검색 + 답변)에 쓸 거면 이걸로 확인해:

```bash
git clone https://github.com/chen700564/RGB.git
cd RGB

python evaluate.py \
  --model_name Qwen/Qwen3.5-122B-A10B \
  --api_base http://localhost:8000/v1 \
  --api_key vllm-local
```

RGB가 측정하는 4가지:

| 항목 | 뭘 보는 거야 |
|------|------------|
| 노이즈 내성 | 관련 없는 문서가 섞여도 답을 찾아? |
| 부정 거부 | 모르면 "모른다"고 해? |
| 정보 통합 | 여러 문서에서 정보를 합쳐? |
| 반사실 견고성 | 잘못된 정보를 걸러내? |

## 퀀트하면 성능이 많이 떨어져?

fp4 퀀트는 풀사이즈 대비 **3~8%p 하락**이야.

근데:
- 번역/요약에는 체감 차이 거의 없어
- 코딩/수학에서 차이가 더 크게 나와
- 결국 직접 비교해보는 게 정확해

---

## 뭘 먼저 해봐?

```
"번역/요약에 쓸 건데" → 기사 10개 비교. 벤치마크보다 빠르고 정확해.

"코딩에 쓸 건데" → HumanEval+ 돌려봐. 70% 이상이면 OK.

"RAG에 쓸 건데" → RGB로 노이즈 내성이랑 부정 거부 확인해.
```

## 한 줄 정리

번역 품질은 기사 10개 비교, 코딩은 HumanEval+ 70% 이상이면 충분해. 퀀트 모델도 번역/요약에는 거의 차이 없어.
