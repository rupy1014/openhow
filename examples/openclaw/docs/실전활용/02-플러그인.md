*검증 기준일: 2026-02-26 (공식 문서·릴리즈 기준)*

**ClawHub는 OpenClaw의 앱스토어야.**

2026-02-07 기준으로 5,705개 이상의 스킬(플러그인)이 등록돼 있어.
필요한 기능을 검색하고, 한 줄로 설치해.

---

## ClawHub CLI 설치

```bash
npm i -g clawhub
```

---

## 스킬 찾기

```bash
# 시맨틱 검색 (키워드가 아니라 의미로 찾아줘)
clawhub search "postgres 백업"
clawhub search "calendar" --limit 20

# 둘러보기
clawhub explore

# 설치 전 상세 정보 확인
clawhub inspect 스킬이름
```

검색은 OpenAI 임베딩 기반이야. "DB 백업"이라고 쳐도 postgres, mysql 관련 스킬이 다 나와.

---

## 설치 & 관리

```bash
# 설치
clawhub install 스킬이름
clawhub install 스킬이름 --version 1.2.0    # 특정 버전

# 업데이트
clawhub update --all                         # 전체 업데이트
clawhub update 스킬이름                       # 개별 업데이트

# 확인 & 삭제
clawhub list                                 # 설치된 스킬 목록
clawhub uninstall 스킬이름                    # 삭제
```

npx로 바로 설치도 돼:

```bash
npx clawhub@latest install 스킬이름
```

## awesome-openclaw-skills

커뮤니티가 검증한 큐레이션 목록이야.

2026-02-07 공개 목록 기준으로 5,705개 중 스팸(1,180), 암호화폐(672), 중복(492), 악성(396)을 걸러서 **2,868개**만 남겼어.

GitHub: `github.com/VoltAgent/awesome-openclaw-skills`

**스킬 고를 때 이 순서로 봐:**
1. awesome 목록에 있는지 확인
2. GitHub 스타 수 확인
3. VirusTotal 스캔 결과 확인 (ClawHub 페이지에 표시돼)
4. SKILL.md 직접 읽어보기

## 스킬 설정하기

설치한 스킬에 API 키나 옵션을 넣으려면:

```json5
{
  skills: {
    entries: {
      "opencode-controller": {
        enabled: true,
        config: {
          apiUrl: "http://127.0.0.1:4096",
          maxIterations: 50,
          timeoutMs: 180000
        }
      },
      "web-search": {
        enabled: true,
        env: { SEARCH_API_KEY: "너의키" }
      }
    }
  }
}
```

설정 바꾸면 `openclaw gateway restart` 잊지 마.

## 직접 스킬 만들기

폴더 하나에 `SKILL.md` 파일만 있으면 돼.

```
내스킬/
  SKILL.md          ← 필수
  scripts/           ← 선택 (보조 스크립트)
```

### SKILL.md 형식

```markdown
---
name: my-skill
description: 이 스킬이 뭘 하는지 (검색 순위에 영향!)
version: 1.0.0
metadata:
  clawdbot:
    requires:
      env: [MY_API_KEY]
      bins: [curl, jq]
    emoji: "🔧"
    homepage: https://github.com/너/my-skill
    os: [macos, linux]
---

# My Skill

에이전트가 이 스킬을 어떻게 써야 하는지 설명.
자연어로 쓰면 돼.

## 언제 써?
이런 상황에서 이 스킬을 활성화해.

## External Endpoints
| URL | 전송 데이터 |
|-----|-----------|
| https://api.example.com | 쿼리 파라미터만 |
```

> **주의**: frontmatter 키는 배포 환경/버전에 따라 다를 수 있어.
> 설치된 스킬의 `SKILL.md`를 먼저 열고 동일한 키 구조를 그대로 맞춰.

### 퍼블리시

```bash
clawhub login                    # GitHub OAuth 인증
clawhub publish ./내스킬 \
  --slug my-skill \
  --version 1.0.0 \
  --changelog "첫 릴리스"

# 여러 스킬 한 번에
clawhub sync --all --dry-run     # 미리보기
clawhub sync --all               # 실제 퍼블리시
```

퍼블리시 후 자동 검증(문법, 권한, 중복, VirusTotal 스캔)을 거쳐. 보통 5분 이내.

## 스킬 품질 연구 결과

SkillsBench 논문 결론:
- AI가 스스로 만든 스킬은 **성능이 오히려 떨어져** (평균 -1.3pp)
- 사람이 큐레이션한 스킬은 **+16.2pp 성능 향상**
- **작업당 2~3개가 최적**. 너무 많으면 역효과
