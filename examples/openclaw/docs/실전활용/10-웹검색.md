*검증 기준일: 2026-03-03 (공식 문서·릴리즈 기준)*

**Brave Search API 키 하나면 AI가 실시간 웹검색을 해.**

초보자들이 가장 좋아하는 확장 기능이야.

---

## 뭐가 돼?

| 도구 | 기능 |
|------|------|
| `web_search` | 실시간 웹 검색 (Brave / Perplexity / Gemini 등) |
| `web_fetch` | URL 내용 가져오기 + 읽기 쉬운 텍스트 추출 |

"오늘 날씨 알려줘", "최신 뉴스 검색해줘" — 이런 게 돼.

---

## 설정 방법

**Step 1.** Brave Search API 키 발급

1. [api.search.brave.com](https://api.search.brave.com) 접속
2. 가입/로그인
3. 무료 플랜으로 API 키 발급 (월 2,000 쿼리 무료)
4. 발급된 API 키를 복사해둬

**Step 2.** OpenClaw에 웹검색 설정

가장 쉬운 방법:

```bash
openclaw configure --section web
```

또는 직접 설정:

```bash
openclaw config set web.search.provider brave
openclaw config set web.search.brave.apiKey "여기에-API-키"
```

**Step 3.** 테스트

대시보드에서 "오늘 날씨 알려줘"라고 입력해봐.
AI가 실시간으로 웹을 검색해서 알려주면 성공이야.

---

## 비용은?

Brave Search 무료 플랜이 월 2,000 쿼리야.
가볍게 쓰면 충분해.

더 많이 쓰면 유료 플랜으로 올리거나 Perplexity, Gemini 같은 다른 프로바이더를 써도 돼.

---

## 한 줄 정리

Brave API 키 발급 → `openclaw configure --section web` → 끝. 이제 AI가 실시간 검색을 해.
