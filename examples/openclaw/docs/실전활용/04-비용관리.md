## 비용 줄이는 7가지 방법

### 1. 모델 티어링 (가장 효과 큼)

모든 작업에 비싼 모델 쓸 필요 없어.

```json5
{
  models: {
    primary: "anthropic/claude-sonnet-4-5",     // 메인 대화
    reasoning: "anthropic/claude-opus-4-6",      // 복잡한 판단만
    simple: "openai/gpt-4o-mini",                // 단순 작업
    sub_agents: "openai/gpt-4o-mini",            // 하위 에이전트
    summarization: "google/gemini-flash"          // 요약
  }
}
```

**효과**: 하루 $3~8 → **$0.30~0.80**. 60~80% 절감.

### 2. 크론 빈도 줄이기

```
❌ 5분마다 AI 추론 = 월 ~$128 (GPT-4o)
✅ 하루 2번 + Gemini Flash = 월 ~$0.50
```

빈도 낮추고, 모델을 가볍게. 이것만으로 240배 차이.

### 3. 세션 컨텍스트 관리

대화가 길어지면 토큰이 누적돼.
매 요청마다 **전체 대화 기록**이 모델한테 가거든.

```json5
{
  sessions: {
    max_context_messages: 20,    // 최근 20개만 유지
    summarize_after: 15          // 15개 넘으면 요약
  }
}
```

**핵심**: 주제가 바뀌면 **새 세션 시작해**. 한 세션에서 계속 이어가면 토큰이 눈덩이처럼 불어.

### 4. 하트비트에 로컬 모델 쓰기

HEARTBEAT.md가 주기적으로 체크하는데, 이걸 클라우드 모델로 하면 비용이 쌓여.

**Ollama 로컬 모델**로 하트비트 돌리면 **$0**.
한 개발자는 이걸로 하루 $5 → **$0** 절감했어.

빈 HEARTBEAT.md는 아예 LLM 호출을 건너뛰니까 비용 $0.

### 5. SOUL.md 짧게 유지

SOUL.md가 길면 매번 토큰을 먹어. **2,000토큰(50~150줄) 이내**로.
참고자료는 필요할 때만 불러오게 해.

### 6. 출력 토큰 제한

답변이 길어지면 출력 토큰도 비용이야.

```bash
openclaw config set model.maxOutputTokens 500
```

간결한 답변만 필요하면 이걸로 제한해.

### 7. 안 쓰는 기능 끄기

음성 안 쓰면 TTS/STT 끄고, 이미지 생성 안 쓰면 해당 스킬 제거.

```json5
{
  auto_speak: "talk_mode_only",   // 음성 모드일 때만 TTS
  max_speak_length: 300           // TTS 최대 길이 제한
}
```

## 지출 한도 설정 (필수!)

**이게 가장 중요해.** 폭주 방지 안 해두면 하루에 $200 나가.

### OpenClaw 내부 한도

```bash
# 일일 지출 한도 (넘으면 자동 일시정지)
openclaw config set limits.maxDailySpend 10

# 사용자별 시간당 메시지 제한
openclaw config set limits.maxMessagesPerUserPerHour 20
```

### AI 제공사 한도 (2차 안전망)

각 제공사 대시보드에서 **월 지출 한도**를 반드시 설정해:
- Anthropic: console.anthropic.com
- OpenAI: platform.openai.com/usage
- Google: aistudio.google.com

**팁**: 커뮤니티에서는 **가상 카드**(Privacy.com 등)를 쓰는 사람도 있어. 카드별 한도를 설정해두면 소프트웨어 설정과 무관하게 절대 한도가 걸려.

## 무료로 쓸 수 있어?

| 방법 | 제약 |
|------|------|
| **Oracle Cloud 프리티어** | ARM 4코어, 24GB RAM 무기한 무료. 서버로 쓸 수 있어 |
| **Gemini Flash 무료 티어** | 속도 제한 있지만 가벼운 작업은 충분 |
| **Ollama 로컬 모델** | 토큰 비용 $0. 근데 실행력이 약해 |

완전 무료는 어렵지만, 인프라 + 경량 모델로 **월 $10 이하**도 가능해.

한 가이드에서는 VPS $4~8 + 무료 모델 조합으로 **월 $0~8** 운영 사례를 소개했어.
