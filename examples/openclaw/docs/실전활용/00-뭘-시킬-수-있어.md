*검증 기준일: 2026-02-26 (공식 문서·릴리즈 기준)*

**반복하는 건 다 시킬 수 있어.**

메일, 파일, 일정, 보고서, 서버 감시, 스마트홈 — "이거 해줘"하면 돼.

---

## 가장 많이 쓰는 것들

### 1. 모닝 브리핑 (제일 인기)

매일 아침 텔레그램으로 이런 메시지가 와:

```
🌤 서울 맑음, 최고 23°C
📅 오늘 일정: 팀 미팅 10시, 클라이언트 콜 2시
📧 중요 메일 2건: [발신자] [제목]
```

설정 한 줄이면 돼:

```
"매일 아침 6:30에 서울 날씨, 오늘 첫 3개 일정, BBC 뉴스 톱 3개를
150단어 이내로 전송해줘"
```

매일 아침 5~6개 앱 열던 게 메시지 하나로 끝나.

### 2. 이메일 자동 분류

| 분류 | 처리 |
|------|------|
| 긴급 | 즉시 알림 |
| 참고 | 읽기만 |
| 프로모션 | 자동 보관 |

초안까지 써주지만 **최종 전송은 사람이 해**. "AI가 판단, 사람이 실행."

실제 성과: 하루 2시간+ → **25분** (78% 감소).

### 3. 회의록 자동 정리

회의 끝나고 이렇게 말해:

```
이 회의 녹음 정리해줘.
1) 3문단 요약
2) 할 일 목록 (담당자 + 기한)
3) 결정 사항 리스트
```

30분짜리 후처리가 메시지 한 번으로 끝나.

### 4. 서버 모니터링

```
디스크 85% 넘거나, CPU 80% 5분 이상 지속되면 알려줘.
프로덕션이랑 개발 서버 임계값은 다르게.
```

GitHub Actions, GitLab CI, Jenkins 빌드 실패도 알려줘.
"테스트가 10분 넘으면 알려줘" — 이런 것도 돼.

### 5. 콘텐츠 멀티 플랫폼 변환

블로그 글 하나를 이렇게 시켜:

```
이 글을 X, 링크드인, 인스타, 틱톡, 뉴스레터용으로 각각 변환해줘.
```

플랫폼별로 톤이랑 길이를 알아서 맞춰.
이미지도 만들어줘 (DALL-E, Gemini 연동).

## 실제 사례

| 누가 | 뭘 했어? |
|------|---------|
| IT 서비스 회사 | OpenClaw 1개로 서버 6대, 이메일 4개, SNS 4개 관리 |
| 중국 개발자 | 에이전트 16개로 SNS 13개 혼자 운영. 1달에 팔로워 3,000명 증가 |
| 브런치 작가 | 목적별 텔레그램 봇 분리. 팀원들은 AI 구독 없이 데이터 분석 |

그 IT 서비스 회사의 구체적인 성과:
- **신규 고객 온보딩**: 3~4시간 → **15분** (계약 → 드라이브 폴더 → 환영 메일 → 킥오프 미팅)
- **보고서 생성**: 4~6시간 → **5분** (데이터 수집 → 차트 → 이메일/슬랙 배포)

## 유튜브 실전에서 바로 가져온 5개 템플릿 (2026-02)

### 1. 회의 준비 자동화 (캘린더 + 리마인더)

```text
매일 오전 7시에 오늘 내 캘린더를 확인해서
각 회의별 준비 문서를 만들어줘.
- 참석자 배경
- 지난 대화 요약
- 오늘 꼭 물어볼 질문 3개
그리고 회의 15분 전에 핵심만 다시 보내줘.
```

### 2. 의존성 점검 + 안전 업데이트

```text
12시간마다 레포를 점검해.
- 취약 의존성/패치 가능 항목 확인
- breaking change 없는 범위에서만 lockfile 업데이트
- 테스트/린트 결과를 포함해서 보고
- 실패하면 원인만 정리하고 자동 배포는 하지 마
```

### 3. 비용 이상 탐지 (클라우드/API)

```text
매시간 비용/사용량을 점검해.
- 최근 60분 급증 탐지
- 원인 후보 3개
- 지금 당장 할 완화 액션 3개
알림은 Discord와 WhatsApp 둘 다 보내줘.
```

### 4. 리서치 파이프라인 (트렌드 → 아웃라인)

```text
2시간마다 내 주제(예: OpenClaw, AI Agent) 트렌드를 수집하고
1) 핵심 링크
2) 요약
3) 영상/글 아웃라인 초안
4) 내가 바로 쓸 액션 아이템
형태로 정리해줘.
```

### 5. 아침 전 To-do 선처리

```text
매일 오전 6시에 내 할 일에서
너가 대신할 수 있는 1~3개를 먼저 처리해.
완료 결과와 근거를 문서로 남기고,
파괴적 작업/외부 발송은 초안까지만 만들고 승인 기다려.
```

## 더 있어?

| 시키는 것 | 어떻게 해? |
|----------|-----------|
| 영수증 → 스프레드시트 | 사진 찍으면 업체명, 날짜, 금액, 카테고리 자동 추출 |
| 장보기 목록 | 채팅에서 "우유 사야 해" 감지 → 공유 문서에 자동 추가 |
| 택배 추적 | 주문 확인 메일에서 송장번호 추출 → 배송 현황 대시보드 |
| PR 리뷰 | 코드 변경 요약, 잠재 이슈 플래그, 리뷰 초안 작성 |
| 의존성 스캔 | package.json 분석 → 업그레이드 우선순위 + 호환성 경고 |
| 스마트홈 | Home Assistant 연동. "사무실 불 켜줘" |
| 개인 문서 분석 | Ollama 연동으로 계약서, 법률문서 **로컬에서만** 분석 |
| 브라우저 자동화 | 반복 폼 입력, 데이터 추출 (**가장 위험한 기능**이기도 해) |
