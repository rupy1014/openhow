**메신저에서 "해줘"라고 치면 AI가 진짜 해.**

이게 OpenClaw(오픈클로)야.

---

## ChatGPT랑 뭐가 달라?

ChatGPT한테 "이메일 정리해줘"라고 하면?
"이렇게 하면 됩니다~" 라고 **알려줘**.

OpenClaw한테 같은 말 하면?
진짜 이메일 열어서 **정리해**.

| | ChatGPT | OpenClaw |
|---|---|---|
| 어디서 말해? | 브라우저 | 텔레그램, 디스코드, 슬랙, 왓츠앱 |
| AI가 뭘 해? | 알려줘 | 직접 해 |
| 파일 만져? | 못 만져 | 만들고, 읽고, 정리해 |
| 이메일 보내? | 못 보내 | 보내 |
| 24시간 돌아? | 브라우저 닫으면 끝 | 백그라운드에서 계속 돌아 |

> "조언하는 AI가 아니라, 실행하는 AI."

---

## 어떻게 돌아가?

중앙 우체국 같은 거야.

```
너 (텔레그램) → "오늘 받은 이메일 정리해줘"
     ↓
OpenClaw Gateway (메신저랑 AI를 연결하는 중간 다리)
     ↓
AI (Claude, GPT, Gemini 등) → 이메일 읽기 → 분류
     ↓
너 (텔레그램) ← "정리 끝! 중요 3건, 광고 12건 삭제했어."
```

텔레그램이든 디스코드든 왓츠앱이든, 메시지가 오면 Gateway가 AI한테 전달해.
AI가 일하고, 결과를 다시 메신저로 보내줘.

채널마다 다른 AI를 붙일 수도 있어.
텔레그램은 Claude, 디스코드는 Gemini — 이런 식으로.

## 실제로 사람들이 뭘 시켜?

| 시키는 것 | 효과 |
|----------|------|
| 매일 아침 날씨·일정·메일 요약 | 앱 5~6개 열던 게 메시지 하나로 |
| 이메일 분류 자동화 | 하루 2시간 → **25분** (78% 감소) |
| 신규 고객 온보딩 | 3~4시간 → **15분** |
| 블로그 → X, 링크드인, 인스타 변환 | 플랫폼별 맞춤 콘텐츠 한 번에 |
| 서버 CPU 80% 넘으면 알림 | 디스코드로 실시간 알림 |

한 개발자는 에이전트 16개로 SNS 13개를 혼자 운영해.
한 IT 회사는 OpenClaw 하나로 서버 6대 + 이메일 4개 + SNS 4개를 관리하고 있어.
