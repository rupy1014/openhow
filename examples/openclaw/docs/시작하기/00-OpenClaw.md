**메신저로 AI한테 일을 시키는 오픈소스 도구야.**

---

## 이게 뭔데 요즘 핫해?

ChatGPT, Claude, Gemini — AI 다 좋아.
근데 전부 브라우저 안에 갇혀 있어.

"이메일 정리해줘" 하면 방법만 알려줘.
직접 해주진 않아.

OpenClaw는 그 AI를 **브라우저 밖으로 꺼내**.
메신저에서 시키면 진짜 해.
오픈소스라 무료고, AI도 골라 써.

| | ChatGPT | OpenClaw |
|---|---|---|
| 뭘 해? | 알려줘 | 직접 해 |
| 파일 만져? | 못 만져 | 만들고, 읽고, 정리해 |
| 이메일 보내? | 못 보내 | 보내 |

> "조언하는 AI가 아니라, 실행하는 AI."

---

## 뭐가 돼?

핵심이 4가지야.

| 핵심 기능 | 한 줄 설명 |
|----------|-----------|
| **리모컨** | 텔레그램, 디스코드, 슬랙 — 메신저가 AI 리모컨이 돼 |
| **AI 선택** | Claude, GPT, Gemini — 구독 중인 AI를 골라 쓰고 바꿔 |
| **직접 실행** | "이렇게 해"가 아니라 진짜 해 |
| **자율 동작** | 24시간, 예약 걸어두면 알아서 돌아 |

---

## 메신저가 리모컨이야?

텔레그램, 디스코드, 슬랙, 왓츠앱 — 네가 쓰는 메신저로 AI를 조종해.

브라우저 안 열어도 돼.
메신저에 "이메일 정리해줘"라고 치면 끝.

```
너 (텔레그램) → "오늘 받은 이메일 정리해줘"
     ↓
OpenClaw (메신저랑 AI를 연결하는 중간 다리)
     ↓
AI → 이메일 읽기 → 분류
     ↓
너 (텔레그램) ← "정리 끝! 중요 3건, 광고 12건 삭제했어."
```

채널마다 다른 AI를 붙일 수도 있어.
텔레그램은 Claude, 디스코드는 Gemini — 이런 식으로.

---

## AI를 골라 써?

Claude, GPT, Gemini, Grok, 로컬 Ollama — 네가 구독 중인 AI를 그대로 쓸 수 있어.

API 키만 연결하면 돼.
바꾸고 싶으면 한 줄이면 끝.

```bash
openclaw config set agent.model "anthropic/claude-sonnet-4-5"
```

작업별로 다른 모델을 배정하면 비용이 확 줄어.

| 작업 | 모델 | 왜? |
|------|------|-----|
| 메인 작업 | Claude Sonnet | 가성비 |
| 단순 작업 | GPT-4o-mini | 저렴 |
| 요약 | Gemini Flash | 제일 저렴 |

하루 $3~8 쓰던 게 **$0.30~0.80**으로 줄어.

---

## 내가 안 봐도 돼?

예약 걸어두면 매일 알아서 해.

```
매일 아침 6:30 → 날씨, 일정, 중요 메일 요약
평일 오후 6시 → 퇴근 리포트
2시간마다 → 서버 상태 체크
```

브라우저 닫아도 상관없어.
서버에 올리면 컴퓨터 꺼도 계속 돌아.

---

## 실제로 뭘 시켜?

| 시키는 것 | 효과 |
|----------|------|
| 매일 아침 날씨·일정·메일 요약 | 앱 5~6개 열던 게 메시지 하나로 |
| 이메일 분류 자동화 | 하루 2시간 → **25분** (78% 감소) |
| 신규 고객 온보딩 | 3~4시간 → **15분** |
| 블로그 → SNS 변환 | 플랫폼별 맞춤 콘텐츠 한 번에 |
| 서버 CPU 80% 넘으면 알림 | 디스코드로 실시간 알림 |

한 개발자는 에이전트 16개로 SNS 13개를 혼자 운영해.
한 IT 회사는 OpenClaw 하나로 서버 6대 + 이메일 4개 + SNS 4개를 관리하고 있어.
