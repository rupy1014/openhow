## 어떻게 설치해?

**Step 1.** 터미널 열기

- **Mac**: Spotlight(⌘ + Space) → "터미널" 검색
- **Windows**: WSL2 터미널 권장 (PowerShell 설치도 가능)
- **Linux**: 터미널 열기

**Step 2.** 설치

```bash
# Mac / Linux
curl -fsSL https://openclaw.ai/install.sh | bash

# Windows (PowerShell)
iwr -useb https://openclaw.ai/install.ps1 | iex
```

npm으로도 돼 (Node.js 22 이상 필요):

```bash
npm install -g openclaw@latest
```

**Step 3.** 설치 확인

```bash
openclaw --version
```

`2026.2.24` 같은 버전이 나오면 성공이야.

---

## 설치 후 바로 해야 할 것

**온보딩 마법사를 실행해.**

```bash
openclaw onboard --install-daemon
```

이거 하나면 다 잡아줘:
1. AI 모델 선택 (Anthropic 추천, 저렴하게 가려면 Gemini Flash)
2. API 키 입력
3. 메신저 채널 선택 (텔레그램, 왓츠앱 등)
4. 채널별 설정 안내
5. Gateway를 백그라운드 서비스로 등록 (`--install-daemon`)

끝나면 바로 메신저에서 AI한테 말 걸 수 있어.

## 문제가 생기면?

```bash
# 상태 확인
openclaw status

# 설정 진단
openclaw doctor

# 자동 수리
openclaw doctor --fix

# 실시간 로그
openclaw logs --follow
```

`openclaw doctor`가 대부분 문제를 알려줘. `--fix` 붙이면 자동으로 고쳐줘.

## Mac에서 설치 안 될 때

SUDO 권한 문제가 주된 원인이야.

```bash
sudo npm install -g openclaw@latest
```

그래도 안 되면 nvm으로 Node.js를 다시 깔아봐.

## 24시간 돌리고 싶으면?

| 방법 | 비용 | 특징 |
|------|------|------|
| **Mac에서 직접** | 전기세 월 ~500~2,000원 | M시리즈는 유휴 3~10W. 사용량에 따라 차이 |
| **DigitalOcean** | $6~20/월 | 원클릭 배포 지원 |
| **Oracle Cloud** | **무료** | ARM 4코어, 24GB RAM. 무기한 프리티어 |
| **Docker** | 서버 비용만 | `docker pull openclaw/openclaw:latest` |

> Mac Mini M시리즈 24시간 전기세 계산:
> 유휴 ~5W × 24h × 30일 = 3.6kWh/월 × 약 120원/kWh = **약 430원/월**.
> Node.js 프로세스 돌리면 ~10W 정도. 그래도 **월 ~1,000원** 수준이야.

서버 권장: CPU 2코어+, RAM 8GB, Ubuntu 22.04 LTS.

**주의**: Mac에서 돌리면 편하지만, 해킹당하면 홈 네트워크가 뚫려.
VPS가 더 안전해. 털려도 빈 리눅스 박스만 있으니까.
