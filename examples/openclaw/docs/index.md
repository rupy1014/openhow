# 메신저로 AI한테 일 시키기

> 텔레그램에서 "이메일 정리해줘"라고 치면 진짜 정리해. 그게 OpenClaw이야.

:::success 이 가이드를 따라하면
- 메신저에서 AI한테 말로 일을 시킬 수 있어
- 이메일, 파일, 서버 관리를 자동화할 수 있어
- 비용도 관리하면서 효율적으로 운영할 수 있어
:::

**대상**: AI 도구를 써보고 싶은데 어디서 시작할지 모르는 사람
**핵심 도구**: OpenClaw — 메신저를 AI 리모컨으로 바꿔주는 오픈소스 프레임워크
**비용**: OpenClaw는 무료. AI 모델 API만 종량제 (월 1~2만원부터)

## 실제 성과

| 뭘 시켰어? | 결과 |
|-----------|------|
| 이메일 분류 자동화 | 하루 2시간 → **25분** (78% 감소) |
| 신규 고객 온보딩 | 3~4시간 → **15분** |
| 보고서 생성 | 4~6시간 → **5분** |

*비용·성과 수치는 사용량/환경에 따라 달라질 수 있어.*

- 텔레그램이나 디스코드를 이미 쓰고 있어
