**텔레그램 봇 하나 만들면 바로 연결돼.**

---

## 어떻게 연결해?

**Step 1.** 텔레그램에서 @BotFather 검색

텔레그램 앱 열고, 검색창에 `@BotFather`를 쳐.
파란 체크 붙은 공식 계정이야.

**Step 2.** 봇 만들기

BotFather한테 이렇게 말해:

```
/newbot
```

봇 이름이랑 사용자명을 물어봐. 대답하면 **토큰**을 줘.
복사해둬.

**Step 3.** 설정에 토큰 넣기

```bash
openclaw config set channels.telegram.botToken "여기에-토큰"
openclaw config set channels.telegram.enabled true
openclaw gateway restart
```

또는 설정 파일에 직접:

```json5
{
  channels: {
    telegram: {
      enabled: true,
      botToken: "1234567890:ABCdefGHIjklMNOpqrSTUvwxYZ",
      dmPolicy: "pairing"
    }
  }
}
```

**Step 4.** 페어링 승인

처음 메시지를 보내면 페어링 코드가 나와.
터미널에서 승인해줘야 해:

```bash
openclaw pairing list telegram
openclaw pairing approve telegram 코드
```

이게 보안 장치야. 아무나 너 봇 쓰는 걸 막아줘.

**Step 5.** 테스트

봇한테 아무 말이나 해봐.

```
오늘 날씨 어때?
```

AI가 대답하면 성공이야.

---

## DM 정책 (누가 말 걸 수 있어?)

| 정책 | 설명 |
|------|------|
| `pairing` (기본) | 처음 말 걸면 승인 코드 필요 |
| `allowlist` | 허용 목록에 있는 사람만 |
| `open` | 아무나 (주의!) |
| `disabled` | DM 차단 |

```bash
# 특정 사람만 허용
openclaw config set channels.telegram.dmPolicy "allowlist"
openclaw config set channels.telegram.allowFrom '["@너의아이디"]'
```

## 그룹 채팅에서 쓰려면?

그룹에 봇을 추가하고, 멘션해야 반응하게 설정해:

```json5
{
  channels: {
    telegram: {
      groups: {
        "*": { requireMention: true }
      }
    }
  }
}
```

봇이 그룹 메시지를 못 읽으면 BotFather에서 프라이버시 모드를 꺼:

```
/setprivacy → Disable
```

## 실시간 스트리밍

AI가 생각하는 과정을 실시간으로 볼 수 있어.
이모지로 상태도 표시돼: 🤔 생각 중 → 🔧 도구 사용 → ✅ 완료

스트리밍 모드 옵션:

| 모드 | 설명 |
|------|------|
| `partial` | 타이핑하듯 실시간 표시 |
| `block` | 완성 후 한 번에 표시 |
| `progress` | 진행 상황만 표시 |

---

## 웹훅 vs 폴링

기본은 폴링(주기적으로 확인)이야.
서버에서 돌리면 웹훅이 더 효율적이야:

```json5
{
  channels: {
    telegram: {
      webhookUrl: "https://너의도메인.com/telegram",
      webhookSecret: "랜덤-시크릿"
    }
  }
}
```
