## 모델 바꾸기

```bash
openclaw config set agent.model "anthropic/claude-sonnet-4-5"
openclaw gateway restart
```

## 모델 티어링 (비용 절약 핵심)

**모든 작업에 비싼 모델 쓸 필요 없어.**
작업 종류별로 다른 모델을 배정하면 비용이 확 줄어.

```json5
{
  models: {
    primary: "anthropic/claude-sonnet-4-5",       // 메인
    reasoning: "anthropic/claude-opus-4-6",        // 복잡한 판단
    simple: "openai/gpt-4o-mini",                  // 단순 작업
    sub_agents: "openai/gpt-4o-mini",              // 하위 에이전트
    summarization: "google/gemini-flash"            // 요약
  }
}
```

**효과**: 프리미엄만 쓰면 하루 $3~8 → 티어링하면 **$0.30~0.80**.

## 폴백 체인 (장애 대비)

AI 서비스가 다운될 때 자동으로 다른 모델로 전환:

```json5
{
  models: {
    failover: [
      { provider: "anthropic", model: "claude-sonnet-4-5" },
      { provider: "openai", model: "gpt-4o" },
      { provider: "google", model: "gemini-1.5-pro" },
      { provider: "openai", model: "gpt-4o-mini" }
    ]
  }
}
```

1번이 안 되면 2번, 2번도 안 되면 3번 — 자동이야.

## 로컬 모델은 실용적이야?

솔직히 아직은 한계가 커.

실사용자 후기: Qwen 3:8b로 "파일 삭제해줘"라고 시켰더니
**설명만 하고 실행을 안 해**. "삭제하려면 이렇게 하면 돼요~" 이러고 끝.

로컬 모델 쓸 곳:
- 민감한 문서(계약서, 법률문서) 분석 — 외부로 안 나가니까
- 단순 텍스트 처리 — 비용 $0

실행력이 필요한 작업은 외부 모델이 필수야.
