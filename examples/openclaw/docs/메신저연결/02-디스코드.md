**디스코드 봇을 만들고 서버에 초대하면 끝.**

채널별로 다른 AI를 붙일 수 있고, 보이스 채널에도 참여해.

---

## 어떻게 연결해?

**Step 1.** [discord.com/developers/applications](https://discord.com/developers/applications) 열어

**Step 2.** "New Application" → 이름 입력 → "Bot" 탭 → "Reset Token" → 토큰 복사

**Step 3.** "OAuth2" → "URL Generator" → `bot` 체크 → 권한 설정 → URL로 서버에 초대

**Step 4.** 설정에 토큰 넣기

```bash
openclaw config set channels.discord.botToken "여기에-토큰"
openclaw config set channels.discord.enabled true
openclaw gateway restart
```

**Step 5.** 봇 멘션하거나 DM으로 말 걸어봐.

---

## 채널별로 다른 AI 쓰기

이게 디스코드의 킬러 포인트야.

```json5
{
  channels: {
    discord: {
      botToken: "...",
      channels: {
        dev: { provider: "anthropic", model: "claude-opus-4-6" },
        writing: { provider: "openai", model: "gpt-4o" },
        research: { provider: "google", model: "gemini-3.1-pro-preview" }
      }
    }
  }
}
```

`#dev`에선 Claude, `#writing`에선 GPT, `#research`에선 Gemini.
팀원은 채널만 바꾸면 다른 AI를 써.

---

## 보이스 채널

`/vc` 명령어로 AI를 보이스 채널에 불러올 수 있어.
말하면 AI가 듣고 대답해.

이모지 리액션으로 상태 표시:
- 🤔 생각 중
- 🔧 도구 사용 중
- ✅ 완료

---

## 추천 채널 구성

| 채널 | 용도 |
|------|------|
| `#admin` (비공개) | 관리자 전용. 설정 변경, 디버깅 |
| `#general` | 팀원들이 자유롭게 질문 |
| `#dev` | 개발 관련 (코드 리뷰, 버그 리포트) |
| `#monitoring` | 서버 알림, CI/CD 결과 |

역할(Role) 기반으로 접근 제한하면 깔끔해.
