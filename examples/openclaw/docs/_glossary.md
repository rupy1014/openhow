# 용어 사전

OpenClaw에서 나오는 단어들을 쉬운 말로 정리했어.

| 용어 | 쉬운 말 |
|------|---------|
| **에이전트 (Agent)** | AI 비서. 시킨 일을 직접 하는 AI |
| **게이트웨이 (Gateway)** | 중간 다리. 메신저랑 AI를 연결해주는 통로 |
| **스킬 (Skill)** | 플러그인. 필요한 기능만 골라 붙이는 부품 |
| **ClawHub** | 스킬 마켓. 앱스토어 같은 곳 |
| **크론 (Cron)** | 예약 실행. "매일 아침 7시에 해줘" |
| **페어링 (Pairing)** | 첫 연결 시 승인 절차. 모르는 사람이 봇 쓰는 걸 막아줘 |
| **세션 (Session)** | 대화 단위. 새 세션 = 새 대화 시작 |
| **API 키** | AI 서비스 이용증. 이걸로 요금이 나가니까 절대 공개하지 마 |
| **토큰 (Token)** | AI가 읽는 글자 단위. 한글 1글자 ≈ 2~3토큰 |
| **컨텍스트 (Context)** | AI가 기억하는 대화 범위. 길어지면 비용 증가 |
| **SOUL.md** | AI의 성격 파일. 말투, 가치관, 판단 기준 정의 |
| **JSON5** | OpenClaw 설정 파일 형식. JSON인데 주석 쓸 수 있어 |
| **프로바이더 (Provider)** | AI 제공사. anthropic, openai, google 같은 거 |
| **티어링 (Tiering)** | 작업별로 다른 모델 배정. 비용 절약의 핵심 |
| **폴백 (Fallback)** | 1순위 모델이 안 되면 자동으로 2순위로 전환 |
| **Ollama** | 내 컴퓨터에서 AI 돌리는 도구. 무료 |
| **하트비트 (Heartbeat)** | 주기적 자동 체크. 30분마다 HEARTBEAT.md를 읽고 할 일이 있으면 실행 |
| **컴팩션 (Compaction)** | 긴 대화를 요약 압축하는 과정. `/compact`으로 수동 실행 가능 |
| **워크스페이스 (Workspace)** | 에이전트 설정 파일들이 모인 폴더. `~/.openclaw/workspace/` |
| **WSL2** | Windows에서 Linux 쓸 수 있게 해주는 기능 |
| **Docker** | 프로그램을 통째로 포장해서 어디서든 똑같이 돌리는 도구 |
| **Tailscale** | VPN 서비스. 외부에서 안전하게 내 서버 접속할 때 |
| **loopback** | 127.0.0.1. 내 컴퓨터에서만 접속 가능한 주소 |
