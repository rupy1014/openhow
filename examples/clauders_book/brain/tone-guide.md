# 문체 가이드

## 확정 톤: 구어적 경어체 (문어체 + 말하듯이)

안티그라비타 레퍼런스와 동일한 경어체(~합니다)를 기본으로 하되,
딱딱한 교과서 문체가 아니라 **강연하듯 술술 읽히는 톤**을 유지한다.

## 목표 톤 예시

### Before (현재 반말체)
> 코딩 몰라도 돼. 근데 판단은 네가 해야 해.
> AI는 유능한 신입이야. 손은 빠르고, 기본기도 있고, 밤새도 불평 안 해.
> 근데 신입은 신입이야.

### After (구어적 경어체)
> 코딩을 몰라도 됩니다. 대신 판단은 여러분이 직접 해야 합니다.
> AI는 유능한 신입사원입니다. 손이 빠르고, 기본기도 있고, 밤새도 불평하지 않습니다.
> 하지만 신입은 신입입니다.

### 핵심 감각
- **강연자가 청중에게 말하는 느낌**: "자, 여기서 중요한 건 이겁니다."
- **짧은 호흡 유지**: 한 문장이 2줄을 넘지 않도록
- **구어적 연결어 적극 사용**: "그런데", "사실은", "솔직히", "예를 들어", "왜냐하면"
- **리듬감 있는 열거**: 짧은 문장을 3개씩 나열하는 패턴 유지

## 변환 규칙

| 현재 (반말) | 변환 (구어적 경어) | 비고 |
|------------|----------------|------|
| ~야 | ~입니다 / ~입니다 | |
| ~거든 | ~기 때문입니다 / ~거든요 | "~거든요"도 OK (구어적) |
| ~지 | ~죠 / ~ㅂ니다 | "~죠"로 자연스럽게 |
| ~해 | ~하세요 / ~합니다 | 지시는 "~하세요", 서술은 "~합니다" |
| ~봐 | ~보세요 / ~보십시오 | "~보세요" 권장 |
| 너/네 | 여러분 / 독자분 | 또는 생략 |
| 나 | 필자 / 저 | "저"도 OK (구어적) |
| 근데 | 그런데 / 하지만 | "그런데"는 구어적이므로 OK |
| 진짜 | 정말 / 실제로 | |
| 걍/그냥 | 그냥 | 유지 가능 (구어적 경어에서도 자연스러움) |

## 허용/금지

### 허용 (구어적 느낌 유지)
- "~거든요", "~잖아요", "~죠" — 경어체지만 구어적 어미
- "자, 그러면", "사실", "솔직히 말하면" — 강연체 연결어
- 짧은 감탄: "됩니다.", "가능합니다.", "충분합니다."
- 질문형 전환: "왜 그럴까요?", "어떻게 해야 할까요?"

### 금지
- 반말 ("~야", "~해", "~거든", "~지")
- 평어체 ("~다", "~이다") — 건조해짐
- 과도한 존칭 ("~하시옵소서", "~이시겠습니다")
- 학술체 ("~한 바 있다", "~로 사료된다")

## 레퍼런스 톤 비교

### 안티그라비타 (목표와 가장 가까움)
> "어제까지, 아이디어는 머릿속에만 머물렀습니다. 하지만 오늘, 모든 것이 바뀝니다.
> 여러분의 노트북, 혹은 데스크톱 컴퓨터가 아이디어를 현실로 만드는 강력한 무기가
> 되어 줄 것입니다."

→ 경어체 + 구어적 + 감성적. 이 톤이 목표.

### OpenClaw (참고만)
> "같은 아침이라도 시작 방식은 크게 다르다. 어떤 사람은 메일, 메모, 메신저를 오가며
> 오늘 할 일을 다시 정리한다."

→ 평어체. 우리 톤과 다름. 구조만 참고.
