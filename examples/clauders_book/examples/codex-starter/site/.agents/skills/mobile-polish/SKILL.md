# Mobile Polish

이 skill은 모바일에서 읽기와 누르기 편한지에만 집중합니다.

## 할 일

1. `site/styles.css`를 먼저 읽습니다.
2. 작은 화면에서 줄 길이, 버튼 간격, 카드 간격을 점검합니다.
3. 한 번에 큰 리디자인 대신 작은 개선만 합니다.
4. 수정 후 모바일에서 무엇이 좋아졌는지 한 줄로 설명합니다.

## 수정 범위

- `site/styles.css`
- 필요하면 `site/index.html`의 클래스 구조

## 하지 말 것

- 데스크톱 스타일 전체를 갈아엎지 말 것
- 색상 테마를 갑자기 바꾸지 말 것
- 새 프레임워크나 라이브러리를 추가하지 말 것
