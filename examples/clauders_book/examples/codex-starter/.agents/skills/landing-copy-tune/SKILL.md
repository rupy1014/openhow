# Landing Copy Tune

이 skill은 랜딩 페이지의 문구만 다룹니다.

레이아웃을 크게 바꾸지 않습니다.

## 할 일

1. 현재 hero, lead, CTA 문구를 먼저 읽습니다.
2. 의미는 유지하고 표현만 더 또렷하게 다듬습니다.
3. 과장된 마케팅 표현은 줄입니다.
4. 수정 파일과 바뀐 핵심 문장을 마지막에 요약합니다.

## 수정 범위

- `site/index.html`
- 필요하면 `site/app.js` 안의 상태 문구

## 하지 말 것

- 색상, 간격, 레이아웃을 먼저 건드리지 말 것
- 없는 기능을 새로 제안하지 말 것
- 카피라이팅 분석을 장황하게 늘어놓지 말 것
